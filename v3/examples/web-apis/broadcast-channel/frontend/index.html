Common Use Cases

User Authentication Sync
When a user logs out in one tab, broadcast the event to log them out in all other tabs instantly.
Theme/Settings Sync
Change dark mode in one tab and have it apply to all open tabs of your app.
Shopping Cart Updates
Add an item to cart in one tab and see it reflected in all other open tabs.
Notifications Deduplication
Coordinate between tabs to show notifications only once, not in every tab.
Collaborative Editing
Sync document changes between multiple tabs editing the same content.
