[WEDM](https://github.com/andrewbytecoder/wedm) is a free, cross-platform ETCD v3 client and GUI built with Wails v2, Go, and Vue 3. The goal of the project is to provide an efficient, modern desktop application for Windows, Linux, and macOS that covers all ETCD functionality. Anything you can do with etcdctl, you should be able to do with this tool as well.
