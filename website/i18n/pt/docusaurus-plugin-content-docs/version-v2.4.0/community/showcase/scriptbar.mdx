[ScriptBar](https://GitHub.com/raguay/ScriptBarApp) é um programa que mostra a saída do servidor embutido [Node-Red](https://nodered.org) no aplicativo [EmailIt](https://GitHub.com/raguay/EmailIt). Também exibe a saída de scripts no seu sistema. ScriptBar não os coloca na barra de menus, mas os tenha todos em uma janela convidada para fácil visualização. Você pode ter várias abas para ter muitas coisas diferentes mostradas. Você também pode manter os links para os sites mais visitados.
