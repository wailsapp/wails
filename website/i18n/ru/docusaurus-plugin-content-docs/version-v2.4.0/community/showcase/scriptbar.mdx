[ScriptBar](https://GitHub.com/raguay/ScriptBarApp) это программа для отображения вывода встроенного сервера [Node-Red](https://nodered.org) в приложении [EmailIt](https://GitHub.com/raguay/EmailIt). Он также отображает вывод скриптов в вашей системе. ScriptBar не помещает их в меню, но все они в удобном окне для удобного просмотра. Вы можете открыть несколько вкладок, чтобы показать множество различных вещей. Вы также можете держать ссылки на наиболее посещаемые вами веб-сайты.
