# 编译您的项目

从项目目录，运行 `wails build`。 这将编译您的项目并将构建的可用于生产的二进制文件保存在 `build/bin` 目录中。

如果您运行二进制文件，您应该会看到默认应用程序：

```mdx-code-block

    width="50%"
    class="screenshot"
  />
</div>
<br />
```

有关编译选项的更多详细信息，请参阅 [构建命令](../reference/cli#构建)。
