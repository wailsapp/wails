[Resizem](https://github.com/barats/resizem) - est une application conçue pour le traitement d'images en bloc. Il est particulièrement utile pour les utilisateurs qui ont besoin de redimensionner, de convertir et de gérer un grand nombre de fichiers image à la fois.
