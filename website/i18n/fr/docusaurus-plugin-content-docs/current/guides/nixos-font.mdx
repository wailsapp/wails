# Bug taille de la police dans NixOS

NixOS/Wayland peut causer un bogue où la propriété `font-size` css n'affecte pas la page gérée correctement. Pour corriger cela, ajoutez ce qui suit à votre devShell.

```shell
    shellHook = with pkgs; ''
      export XDG_DATA_DIRS=${gsettings-desktop-schemas}/share/gsettings-schemas/${gsettings-desktop-schemas.name}:${gtk3}/share/gsettings-schemas/${gtk3.name}:$XDG_DATA_DIRS;
      export GIO_MODULE_DIR="${pkgs.glib-networking}/lib/gio/modules/";
    '';
```
