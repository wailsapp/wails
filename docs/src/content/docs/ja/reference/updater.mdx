# アップデーターAPIリファレンス

アップデーターAPIは、Wailsアプリケーションの自動更新機能を提供します。これには、更新のチェック、ダウンロード、およびインストールが含まれます。

## Go API

### CreateUpdaterService

新しいアップデーターサービスを作成します。

```go
func CreateUpdaterService(currentVersion string, opts ...UpdaterOption) (*UpdaterService, error)
```

**パラメータ:**
- `currentVersion` - アプリケーションの現在のバージョン（例: "1.0.0"）
- `opts` - 設定オプション

**戻り値:**
- `*UpdaterService` - アップデーターサービスのインスタンス
- `error` - 作成に失敗した場合のエラー

**例:**
```go
updater, err := application.CreateUpdaterService(
    "1.0.0",
    application.WithUpdateURL("https://updates.example.com/myapp/"),
)
```

### UpdaterOption 関数

#### WithUpdateURL

更新マニフェストのベースURLを設定します。

```go
func WithUpdateURL(url string) UpdaterOption
```

**必須。** URLは `update.json` マニフェストを含むディレクトリを指している必要があります。

#### WithCheckInterval

自動バックグラウンド更新チェックの間隔を設定します。

```go
func WithCheckInterval(interval time.Duration) UpdaterOption
```

自動チェックを無効にするには `0` に設定します。デフォルトは `0`（無効）です。

#### WithAllowPrerelease

プレリリースバージョンを更新として考慮できるようにします。

```go
func WithAllowPrerelease(allow bool) UpdaterOption
```

デフォルトは `false` です。

#### WithChannel

更新チャネル（例: "stable", "beta", "canary"）を設定します。

```go
func WithChannel(channel string) UpdaterOption
```

チャネルは更新URLパスに追加されます。

#### WithPublicKey

署名検証用の公開鍵を設定します。

```go
func WithPublicKey(key string) UpdaterOption
```

鍵はbase64エンコードされたEd25519公開鍵である必要があります。

#### WithRequireSignature

すべての更新に署名を要求します。

```go
func WithRequireSignature(require bool) UpdaterOption
```

デフォルトは `false` です。

### UpdaterService メソッド

#### GetCurrentVersion

現在のアプリケーションバージョンを返します。

```go
func (u *UpdaterService) GetCurrentVersion() string
```

#### CheckForUpdate

新しいバージョンが利用可能かどうかをチェックします。

```go
func (u *UpdaterService) CheckForUpdate() (*UpdateInfo, error)
```

**戻り値:**
- `*UpdateInfo` - 更新情報。更新がない場合は `nil`
- `error` - チェックに失敗した場合のエラー

#### DownloadUpdate

利用可能な更新をダウンロードします。

```go
func (u *UpdaterService) DownloadUpdate() error
```

ダウンロード中に `updater:progress` イベントを発行します。

#### ApplyUpdate

ダウンロードした更新を適用します。

```go
func (u *UpdaterService) ApplyUpdate() error
```

これによりアプリケーションが再起動します。

#### DownloadAndApply

1つの呼び出しで更新をダウンロードして適用します。

```go
func (u *UpdaterService) DownloadAndApply() error
```

#### GetState

現在のアップデーター状態を返します。

```go
func (u *UpdaterService) GetState() string
```

**可能な値:**
- `"idle"` - 更新進行中なし
- `"checking"` - 更新をチェック中
- `"available"` - 更新可能
- `"downloading"` - 更新をダウンロード中
- `"ready"` - 更新をダウンロード済み、インストール準備完了
- `"installing"` - 更新をインストール中
- `"error"` - エラー発生

#### GetUpdateInfo

利用可能な更新に関する情報を返します。

```go
func (u *UpdaterService) GetUpdateInfo() *UpdateInfo
```

#### GetLastError

最後のエラーメッセージを返します。

```go
func (u *UpdaterService) GetLastError() string
```

#### Reset

アップデーター状態をリセットします。

```go
func (u *UpdaterService) Reset()
```

### UpdateInfo

利用可能な更新に関する情報。

```go
type UpdateInfo struct {
    Version      string    `json:"version"`
    ReleaseDate  time.Time `json:"releaseDate"`
    ReleaseNotes string    `json:"releaseNotes"`
    Size         int64     `json:"size"`
    PatchSize    int64     `json:"patchSize,omitempty"`
    Mandatory    bool      `json:"mandatory"`
    HasPatch     bool      `json:"hasPatch"`
}
```

| フィールド | 型 | 説明 |
|-------|------|-------------|
| `Version` | `string` | 新しいバージョン番号 |
| `ReleaseDate` | `time.Time` | このバージョンがリリースされた日時 |
| `ReleaseNotes` | `string` | リリースノート（マークダウン） |
| `Size` | `int64` | 完全なダウンロードサイズ（バイト） |
| `PatchSize` | `int64` | パッチサイズ（バイト、利用可能な場合） |
| `Mandatory` | `bool` | この更新が必須かどうか |
| `HasPatch` | `bool` | パッチが利用可能かどうか |

### UpdaterConfig

アップデーターの設定。

```go
type UpdaterConfig struct {
    UpdateURL        string
    CheckInterval    time.Duration
    AllowPrerelease  bool
    PublicKey        string
    RequireSignature bool
    Channel          string
}
```

## JavaScript/TypeScript API

アップデーターサービスは、サービスとして追加されるとフロントエンドに自動的にバインドされます。

interface DownloadProgress {
  downloaded: number;
  total: number;

### イベント

#### updater:progress

ダウンロード中に進行状況情報とともに発行されます。

```typescript
import { Events } from '@wailsio/runtime';

Events.On('updater:progress', (data: DownloadProgress) => {
  console.log(`${data.percentage}% ダウンロード済み`);
});
```

## 更新マニフェスト形式

更新マニフェスト（`update.json`）は、あなたの更新URLにホストされている必要があります。

### スキーマ

```typescript
interface Manifest {
  version: string;
  release_date: string;      // ISO 8601
  release_notes?: string;
  platforms: {
    [platform: string]: PlatformUpdate;
  };
  minimum_version?: string;
  mandatory?: boolean;
}

interface PlatformUpdate {
  url: string;
  size: number;
  checksum: string;          // "sha256:..."
  signature?: string;
  patches?: PatchInfo[];
}

interface PatchInfo {
  from: string;              // このパッチが適用されるバージョン
  url: string;
  size: number;
  checksum: string;
  signature?: string;
}
```

### プラットフォームキー

| プラットフォーム | キー |
|----------|-----|
| macOS (Apple Silicon) | `macos-arm64` |
| macOS (Intel) | `macos-amd64` |
| Windows (64-bit) | `windows-amd64` |
| Linux (64-bit) | `linux-amd64` |
| Linux (ARM64) | `linux-arm64` |

## サービスの統合

### サービスの登録

```go
app := application.New(application.Options{
    Name: "MyApp",
    Services: []application.Service{
        application.NewService(updater),
    },
})
```

### サービスのライフサイクル

アップデーターサービスは `Service` インターフェースを実装しています:

```go
type Service interface {
    ServiceName() string
    ServiceStartup(ctx context.Context, options ServiceOptions) error
    ServiceShutdown() error
}
```

- **ServiceStartup**: 設定されている場合、バックグラウンドでの更新チェックを開始します
- **ServiceShutdown**: バックグラウンドでのチェックを停止します

### イベントハンドラー

Go でハンドラーを登録します:

```go
updater.OnUpdateAvailable(func(info *UpdateInfo) {
    log.Printf("Update available: %s", info.Version)
})

updater.OnDownloadProgress(func(downloaded, total int64, percentage float64) {
    log.Printf("Download: %.1f%%", percentage)
})

updater.OnError(func(err string) {
    log.Printf("Update error: %s", err)
})
```

## セキュリティに関する考慮事項

### HTTPS

中間者攻撃を防ぐために、更新 URL には常に HTTPS を使用してください。

### チェックサム検証

すべてのダウンロードは、インストール前に SHA256 チェックサムに対して検証されます。

### 署名検証

本番環境のアプリケーションでは署名検証を有効にしてください:

```go
updater, _ := application.CreateUpdaterService(
    "1.0.0",
    application.WithUpdateURL("https://updates.example.com/myapp/"),
    application.WithRequireSignature(true),
    application.WithPublicKey("MCowBQYDK2VwAyEA..."),
)
```

### コード署名

アプリケーションのバイナリが適切にコード署名されていることを確認してください:
- macOS: Developer ID Application 証明書
- Windows: タイムスタンプ付きのコード署名証明書

詳細については、[コード署名ガイド](/ja/guides/build/signing) を参照してください。
