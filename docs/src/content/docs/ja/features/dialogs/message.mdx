## メッセージダイアログ

Wails は、プラットフォームに適切な外観を持つ**ネイティブメッセージダイアログ**を提供します。情報、警告、エラー、質問のダイアログがあり、タイトル、メッセージ、ボタンをカスタマイズできます。シンプルな API、ネイティブな動作、デフォルトでアクセシビリティに対応しています。

## ダイアログの作成

メッセージダイアログは `app.Dialog` マネージャーを通じてアクセスします:

```go
app.Dialog.Info()
app.Dialog.Question()
app.Dialog.Warning()
app.Dialog.Error()
```

すべてのメソッドは、メソッドチェーンを使用して構成できる `*MessageDialog` を返します。

## 情報ダイアログ

情報メッセージを表示します:

```go
app.Dialog.Info().
    SetTitle("Success").
    SetMessage("File saved successfully!").
    Show()
```

**使用例:**
- 成功の確認
- 完了通知
- 情報メッセージ
- ステータス更新

**例 - 保存確認:**

```go
func saveFile(app *application.App, path string, data []byte) error {
    if err := os.WriteFile(path, data, 0644); err != nil {
        return err
    }

    app.Dialog.Info().
        SetTitle("File Saved").
        SetMessage(fmt.Sprintf("Saved to %s", filepath.Base(path))).
        Show()

    return nil
}
```

## 警告ダイアログ

警告を表示します:

```go
app.Dialog.Warning().
    SetTitle("Warning").
    SetMessage("This action cannot be undone.").
    Show()
```

**使用例:**
- 非クリティカルな警告
- 非推奨通知
- 注意メッセージ
- 潜在的な問題

**例 - ディスク容量の警告:**

```go
func checkDiskSpace(app *application.App) {
    available := getDiskSpace()

    if available < 100*1024*1024 { // 100MB未満
        app.Dialog.Warning().
            SetTitle("Low Disk Space").
            SetMessage(fmt.Sprintf("Only %d MB available.", available/(1024*1024))).
            Show()
    }
}
```

## エラーダイアログ

エラーを表示します:

```go
app.Dialog.Error().
    SetTitle("Error").
    SetMessage("Failed to connect to server.").
    Show()
```

**使用例:**
- エラーメッセージ
- 失敗通知
- 例外処理
- クリティカルな問題

**例 - ネットワークエラー:**

```go
func fetchData(app *application.App, url string) ([]byte, error) {
    resp, err := http.Get(url)
    if err != nil {
        app.Dialog.Error().
            SetTitle("Network Error").
            SetMessage(fmt.Sprintf("Failed to connect: %v", err)).
            Show()
        return nil, err
    }
    defer resp.Body.Close()

    return io.ReadAll(resp.Body)
}
```

## 質問ダイアログ

ユーザーに質問をし、ボタンコールバックを通じて応答を処理します:

```go
dialog := app.Dialog.Question().
    SetTitle("Confirm").
    SetMessage("Save changes before closing?")

save := dialog.AddButton("Save")
save.OnClick(func() {
    saveChanges()
})

dontSave := dialog.AddButton("Don't Save")
dontSave.OnClick(func() {
    // 保存せずに続行
})

cancel := dialog.AddButton("Cancel")
cancel.OnClick(func() {
    // 閉じない
})

dialog.SetDefaultButton(save)
dialog.SetCancelButton(cancel)
dialog.Show()
```

**使用例:**
- アクションの確認
- はい/いいえの質問
- 複数選択
- ユーザーの判断

**例 - 未保存の変更:**

```go
func closeDocument(app *application.App) {
    if !hasUnsavedChanges() {
        doClose()
        return
    }

    dialog := app.Dialog.Question().
        SetTitle("Unsaved Changes").
        SetMessage("Do you want to save your changes?")

    save := dialog.AddButton("Save")
    save.OnClick(func() {
        if saveDocument() {
            doClose()
        }
    })

    dontSave := dialog.AddButton("Don't Save")
    dontSave.OnClick(func() {
        doClose()
    })

    cancel := dialog.AddButton("Cancel")
    // キャンセルボタンにはコールバックなし - ダイアログを閉じるだけ

    dialog.SetDefaultButton(save)
    dialog.SetCancelButton(cancel)
    dialog.Show()
}
```

## ダイアログのオプション

### タイトルとメッセージ

```go
dialog := app.Dialog.Info().
    SetTitle("Operation Complete").
    SetMessage("All files have been processed successfully.")
```

**ベストプラクティス:**
- **タイトル:** 短く、説明的（2〜5語）
- **メッセージ:** 明確で具体的、実行可能
- **専門用語を避ける:** 平易な言語を使用

### ボタン

**単一ボタン（情報/警告/エラー）:**

情報、警告、エラーダイアログはデフォルトで「OK」ボタンを表示します:

```go
app.Dialog.Info().
    SetMessage("Done!").
    Show()
```

カスタムボタンを追加することもできます:

```go
dialog := app.Dialog.Info().
    SetMessage("Done!")
ok := dialog.AddButton("Got it!")
dialog.SetDefaultButton(ok)
dialog.Show()
```

**複数ボタン（質問）:**

`AddButton()` を使用してボタンを追加し、返される `*Button` を構成できます:

```go
dialog := app.Dialog.Question().
    SetMessage("Choose an action")

option1 := dialog.AddButton("Option 1")
option1.OnClick(func() {
    handleOption1()
})

option2 := dialog.AddButton("Option 2")
option2.OnClick(func() {
    handleOption2()
})

option3 := dialog.AddButton("Option 3")
option3.OnClick(func() {
    handleOption3()
})

dialog.Show()
```

**デフォルトボタンとキャンセルボタン:**

`SetDefaultButton()` を使用して、ハイライトされ Enter キーでトリガーされるボタンを指定します。
`SetCancelButton()` を使用して、Escape キーでトリガーされるボタンを指定します。

```go
dialog := app.Dialog.Question().
    SetMessage("Delete file?")

deleteBtn := dialog.AddButton("Delete")
deleteBtn.OnClick(func() {
    performDelete()
})

cancelBtn := dialog.AddButton("Cancel")
// コールバックは不要 - ダイアログを閉じるだけ

dialog.SetDefaultButton(cancelBtn)  // デフォルトとして安全なオプション
dialog.SetCancelButton(cancelBtn)   // Escape でキャンセルをトリガー
dialog.Show()
```

ボタンで流暢な `SetAsDefault()` および `SetAsCancel()` メソッドを使用することもできます:

```go
dialog := app.Dialog.Question().
    SetMessage("Delete file?")

dialog.AddButton("Delete").OnClick(func() {
    performDelete()
})

dialog.AddButton("Cancel").SetAsDefault().SetAsCancel()

dialog.Show()
```

**ベストプラクティス:**
- **1〜3つのボタン:** ユーザーを混乱させない
- **明確なラベル:** 「OK」ではなく「保存」
- **安全なデフォルト:** 破壊的でないアクション
- **順序が重要:** 最も可能性の高いアクションを最初に表示（キャンセルを除く）

### カスタムアイコン

ダイアログにカスタムアイコンを設定します:

```go
app.Dialog.Info().
    SetTitle("Custom Icon Example").
    SetMessage("Using a custom icon").
    SetIcon(myIconBytes).
    Show()
```

### ウィンドウへのアタッチ

特定のウィンドウにアタッチします:

```go
dialog := app.Dialog.Question().
    SetMessage("Window-specific question").
    AttachToWindow(window)

dialog.AddButton("OK")
dialog.Show()
```

**利点:**
- ダイアログが正しいウィンドウに表示される
- 表示中は親ウィンドウが無効化される
- マルチウィンドウ UX の向上

## 完全な例

### 破壊的アクションの確認

```go
func deleteFiles(app *application.App, paths []string) {
    // 削除の確認
    message := fmt.Sprintf("Delete %d file(s)?", len(paths))
    if len(paths) == 1 {
        message = fmt.Sprintf("Delete %s?", filepath.Base(paths[0]))
    }

    dialog := app.Dialog.Question().
        SetTitle("Confirm Delete").
        SetMessage(message)

    deleteBtn := dialog.AddButton("Delete")
    deleteBtn.OnClick(func() {
        // 削除を実行
        var errs []error
        for _, path := range paths {
            if err := os.Remove(path); err != nil {
                errs = append(errs, err)
            }
        }

        // 結果を表示
        if len(errs) > 0 {
            app.Dialog.Error().
                SetTitle("Delete Failed").
                SetMessage(fmt.Sprintf("Failed to delete %d file(s)", len(errs))).
                Show()
        } else {
            app.Dialog.Info().
                SetTitle("Delete Complete").
                SetMessage(fmt.Sprintf("Deleted %d file(s)", len(paths))).
                Show()
        }
    })

    cancelBtn := dialog.AddButton("Cancel")
    dialog.SetDefaultButton(cancelBtn)
    dialog.SetCancelButton(cancelBtn)
    dialog.Show()
}
```
