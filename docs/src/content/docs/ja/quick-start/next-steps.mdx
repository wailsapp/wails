## 機能

Wails はネイティブなデスクトップ機能を提供します：

- [Windows](/features/windows/basics) - 複数ウィンドウ、フレームレスウィンドウ、配置
- [メニュー](/features/menus/application) - アプリケーションメニュー、コンテキストメニュー、システムトレイ
- [ダイアログ](/features/dialogs/file) - ファイルの開く/保存、メッセージダイアログ
- [イベント](/features/events/system) - コンポーネント間の通信
- [バインディング](/features/bindings/methods) - 型安全な Go ↔ JavaScript 呼び出し
- [クリップボード](/features/clipboard) - コピー/ペースト操作
- [ドラッグ & ドロップ](/features/drag-and-drop/files) - ファイルのドラッグ & ドロップ
- [キーボード](/features/keyboard) - グローバルショートカット
