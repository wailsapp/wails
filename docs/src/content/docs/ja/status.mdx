## 現在のステータス: アルファ

最新の状態については [変更履歴](./changelog) をご確認ください。

私たちの目標はベータステータスに到達することです。このロードマップでは、ベータへの移行前に実装する必要がある主要な機能と改善点を概説しています。これは生文書であり、優先順位の変更や新たな知見の出現に応じて更新される可能性があることにご注意ください。

## 貢献方法

- 最新のアルファ版をテストし、問題があれば報告する
- ドキュメントや例の作成に貢献する
- ディスカッションに参加し、提案された機能についてフィードバックを提供する
- バグ修正や新機能のためのプルリクエストを提出する

コミュニティからの貢献を歓迎しています。これらの目標のいずれかについてお手伝いいただける場合や、ロードマップに関する提案がある場合は、Issue を作成するかコミュニティのディスカッションにご参加ください。

## フィードバックと更新

このロードマップは、コミュニティからのフィードバックやプロジェクトの優先順位に基づいて変更される場合があります。進捗状況や方向性の変更を反映するため、定期的に更新していきます。あなたの意見は Wails の未来を形作る上で非常に重要です。このロードマップは生文書であり、変更される可能性があります。ご提案がある場合は、Issue を作成してください。各マイルストーンには、達成を目指す目標のセットが設定されています。これらも変更される可能性があります。
