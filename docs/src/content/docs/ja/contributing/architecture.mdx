Wails v3 は、Go ランタイム、JavaScript ブリッジ、タスク駆動型ツールチェーン、およびモダンな Web 技術を活用したネイティブアプリケーションをリリースするためのテンプレート群で構成される**フルスタックのデスクトップフレームワーク**です。

このページでは、以下の 4 つの図解を通じて全体像を示します：

1. **全体アーキテクチャ** – 各サブシステムがどのように接続されているか
2. **ランタイムフロー** – JS が Go を呼び出し、その逆の場合に何が起こるか
3. **開発環境と本番環境** – アセットサーバーの 2 つのモード
4. **プラットフォーム実装** – OS 固有のコードがどこに存在するか

重要なポイント：

* **HTTP / IPC なし** – ブリッジはネイティブ WebView のメモリ内チャネルを使用します
* **メソッド ID** – 決定論的な FNV ハッシュにより、Go 側で O(1) のルックアップが可能になります

**[プラットフォーム分割図のプレースホルダー]**

すべての機能は以下のパターンに従います：

1. `pkg/application` 内の**共通インターフェース**
2. `pkg/application/messageprocessor_*.go` 内の**メッセージプロセッサ**のエントリ
3. ビルドタグで保護された `internal/runtime/*.go` 配下の **OS 別実装**

ある OS で機能が不足している場合は、`ErrCapability` を返し、`internal/capabilities` 経由で利用可能性を登録する必要があります。
