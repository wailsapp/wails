[Peirato's Piano Keyboard](https://github.com/Peiratooo/Peirato-s-Piano-Keyboard) is a lightweight desktop piano keyboard built with Wails. It is designed as a compact performance and practice tool that can stay on your desktop while you play, learn, record, stream, or demonstrate music.

The app supports real-time performance using a computer keyboard, an external MIDI piano keyboard, or mouse input. Notes are reflected instantly on the virtual keyboard, making it useful both as a playable instrument and as a visual keyboard display.

Beyond simple playback, Peirato's Piano Keyboard includes SoundFont support, allowing users to import and switch custom `.sf2` sound sources. It can also import MIDI files for playback, previewing, looped practice, and follow-along training. MIDI interaction is bidirectional, so the app can respond to external MIDI input while also working with MIDI playback and output workflows.

Key features include:

- Real-time desktop piano keyboard performance
- Computer keyboard, mouse, and external MIDI keyboard input
- MIDI input and output support
- Custom `.sf2` SoundFont import and management
- MIDI file import, playback, preview, and practice
- Follow-along MIDI learning mode
- Visual key highlighting for performance and practice
- Lightweight Wails-based desktop experience
