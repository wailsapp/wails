## Visão geral

A CLI do Wails (`wails3`) fornece comandos para criar, desenvolver, compilar e gerenciar aplicações Wails.

## Em breve

Esta seção está em construção. Por enquanto, use `wails3 --help` ou `wails3 [command] --help` para a documentação dos comandos.

**Comandos comuns:**

```bash
wails3 init          # Create a new project
wails3 dev           # Run in development mode
wails3 build         # Build for production
wails3 generate      # Generate bindings or runtime
wails3 doctor        # Check development environment
```
