## Visão Geral

A segurança é crítica para aplicativos de desktop. Siga estas práticas para manter seu aplicativo seguro.

## Validação de Entrada

### Sempre Valide

### Sanitizar HTML

```go
import "html"

func (s *Service) SaveComment(text string) error {
```

## Autenticação

### Armazenamento Seguro de Senhas

```go
import "golang.org/x/crypto/bcrypt"

func hashPassword(password string) (string, error) {
    hash, err := bcrypt.GenerateFromPassword([]byte(password), bcrypt.DefaultCost)
    return string(hash), err
}

func verifyPassword(hash, password string) bool {
    err := bcrypt.CompareHashAndPassword([]byte(hash), []byte(password))
    return err == nil
}
```

```go
type Session struct {

func (a *AuthService) CreateSession(userID int) (*Session, error) {
```

## Proteção de Dados

### Criptografar Dados Sensíveis

```go
import "crypto/aes"
import "crypto/cipher"

func encrypt(data []byte, key []byte) ([]byte, error) {
    block, err := aes.NewCipher(key)
    if err != nil {
        return nil, err
    }
    
    gcm, err := cipher.NewGCM(block)
    if err != nil {
        return nil, err
    }
    
    nonce := make([]byte, gcm.NonceSize())
    return gcm.Seal(nonce, nonce, data, nil), nil
}
```

## Segurança de Rede

### Usar HTTPS

```go
func makeAPICall(url string) (*Response, error) {
    // Always use HTTPS
    if !strings.HasPrefix(url, "https://") {
        return nil, errors.New("only HTTPS allowed")
    }
    
    return http.Get(url)
}
```

### Verificar Certificados

```go
import "crypto/tls"

func secureClient() *http.Client {
    return &http.Client{
        Transport: &http.Transport{
            TLSClientConfig: &tls.Config{
                MinVersion: tls.VersionTLS12,
            },
        },
    }
}
```

```go
func readFile(path string) ([]byte, error) {
```

## Limitação de Taxa

```go
type RateLimiter struct {
    requests map[string][]time.Time
    mu       sync.Mutex
    limit    int
    window   time.Duration
}

func (r *RateLimiter) Allow(key string) bool {
    r.mu.Lock()
    defer r.mu.Unlock()
    
    now := time.Now()
    
    // Clean old requests
    var recent []time.Time
    for _, t := range r.requests[key] {
        if now.Sub(t) < r.window {
            recent = append(recent, t)
        }
    }
    
    if len(recent) >= r.limit {
        return false
    }
    
    r.requests[key] = append(recent, now)
    return true
}
```

## Melhores Práticas

### ✅ Faça

- Valide todas as entradas
- Use HTTPS para chamadas de rede
- Criptografe dados sensíveis
- Use hash seguro de senhas
- Implemente limitação de taxa
- Mantenha as dependências atualizadas
- Registre eventos de segurança
- Use keychains do sistema operacional

### ❌ Não Faça

- Não confie na entrada do usuário
- Não armazene senhas em texto puro
- Não codifique segredos no código
- Não pule a verificação de certificados
- Não exponha dados sensíveis nos logs
- Não use criptografia fraca
- Não ignore atualizações de segurança

## Checklist de Segurança

- [ ] Todas as entradas de usuário validadas
- [ ] Senhas hash com bcrypt
- [ ] Dados sensíveis criptografados
- [ ] HTTPS usado para todas as chamadas de rede
- [ ] Limitação de taxa implementada
- [ ] Caminhos de arquivo validados
- [ ] Dependências atualizadas
- [ ] Log de segurança habilitado
- [ ] Mensagens de erro não vazam informações
- [ ] Código revisado quanto a vulnerabilidades
