Neste tutorial, você irá construir um aplicativo de notas que demonstra operações de arquivo, diálogos nativos e padrões modernos de aplicativos de desktop usando o Wails v3.

![Captura de Tela do App de Notas](../../../../assets/notes-app.png)

## O que você irá construir

- Um aplicativo de notas completo com funcionalidades de criar, editar e excluir
- Diálogos nativos de salvar/abrir arquivos para importar e exportar notas
- Salvamento automático ao digitar com debounce para reduzir atualizações desnecessárias
- Layout profissional de duas colunas (barra lateral + editor) que imita o Apple Notes

## O que você irá aprender

- Usando diálogos de arquivo nativos no Wails (`SaveFileDialog`, `OpenFileDialog`, `InfoDialog`)
- Trabalhando com arquivos JSON para persistência de dados
- Implementando padrões de auto-save com debounce
- Construindo interfaces de desktop profissionais com CSS moderno
- Serialização JSON correta de structs Go

2. **Crie o NotesService**

   Crie um novo arquivo `notesservice.go` na raiz do projeto:

   ```go
   package main

   import (
   	"encoding/json"
   	"errors"
   	"os"
   	"time"

   	"github.com/wailsapp/wails/v3/pkg/application"
   )

   type Note struct {
   	ID        string    `json:"id"`
   	Title     string    `json:"title"`
   	Content   string    `json:"content"`
   	CreatedAt time.Time `json:"createdAt"`
   	UpdatedAt time.Time `json:"updatedAt"`
   }

   type NotesService struct {
   	notes []Note
   }

   func NewNotesService() *NotesService {
   	return &NotesService{
   		notes: make([]Note, 0),
   	}
   }

   // GetAll retorna todas as notas
   func (n *NotesService) GetAll() []Note {
   	return n.notes
   }

   // Create cria uma nova nota
   func (n *NotesService) Create(title, content string) Note {
   	note := Note{
   		ID:        generateID(),
   		Title:     title,
   		Content:   content,
   		CreatedAt: time.Now(),
   		UpdatedAt: time.Now(),
   	}
   	n.notes = append(n.notes, note)
   	return note
   }

   // Update atualiza uma nota existente
   func (n *NotesService) Update(id, title, content string) error {
   	for i := range n.notes {
   		if n.notes[i].ID == id {
   			n.notes[i].Title = title
   			n.notes[i].Content = content
   			n.notes[i].UpdatedAt = time.Now()
   			return nil
   		}
   	}
   	return errors.New("note not found")
   }

   // Delete exclui uma nota
   func (n *NotesService) Delete(id string) error {
   	for i := range n.notes {
   		if n.notes[i].ID == id {
   			n.notes = append(n.notes[:i], n.notes[i+1:]...)
   			return nil
   		}
   	}
   	return errors.New("note not found")
   }

   // SaveToFile salva as notas em um arquivo
   func (n *NotesService) SaveToFile() error {
   	path, err := application.Get().Dialog.SaveFile().
   		SetFilename("notes.json").
   		AddFilter("JSON Files", "*.json").
   		PromptForSingleSelection()

   	if err != nil {
   		return err
   	}

   	data, err := json.MarshalIndent(n.notes, "", "  ")
   	if err != nil {
   		return err
   	}

   	if err := os.WriteFile(path, data, 0644); err != nil {
   		return err
   	}

   	application.Get().Dialog.Info().
   		SetTitle("Success").
   		SetMessage("Notes saved successfully!").
   		Show()

   	return nil
   }

   // LoadFromFile carrega as notas de um arquivo
   func (n *NotesService) LoadFromFile() error {
   	path, err := application.Get().Dialog.OpenFile().
   		AddFilter("JSON Files", "*.json").
   		PromptForSingleSelection()

   	if err != nil {
   		return err
   	}

   	data, err := os.ReadFile(path)
   	if err != nil {
   		return err
   	}

   	var notes []Note
   	if err := json.Unmarshal(data, &notes); err != nil {
   		return err
   	}

   	n.notes = notes

   	application.Get().Dialog.Info().
   		SetTitle("Success").
   		SetMessage("Notes loaded successfully!").
   		Show()

   	return nil
   }

   func generateID() string {
   	return time.Now().Format("20060102150405")
   }
   ```

   **O que está acontecendo aqui:**

   - **Struct Note**: Define a estrutura de dados com tags JSON (minúsculas) para serialização correta
   - **Operações CRUD**: GetAll, Create, Update e Delete para gerenciar notas na memória
   - **Diálogos de arquivo**: Usa `application.Get().Dialog.SaveFile()` e `application.Get().Dialog.OpenFile()` para acessar diálogos nativos
   - **Diálogos de informação**: Exibe mensagens de sucesso usando `application.Get().Dialog.Info()`
   - **Geração de ID**: Gerador de ID simples baseado em timestamp

3. **Atualize o main.go**

   Substitua o conteúdo de `main.go`:

   ```go
   package main

   import (
   	"embed"
   	_ "embed"
   	"log"

   	"github.com/wailsapp/wails/v3/pkg/application"
   )

   //go:embed all:frontend/dist
   var assets embed.FS

   func main() {
   	app := application.New(application.Options{
   		Name:        "Notes App",
   		Description: "A simple notes application",
   		Services: []application.Service{
   			application.NewService(NewNotesService()),
   		},
   		Assets: application.AssetOptions{
   			Handler: application.AssetFileServerFS(assets),
   		},
   		Mac: application.MacOptions{
   			ApplicationShouldTerminateAfterLastWindowClosed: true,
   		},
   	})

   	app.Window.NewWithOptions(application.WebviewWindowOptions{
   		Title:            "Notes App",
   		Width:            1000,
   		Height:           700,
   		BackgroundColour: application.NewRGB(255, 255, 255),
   		URL:              "/",
   	})

   	err := app.Run()
   	if err != nil {
   		log.Fatal(err)
   	}
   }
   ```

   **O que está acontecendo aqui:**

   - Registra o `NotesService` com a aplicação
   - Cria uma janela com dimensões (1000x700) que imita o Apple Notes
   - Configura o comportamento adequado do macOS para encerrar quando a última janela for fechada

6. **Implemente a lógica em JavaScript**

   Substitua `frontend/src/main.js`:

   ```javascript
   import { NotesService } from '../bindings/changeme'

   let notes = []
   let currentNote = null

   // Load notes on startup
   async function loadNotes() {
       notes = await NotesService.GetAll()
       renderNotesList()
   }

   // Render notes list
   function renderNotesList() {
       const notesList = document.getElementById('notes-list')

       if (notes.length === 0) {
           notesList.innerHTML = '<div style="padding: 20px; text-align: center; color: #999;">No notes yet</div>'
           return
       }

       notesList.innerHTML = notes.map(note => `
           <div class="note-item ${currentNote?.id === note.id ? 'active' : ''}" data-id="${note.id}">
               <h3>${note.title || 'Untitled'}</h3>
               <p>${note.content || 'No content'}</p>
           </div>
       `).join('')

       // Add click handlers
       document.querySelectorAll('.note-item').forEach(item => {
           item.addEventListener('click', () => {
               const id = item.dataset.id
               selectNote(id)
           })
       })
   }

   // Select a note
   function selectNote(id) {
       currentNote = notes.find(n => n.id === id)
       if (currentNote) {
           document.getElementById('empty-state').style.display = 'none'
           document.getElementById('note-editor').style.display = 'flex'
           document.getElementById('note-title').value = currentNote.title
           document.getElementById('note-content').value = currentNote.content
           document.getElementById('last-updated').textContent =
               `Last updated: ${new Date(currentNote.updatedAt).toLocaleString()}`
           renderNotesList()
       }
   }

   // Create new note
   document.getElementById('new-note-btn').addEventListener('click', async () => {
       const note = await NotesService.Create('Untitled', '')
       notes.push(note)
       selectNote(note.id)
       // Focus the title input and select all text so user can immediately type
       const titleInput = document.getElementById('note-title')
       titleInput.focus()
       titleInput.select()
   })

   // Update note on input
   let updateTimeout
   function scheduleUpdate() {
       clearTimeout(updateTimeout)
       updateTimeout = setTimeout(async () => {
           if (currentNote) {
               const title = document.getElementById('note-title').value
               const content = document.getElementById('note-content').value

               await NotesService.Update(currentNote.id, title, content)

               // Update local copy
               const note = notes.find(n => n.id === currentNote.id)
               if (note) {
                   note.title = title
                   note.content = content
                   note.updatedAt = new Date().toISOString()
               }

               renderNotesList()
               document.getElementById('last-updated').textContent =
                   `Last updated: ${new Date().toLocaleString()}`
           }
       }, 500)
   }

   document.getElementById('note-title').addEventListener('input', scheduleUpdate)
   document.getElementById('note-content').addEventListener('input', scheduleUpdate)

   // Delete note
   document.getElementById('delete-btn').addEventListener('click', async () => {
       if (!currentNote) return

       try {
           await NotesService.Delete(currentNote.id)
           notes = notes.filter(n => n.id !== currentNote.id)
           currentNote = null
           document.getElementById('empty-state').style.display = 'flex'
           document.getElementById('note-editor').style.display = 'none'
           renderNotesList()
       } catch (error) {
           console.error('Delete failed:', error)
       }
   })

   // Save to file
   document.getElementById('save-btn').addEventListener('click', async () => {
       try {
           await NotesService.SaveToFile()
       } catch (error) {
           if (error) console.error('Save failed:', error)
       }
   })// Carregar do arquivo
   document.getElementById('load-btn').addEventListener('click', async () => {
       try {
           await NotesService.LoadFromFile()
           notes = await NotesService.GetAll()
           currentNote = null
           document.getElementById('empty-state').style.display = 'flex'
           document.getElementById('note-editor').style.display = 'none'
           renderNotesList()
       } catch (error) {
           if (error) console.error('Falha ao carregar:', error)
       }
   })

   // Inicializar
   loadNotes()
   ```

   **O que está acontecendo aqui:**

   - **Salvamento automático**: um debounce de 500ms evita chamadas excessivas ao backend enquanto o usuário digita
   - **Acesso às propriedades**: usa nomes de propriedades em minúsculas (`.id`, `.title`) correspondentes às tags JSON do Go
   - **Gerenciamento de foco**: foca automaticamente e seleciona o título ao criar novas notas
   - **Exclusão sem confirmação**: o `confirm()` do navegador não funciona nas webviews do Wails
   - **Operações de arquivo**: diálogos nativos lidam com salvar/carregar com tratamento de erros adequado

## Conceitos Chave

### Funções de Diálogo em Nível de Pacote

No Wails v3, os diálogos de arquivo e mensagem são funções em nível de pacote, não métodos na instância do aplicativo:

```go
// Correto - função em nível de pacote
path, err := application.SaveFileDialog().
    SetFilename("notes.json").
    AddFilter("Arquivos JSON", "*.json").
    PromptForSingleSelection()

// Errado - não disponível
path, err := app.Dialog.SaveFile() // Isso não existe
```

### Mapeamento de Tags JSON

As tags JSON das structs do Go devem estar em minúsculas para corresponder ao acesso de propriedades no JavaScript:

```go
type Note struct {
    ID string `json:"id"` // Deve estar em minúsculas
}
```

```javascript
// JavaScript acessa com minúsculas
const noteId = note.id // Correto
const noteId = note.ID // Seria undefined
```

### Salvamento Automático com Debounce

O debounce de 500ms reduz chamadas desnecessárias ao backend:

```javascript
let updateTimeout
function scheduleUpdate() {
    clearTimeout(updateTimeout) // Cancela o timer anterior
    updateTimeout = setTimeout(async () => {
        // Salva apenas se o usuário parar de digitar por 500ms
        await NotesService.Update(currentNote.id, title, content)
    }, 500)
}
```
