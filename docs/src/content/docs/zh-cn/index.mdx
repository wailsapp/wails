## 快速开始

<Tabs syncKey="os">
  <TabItem label="macOS" icon="apple">
```bash
# Install Wails
go install github.com/wailsapp/wails/v3/cmd/wails3@latest

# Create your application
wails3 init -n myapp -t vanilla

# Run with hot reload
cd myapp && wails3 dev
```
  </TabItem>
  
  <TabItem label="Windows" icon="seti:windows">
```powershell
# Install Wails
go install github.com/wailsapp/wails/v3/cmd/wails3@latest

# Create your application
wails3 init -n myapp -t vanilla

# Run with hot reload
cd myapp; wails3 dev
```
  </TabItem>
  
  <TabItem label="Linux" icon="linux">
```bash
# Install Wails
go install github.com/wailsapp/wails/v3/cmd/wails3@latest

# Create your application
wails3 init -n myapp -t vanilla

# Run with hot reload
cd myapp && wails3 dev
```
  </TabItem>
</Tabs>

**您的应用现已运行**，支持热重载和类型安全的 Go 到 JS 绑定。


## 为什么选择 Wails？

<CardGrid>
  <Card title="用户可感知的性能" icon="rocket">
    - 约 15MB 的二进制文件，对比 Electron 的 150MB
    - 约 10MB 的基础内存占用，对比 100MB+
    - 启动时间 &lt;0.5 秒，对比 2-3 秒
    - 使用系统 WebView 进行原生渲染
    - 无内置浏览器开销
  </Card>
  
  <Card title="开发者体验" icon="setting">
    - 一套 Go 代码库适用于所有平台
    - 支持任意 Web 框架 - React, Vue, Svelte
    - 开发期间支持热重载
    - 自动生成绑定，轻松从 JavaScript 调用 Go
    - 内存内 IPC。无需网络端口
  </Card>
  
  <Card title="生产就绪" icon="approve-check">
    - 支持多窗口及其生命周期管理
    - 原生菜单和系统托盘
    - 平台原生文件对话框
    - 系统集成与快捷键支持
    - 代码签名与打包工具
  </Card>
  
  <Card title="跨平台原生体验" icon="laptop">
    - 一套代码适用于 Windows, macOS, Linux
    - 按需使用平台特定功能
    - 用户体验毫不妥协
    - 一次构建，部署至所有平台
    - 移动端即将推出...
  </Card>
</CardGrid>
