## API 约定

<details>
<summary><strong>Go API 约定</strong> - 适用于 Go 新手开发者</summary>

#### 命名

- **类型**: PascalCase（例如 `WebviewWindow`）
- **方法**: PascalCase（例如 `SetTitle()`）
- **选项**: PascalCase 结构体（例如 `WindowOptions`）
- **常量**: PascalCase（例如 `WindowStartStateMaximised`）

#### 错误处理

大多数可能失败的方法将 `error` 作为最后一个返回值返回：

```go
window, err := app.Window.New()
if err != nil {
    log.Fatal(err)
}
```

#### 上下文 (Context)

可能阻塞或需要取消的方法接受 `context.Context`：

```go
err := app.RunWithContext(ctx)
```

#### 选项模式

配置使用选项结构体：

```go
app := application.New(application.Options{
    Name: "My App",
    Description: "A demo application",
    Services: []application.Service{
        application.NewService(&MyService{}),
    },
})
```

</details>

### JavaScript API 约定

#### 命名

- **函数**: camelCase（例如 `setTitle()`）
- **常量**: SCREAMING_SNAKE_CASE（例如 `WINDOW_EVENT_FOCUS`）

#### 默认异步

所有 Go 方法调用均返回 Promises：

```javascript
// Async/await（推荐）
const result = await MyService.DoSomething()

// Promise 链
MyService.DoSomething()
    .then(result => console.log(result))
    .catch(error => console.error(error))
```

#### 错误处理

Go 错误会转换为 JavaScript 异常：

```javascript
try {
    await MyService.MightFail()
} catch (error) {
    console.error('Go error:', error)
}
```

#### 类型安全

TypeScript 定义是自动生成的：

```typescript
// 完全类型化
import { Greet } from './bindings/GreetService'

const message: string = await Greet("World")
```

## 平台差异

某些 API 在不同平台上的行为有所不同：

| 功能 | Windows | macOS | Linux |
|---------|---------|-------|-------|
| **应用程序菜单** | 窗口菜单栏 | 全局菜单栏 | 窗口菜单栏 |
| **系统托盘** | 通知区域 | 菜单栏 | 系统托盘 |
| **Dock** | 不适用 | ✅ 可用 | 不适用 |
| **文件对话框** | 原生 | 原生 | 原生 (GTK) |
| **透明度** | ✅ 完全支持 | ✅ 完全支持 | ⚠️ 有限支持 |

特定于平台的行为在每个 API 部分中均有文档说明。

## 版本控制

Wails v3 遵循语义化版本控制：

- **主版本** (v3.x.x): 破坏性变更
- **次版本** (v3.x.x): 新功能，向后兼容
- **补丁版本** (v3.x.x): 错误修复，向后兼容

**当前状态：** Alpha（API 稳定，持续优化中）

## 弃用策略

当 API 被弃用时：

1. **在文档中标记** 弃用通知
2. **提供替代方案** 及迁移指南
3. **在移除前维护 1 个主版本**
4. **编译器警告**（如可能）

## API 稳定性

### 稳定 API ✅

这些 API 是稳定的，可用于生产环境：

- 核心应用程序 API
- 窗口管理
- 菜单系统
- 事件系统
- 文件对话框
- 服务绑定

### 不稳定 API ⚠️

这些 API 在最终发布前可能会发生变化：

- 某些高级窗口选项
- 特定于平台的功能
- 实验性功能

不稳定 API 在文档中均有标记。
