你已经构建了第一个 Wails 应用并了解了基础知识。以下是接下来的学习方向。

## 深入学习

通过构建完整的应用程序来练习：

- [待办事项应用](/tutorials/02-todo-vanilla) - CRUD 操作、状态管理、现代 UI
- [所有教程](/tutorials/overview) - 笔记应用、仪表盘、系统托盘应用等

## 功能特性

Wails 提供原生桌面能力：

- [Windows](/features/windows/basics) - 多窗口、无边框窗口、窗口定位
- [菜单](/features/menus/application) - 应用菜单、上下文菜单、系统托盘
- [对话框](/features/dialogs/file) - 文件打开/保存、消息对话框
- [事件](/features/events/system) - 组件间通信
- [绑定](/features/bindings/methods) - 类型安全的 Go ↔ JavaScript 调用
- [剪贴板](/features/clipboard) - 复制/粘贴操作
- [拖放](/features/drag-and-drop/files) - 文件拖放
- [键盘](/features/keyboard) - 全局快捷键
