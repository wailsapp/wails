## Введение

Wails предоставляет кроссплатформенный сервис Dock для десктопных приложений. Этот сервис позволяет:

- Скрывать и показывать значок приложения в доке macOS
- Отображать бейджи на плитке приложения или значке в доке/панели задач (macOS и Windows)

## Базовое использование

### Создание сервиса

Сначала инициализируйте сервис dock:

```go
import "github.com/wailsapp/wails/v3/pkg/application"
import "github.com/wailsapp/wails/v3/pkg/services/dock"

// Создаем новый сервис Dock
dockService := dock.New()

// Регистрируем сервис в приложении
app := application.New(application.Options{
    Services: []application.Service{
        application.NewService(dockService),
    },
})
```

### Создание сервиса с пользовательскими параметрами бейджа (только Windows)

В Windows можно настроить внешний вид бейджа с помощью различных параметров:

```go
import "github.com/wailsapp/wails/v3/pkg/application"
import "github.com/wailsapp/wails/v3/pkg/services/dock"
import "image/color"

// Создаем сервис dock с пользовательскими параметрами бейджа
options := dock.BadgeOptions{
    TextColour:       color.RGBA{255, 255, 255, 255}, // Белый текст
    BackgroundColour: color.RGBA{0, 0, 255, 255},     // Синий фон
    FontName:         "consolab.ttf",                 // Жирный шрифт Consolas
    FontSize:         20,                             // Размер шрифта для одного символа
    SmallFontSize:    14,                             // Размер шрифта для нескольких символов
}

dockService := dock.NewWithOptions(options)

// Регистрируем сервис в приложении
app := application.New(application.Options{
    Services: []application.Service{
        application.NewService(dockService),
    },
})
```

## Операции с Dock

### Скрытие значка приложения в доке

Скройте значок приложения из дока macOS:

```go
// Скрыть значок приложения
dockService.HideAppIcon()
```

### Показ значка приложения в доке

Покажите значок приложения в доке macOS:

```go
// Показать значок приложения
dockService.ShowAppIcon()
```

## Операции с бейджами

### Установка бейджа

Установите бейдж на плитку/значок приложения:

```go
// Установить бейдж по умолчанию
dockService.SetBadge("")

// Установить числовой бейдж
dockService.SetBadge("3")

// Установить текстовый бейдж
dockService.SetBadge("New")
```

### Установка пользовательского бейджа (только Windows)

Установите бейдж с применением одноразовых параметров:

```go
options := dock.BadgeOptions{
    BackgroundColour: color.RGBA{0, 255, 255, 255},
    FontName:         "arialb.ttf", // Системный шрифт
    FontSize:         16,
    SmallFontSize:    10,
    TextColour:       color.RGBA{0, 0, 0, 255},
}

// Установить бейдж по умолчанию
dockService.SetCustomBadge("", options)

// Установить числовой бейдж
dockService.SetCustomBadge("3", options)

// Установить текстовый бейдж
dockService.SetCustomBadge("New", options)
```

### Удаление бейджа

Удалите бейдж с значка приложения:

```go
dockService.RemoveBadge()
```

### Получение установленного бейджа
```go
dockService.GetBadge()
``` 

## Особенности платформ

<Tabs>
  <TabItem label="macOS" icon="fa-brands:apple">
    
    На macOS:
    
    - Значок в доке можно **скрывать** и **показывать**
    - Бейджи отображаются непосредственно на значке в доке
    - Параметры бейджа **не настраиваются** (любые параметры, переданные в `NewWithOptions`/`SetCustomBadge`, игнорируются)
    - Используется стандартное оформление бейджей macOS, которое автоматически адаптируется к теме оформления
    - Переполнение метки обрабатывается системой
    - Пустая метка отображает бейдж по умолчанию в виде "●"
    
  </TabItem>
  
  <TabItem label="Windows" icon="fa-brands:windows">
    
    На Windows:
    
    - Скрытие/показ значка на панели задач в данный момент не поддерживается этим сервисом
    - Бейджи отображаются как наложенный значок на панели задач
    - Бейджи поддерживают текстовые значения
    - Внешний вид бейджа можно настроить через `BadgeOptions`
    - Для отображения бейджей у приложения должно быть окно
    - Для меток с несколькими символами автоматически используется уменьшенный размер шрифта
    - Переполнение метки не обрабатывается
    - Параметры настройки:
      - **TextColour**: Цвет текста (по умолчанию: белый)
      - **BackgroundColour**: Цвет фона бейджа (по умолчанию: красный)
      - **FontName**: Имя файла шрифта (по умолчанию: "segoeuib.ttf")
      - **FontSize**: Размер шрифта для одного символа (по умолчанию: 18)
      - **SmallFontSize**: Размер шрифта для нескольких символов (по умолчанию: 14)
    
  </TabItem>
  
  <TabItem label="Linux" icon="fa-brands:linux">
    
    На Linux:
    
    - Видимость значка в доке и функциональность бейджей недоступны
    
  </TabItem>
</Tabs>

## Лучшие практики

1. **При скрытии значка в доке (macOS):**
   - Убедитесь, что пользователи всё ещё могут получить доступ к вашему приложению (например, через [системный трей](https://v3.wails.io/learn/systray/))
   - Включите опцию "Выход" в альтернативном интерфейсе
   - Приложение не будет отображаться в переключателе приложений Command+Tab
   - Открытые окна остаются видимыми и функциональными
   - Закрытие всех окон может не завершить работу приложения (поведение macOS варьируется)
   - Пользователи теряют стандартный способ выхода через правый клик по значку в доке

2. **Используйте бейджи умеренно:**
   - Частое обновление бейджей может отвлекать пользователей
   - Оставляйте бейджи для важных уведомлений

3. **Делайте текст бейджа коротким:**
   - Числовые бейджи наиболее эффективны
   - На macOS текстовые бейджи должны быть краткими

4. **Для настройки бейджей в Windows:**
   - Обеспечьте высокий контраст между цветом текста и фона
   - Тестируйте с разной длиной текста, так как размер шрифта уменьшается с увеличением длины
   - Используйте распространенные системные шрифты для обеспечения их доступности


## Справочник API

### Управление сервисом
| Метод                                     | Описание                                           |
|--------------------------------------------|-------------------------------------------------------|
| `New()`                                    | Создает новый сервис dock                            |
| `NewWithOptions(options BadgeOptions)`     | Создает новый сервис dock с пользовательскими параметрами бейджа (только Windows; параметры игнорируются на macOS и Linux) |

### Операции с Dock
| Метод                                       | Описание                                                |
|--------------------------------|-------------------------------------------------------------|
| `HideAppIcon()`                | Скрывает значок приложения из дока macOS (только macOS)         |
| `ShowAppIcon()`                | Показывает значок приложения в доке macOS (только macOS)           |

### Операции с бейджами
| Метод                                            | Описание                                                |
|---------------------------------------------------|------------------------------------------------------------|
| `SetBadge(label string) error`                    | Устанавливает бейдж с указанной меткой                      |
| `SetCustomBadge(label string, options BadgeOptions) error` | Устанавливает бейдж с указанной меткой и пользовательскими параметрами оформления (только Windows) |
| `RemoveBadge() error`                             | Удаляет бейдж с значка приложения                |
| `GetBadge() *string`                             | Получает текущий бейдж                |

### Структуры и типы

```go
// Параметры для настройки внешнего вида бейджа (только Windows)
type BadgeOptions struct {
    TextColour       color.RGBA  // Цвет текста бейджа
    BackgroundColour color.RGBA  // Цвет фона бейджа
    FontName         string      // Имя файла шрифта (например, "segoeuib.ttf")
    FontSize         int         // Размер шрифта для одного символа
    SmallFontSize    int         // Размер шрифта для нескольких символов
}
```
