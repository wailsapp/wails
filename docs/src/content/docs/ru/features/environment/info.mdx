Wails предоставляет исчерпывающую информацию об окружении через API EnvironmentManager. Это позволяет вашему приложению определять свойства системы, предпочтения темы и интегрироваться с файловым менеджером операционной системы.

## Доступ к менеджеру окружения

Менеджер окружения доступен через свойство `Env` экземпляра вашего приложения:

```go
app := application.New(application.Options{
    Name: "Environment Demo",
})

// Доступ к менеджеру окружения
env := app.Env
```

## Информация о системе

### Получение информации об окружении

Получите исчерпывающую информацию о среде выполнения:

```go
envInfo := app.Env.Info()

app.Logger.Info("Environment information",
    "os", envInfo.OS,           // "windows", "darwin", "linux"
    "arch", envInfo.Arch,       // "amd64", "arm64", и т.д.
    "debug", envInfo.Debug,     // Флаг режима отладки
)

// Детали операционной системы
if envInfo.OSInfo != nil {
    app.Logger.Info("OS details",
        "name", envInfo.OSInfo.Name,
        "version", envInfo.OSInfo.Version,
    )
}

// Информация, специфичная для платформы
for key, value := range envInfo.PlatformInfo {
    app.Logger.Info("Platform info", "key", key, "value", value)
}
```

### Структура окружения

Информация об окружении включает несколько важных полей:

```go
type EnvironmentInfo struct {
    OS           string                 // Операционная система: "windows", "darwin", "linux"
    Arch         string                 // Архитектура: "amd64", "arm64", "386", и т.д.
    Debug        bool                   // Запущено ли в режиме отладки
    OSInfo       *operatingsystem.OS    // Подробная информация об ОС
    PlatformInfo map[string]any         // Детали, специфичные для платформы
}
```

## Определение темы

### Определение темного режима

Определите, использует ли система темный режим:

```go
if app.Env.IsDarkMode() {
    app.Logger.Info("System is in dark mode")
    // Примените темную тему к вашему приложению
    applyDarkTheme()
} else {
    app.Logger.Info("System is in light mode")
    // Примените светлую тему к вашему приложению
    applyLightTheme()
}
```

### Мониторинг изменений темы

Подписывайтесь на изменения темы, чтобы динамически обновлять ваше приложение:

```go
import "github.com/wailsapp/wails/v3/pkg/events"

// Подписка на изменения темы
app.Event.OnApplicationEvent(events.Common.ThemeChanged, func(event *application.ApplicationEvent) {
    if app.Env.IsDarkMode() {
        app.Logger.Info("Switched to dark mode")
        updateApplicationTheme("dark")
    } else {
        app.Logger.Info("Switched to light mode")
        updateApplicationTheme("light")
    }
})

func updateApplicationTheme(theme string) {
    // Обновите тему вашего приложения
    // Это может отправить событие во фронтенд
    app.Event.Emit("theme:changed", theme)
}
```

## Интеграция с файловым менеджером

### Открытие файлового менеджера

Откройте файловый менеджер системы в определенном месте:

```go
// Открыть файловый менеджер в директории
err := app.Env.OpenFileManager("/Users/username/Documents", false)
if err != nil {
    app.Logger.Error("Failed to open file manager", "error", err)
}

// Открыть файловый менеджер и выделить конкретный файл
err = app.Env.OpenFileManager("/Users/username/Documents/report.pdf", true)
if err != nil {
    app.Logger.Error("Failed to open file manager with selection", "error", err)
}
```

### Распространенные варианты использования

Показывайте файлы или папки в файловом менеджере из вашего приложения:

```go
func showInFileManager(app *application.App, path string) {
    err := app.Env.OpenFileManager(path, true)
    if err != nil {
        // Резервный вариант: попробуйте открыть только директорию
        dir := filepath.Dir(path)
        err = app.Env.OpenFileManager(dir, false)
        if err != nil {
            app.Logger.Error("Could not open file manager", "path", path, "error", err)
            
            // Покажите ошибку пользователю
            app.Dialog.Error().
                SetTitle("File Manager Error").
                SetMessage("Could not open file manager").
                Show()
        }
    }
}

// Примеры использования
func setupFileMenu(app *application.App) {
    menu := app.Menu.New()
    fileMenu := menu.AddSubmenu("File")
    
    fileMenu.Add("Show Downloads Folder").OnClick(func(ctx *application.Context) {
        homeDir, _ := os.UserHomeDir()
        downloadsDir := filepath.Join(homeDir, "Downloads")
        showInFileManager(app, downloadsDir)
    })
    
    fileMenu.Add("Show Application Data").OnClick(func(ctx *application.Context) {
        configDir, _ := os.UserConfigDir()
        appDir := filepath.Join(configDir, "MyApp")
        showInFileManager(app, appDir)
    })
}
```

## Диалог информации об окружении

### Отображение информации о системе

Создайте диалог, показывающий информацию об окружении:

```go
func showEnvironmentDialog(app *application.App) {
    envInfo := app.Env.Info()
    
    details := fmt.Sprintf(`Environment Information:

Operating System: %s
Architecture: %s
Debug Mode: %t

Dark Mode: %t

Platform Information:`, 
        envInfo.OS, 
        envInfo.Arch, 
        envInfo.Debug,
        app.Env.IsDarkMode())
    
    // Добавьте детали, специфичные для платформы
    for key, value := range envInfo.PlatformInfo {
        details += fmt.Sprintf("\n%s: %v", key, value)
    }    if envInfo.OSInfo != nil {
        details += fmt.Sprintf("\n\nOS Details:\nName: %s\nVersion: %s", 
            envInfo.OSInfo.Name, 
            envInfo.OSInfo.Version)
    }
    
    dialog := app.Dialog.Info()
    dialog.SetTitle("Environment Information")
    dialog.SetMessage(details)
    dialog.Show()
}
```

## Лучшие практики

1. **Кэшируйте информацию об окружении**: Информация об окружении редко меняется во время выполнения:
   ```go
   type App struct {
       envInfo *application.EnvironmentInfo
   }
   
   func (a *App) getEnvInfo() application.EnvironmentInfo {
       if a.envInfo == nil {
           info := a.app.Env.Info()
           a.envInfo = &info
       }
       return *a.envInfo
   }
   ```

2. **Обрабатывайте изменения темы**: Подписывайтесь на изменения системной темы:
   ```go
   app.Event.OnApplicationEvent(events.Common.ThemeChanged, func(event *application.ApplicationEvent) {
       updateTheme(app.Env.IsDarkMode())
   })
   ```

3. **Обеспечивайте корректную обработку ошибок файлового менеджера**: Всегда обрабатывайте ошибки файлового менеджера:
   ```go
   func openFileManagerSafely(app *application.App, path string) {
       err := app.Env.OpenFileManager(path, false)
       if err != nil {
           // Предоставьте запасной вариант или уведомление пользователя
           app.Logger.Warn("Could not open file manager", "path", path)
       }
   }
   ```

4. **Используйте специфичные для платформы функции**: Используйте информацию об окружении для включения функций платформы:
   ```go
   envInfo := app.Env.Info()
   if envInfo.OS == "darwin" {
       // Включение функций, специфичных для macOS
   }
   ```

## Полный пример

Ниже приведён полный пример, демонстрирующий управление окружением:

```go
package main

import (
    "fmt"
    "os"
    "path/filepath"
    "runtime"
    "github.com/wailsapp/wails/v3/pkg/application"
    "github.com/wailsapp/wails/v3/pkg/events"
)

func main() {
    app := application.New(application.Options{
        Name: "Environment Demo",
    })

    // Настройка приложения в зависимости от окружения
    setupForEnvironment(app)

    // Мониторинг изменений темы
    monitorThemeChanges(app)

    // Создание меню с функциями окружения
    setupEnvironmentMenu(app)

    // Создание главного окна
    window := app.Window.New()
    window.SetTitle("Environment Demo")

    err := app.Run()
    if err != nil {
        panic(err)
    }
}

func setupForEnvironment(app *application.App) {
    envInfo := app.Env.Info()
    
    app.Logger.Info("Application environment",
        "os", envInfo.OS,
        "arch", envInfo.Arch,
        "debug", envInfo.Debug,
        "darkMode", app.Env.IsDarkMode(),
    )

    // Настройка для платформы
    switch envInfo.OS {
    case "darwin":
        app.Logger.Info("Configuring for macOS")
        // Настройка, специфичная для macOS
    case "windows":
        app.Logger.Info("Configuring for Windows")
        // Настройка, специфичная для Windows
    case "linux":
        app.Logger.Info("Configuring for Linux")
        // Настройка, специфичная для Linux
    }

    // Применение начальной темы
    if app.Env.IsDarkMode() {
        applyDarkTheme(app)
    } else {
        applyLightTheme(app)
    }
}

func monitorThemeChanges(app *application.App) {
    app.Event.OnApplicationEvent(events.Common.ThemeChanged, func(event *application.ApplicationEvent) {
        if app.Env.IsDarkMode() {
            app.Logger.Info("System switched to dark mode")
            applyDarkTheme(app)
        } else {
            app.Logger.Info("System switched to light mode")
            applyLightTheme(app)
        }
    })
}

func setupEnvironmentMenu(app *application.App) {
    menu := app.Menu.New()
    
    // Добавление меню приложения, специфичного для платформы
    if runtime.GOOS == "darwin" {
        menu.AddRole(application.AppMenu)
    }
    
    // Меню инструментов
    toolsMenu := menu.AddSubmenu("Tools")
    
    toolsMenu.Add("Show Environment Info").OnClick(func(ctx *application.Context) {
        showEnvironmentInfo(app)
    })
    
    toolsMenu.Add("Open Downloads Folder").OnClick(func(ctx *application.Context) {
        openDownloadsFolder(app)
    })
    
    toolsMenu.Add("Toggle Theme").OnClick(func(ctx *application.Context) {
        // Обычно это обрабатывается системой,
        // но показано здесь для демонстрации
        toggleTheme(app)
    })
    
    app.Menu.Set(menu)
}

func showEnvironmentInfo(app *application.App) {
    envInfo := app.Env.Info()
    
    message := fmt.Sprintf(`Environment Information:

Operating System: %s
Architecture: %s
Debug Mode: %t
Dark Mode: %t

Platform Details:`,
        envInfo.OS,
        envInfo.Arch,
        envInfo.Debug,
        app.Env.IsDarkMode())
    
    for key, value := range envInfo.PlatformInfo {
        message += fmt.Sprintf("\n%s: %v", key, value)
    }
    
    if envInfo.OSInfo != nil {
        message += fmt.Sprintf("\n\nOS Information:\nName: %s\nVersion: %s",
            envInfo.OSInfo.Name,
            envInfo.OSInfo.Version)
    }
    
    dialog := app.Dialog.Info()
    dialog.SetTitle("Environment Information")
    dialog.SetMessage(message)
    dialog.Show()
}

func openDownloadsFolder(app *application.App) {
    homeDir, err := os.UserHomeDir()
    if err != nil {
        app.Logger.Error("Could not get home directory", "error", err)
        return
    }
    
    downloadsDir := filepath.Join(homeDir, "Downloads")
    err = app.Env.OpenFileManager(downloadsDir, false)
    if err != nil {
        app.Logger.Error("Could not open Downloads folder", "error", err)
        
        app.Dialog.Error().
            SetTitle("File Manager Error").
            SetMessage("Could not open Downloads folder").
            Show()
    }
}

func applyDarkTheme(app *application.App) {
    app.Logger.Info("Applying dark theme")
    // Передача изменения темы на фронтенд
    app.Event.Emit("theme:apply", "dark")
}

func applyLightTheme(app *application.App) {
    app.Logger.Info("Applying light theme")
    // Передача изменения темы на фронтенд
    app.Event.Emit("theme:apply", "light")
}

func toggleTheme(app *application.App) {
    // Это приведено только для демонстрации
    // Реальные изменения темы должны поступать от системы
    currentlyDark := app.Env.IsDarkMode()
    if currentlyDark {
        applyLightTheme(app)
    } else {
        applyDarkTheme(app)
    }
}:::tip[Совет]
Используйте информацию об окружении для обеспечения соответствующего пользовательского опыта на разных платформах. Например, используйте комбинации клавиш с клавишей Command на macOS и с клавишей Control на Windows/Linux.
:::

:::danger[Предупреждение]
Информация об окружении, как правило, остаётся стабильной во время выполнения приложения, но предпочтения темы могут изменяться. Всегда прослушивайте события изменения темы, чтобы поддерживать синхронизацию пользовательского интерфейса.
:::
