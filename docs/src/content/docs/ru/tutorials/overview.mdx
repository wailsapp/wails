### Сервис QR-кодов

<Image src={qr1} alt="QR Code Example" style="max-width: 400px; margin: 1rem auto; display: block;" />

Изучите основы сервисов Wails, создав генератор QR-кодов. Это руководство познакомит вас с ключевыми концепциями организации логики приложения в виде переиспользуемых сервисов.

**Что вы узнаете:**
- Как создавать и структурировать сервис Wails
- Управление внешними зависимостями Go
- Привязка методов Go к вашему фронтенду
- Передача данных между Go и JavaScript
- Организация кода для удобства поддержки

**Для кого:** Для пользователей Wails, которые хотят понять архитектуру сервисов

### Список задач

<Image src={todoApp} alt="TODO List Application" style="max-width: 400px; margin: 1rem auto; display: block;" />

Создайте полноценное приложение со списком задач и красивым современным интерфейсом. Это практическое руководство обучит вас основным паттернам Wails на примере реального приложения, написанного на чистом JavaScript.

**Что вы узнаете:**
- Архитектура на основе сервисов с управлением состоянием, безопасным для потоков
- Операции CRUD (создание, чтение, обновление, удаление)
- Типобезопасные привязки между Go и JavaScript
- Создание современных интерфейсов без сложности фреймворков
- Правильная обработка ошибок и паттерны валидации

**Время на выполнение:** ~20 минут

**Для кого:** Для вашего первого полноценного приложения Wails — идеально подходит для понимания основ перед добавлением сложности фреймворков

### Заметки

<Image src={notesApp} alt="Notes Application" style="max-width: 600px; margin: 1rem auto; display: block;" />

Создайте приложение в стиле Apple Notes с нативными диалогами файлов и функцией автосохранения. Это руководство демонстрирует специфичные для десктопа функции, такие как операции с файлами, нативные диалоги и профессиональные паттерны интерфейса.

**Что вы узнаете:**
- Нативные диалоги файлов (Сохранить, Открыть, Информация)
- Персистентность данных на основе JSON
- Паттерны автосохранения с дебаунсингом
- Профессиональные двухколоночные десктопные макеты
- Работа с операциями файловой системы в Go

**Время на выполнение:** ~30 минут

**Для кого:** Для изучения специфичных для десктопа функций, таких как операции с файлами и нативные диалоги ОС
