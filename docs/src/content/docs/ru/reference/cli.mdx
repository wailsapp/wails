## Обзор

Wails CLI (`wails3`) предоставляет команды для создания, разработки, сборки и управления приложениями Wails.

## Скоро

Этот раздел находится в разработке. Пока что используйте `wails3 --help` или `wails3 [command] --help` для получения документации по командам.

**Основные команды:**

```bash
wails3 init          # Create a new project
wails3 dev           # Run in development mode
wails3 build         # Build for production
wails3 generate      # Generate bindings or runtime
wails3 doctor        # Check development environment
```
