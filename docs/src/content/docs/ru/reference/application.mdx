## Обзор

`Application` — это ядро вашего приложения Wails. Оно управляет окнами, сервисами, событиями и предоставляет доступ ко всем платформенным функциям.

## Создание приложения

```go
import "github.com/wailsapp/wails/v3/pkg/application"

app := application.New(application.Options{
    Name:        "My App",
    Description: "My awesome application",
    Services: []application.Service{
        application.NewService(&MyService{}),
    },
})
```

## Основные методы

### Run()

Запускает цикл обработки событий приложения.

```go
func (a *App) Run() error
```

**Пример:**

```go
err := app.Run()
if err != nil {
    log.Fatal(err)
}
```

**Возвращает:** Ошибка, если запуск не удался

### Quit()

Корректно завершает работу приложения.

```go
func (a *App) Quit()
```

**Пример:**

```go
// В обработчике меню
menu.Add("Quit").OnClick(func(ctx *application.Context) {
    app.Quit()
})
```

### Config()

Возвращает конфигурацию приложения.

```go
func (a *App) Config() Options
```

**Пример:**

```go
config := app.Config()
fmt.Println("App name:", config.Name)
```

## Управление окнами

### app.Window.New()

Создает новое окно webview с параметрами по умолчанию.

```go
func (wm *WindowManager) New() *WebviewWindow
```

**Пример:**

```go
window := app.Window.New()
window.Show()
```

### app.Window.NewWithOptions()

Создает новое окно webview с пользовательскими параметрами.

```go
func (wm *WindowManager) NewWithOptions(options WebviewWindowOptions) *WebviewWindow
```

**Пример:**

```go
window := app.Window.NewWithOptions(application.WebviewWindowOptions{
    Title:  "My Window",
    Width:  800,
    Height: 600,
    BackgroundColour: application.NewRGB(255, 255, 255),
})
```

### app.Window.GetByName()

Получает окно по его имени.

```go
func (wm *WindowManager) GetByName(name string) Window
```

**Пример:**

```go
window := app.Window.GetByName("main")
if window != nil {
    window.Show()
}
```

### app.Window.GetAll()

Возвращает все окна приложения.

```go
func (wm *WindowManager) GetAll() []Window
```

**Пример:**

```go
windows := app.Window.GetAll()
for _, window := range windows {
    fmt.Println("Window:", window.Name())
}
```

## Менеджеры

Приложение предоставляет доступ к различным менеджерам через свойства:

```go
app.Window      // Управление окнами
app.Menu        // Управление меню
app.Dialog      // Управление диалогами
app.Event       // Управление событиями
app.Clipboard   // Операции с буфером обмена
app.Screen      // Информация о экранах
app.SystemTray  // Системный трей
app.Browser     // Операции браузера
app.Env         // Переменные окружения
```

### Пример использования

```go
// Создание окна
window := app.Window.New()

// Показ диалога
app.Dialog.Info().SetMessage("Hello!").Show()

// Копирование в буфер обмена
app.Clipboard.SetText("Copied text")

// Получение списка экранов
screens := app.Screen.GetAll()
```

## Управление сервисами

### RegisterService()

Регистрирует сервис в приложении.

```go
func (a *App) RegisterService(service Service) error
```

**Пример:**

```go
type MyService struct {
    app *application.Application
}

func NewMyService(app *application.Application) *MyService {
    return &MyService{app: app}
}

// Регистрация после создания приложения
app.RegisterService(application.NewService(NewMyService(app)))
```

## Управление событиями

### app.Event.Emit()

Генерирует пользовательское событие.

```go
func (em *EventManager) Emit(name string, data ...interface{})
```

**Пример:**

```go
// Генерация события с данными
app.Event.Emit("user-logged-in", map[string]interface{}{
    "username": "john",
    "timestamp": time.Now(),
})
```

### app.Event.On()

Подписывается на пользовательские события.

```go
func (em *EventManager) On(name string, callback func(*CustomEvent))
```

**Пример:**

```go
app.Event.On("user-logged-in", func(e *application.CustomEvent) {
    data := e.Data.(map[string]interface{})
    username := data["username"].(string)
    fmt.Println("User logged in:", username)
})
```

### app.Event.OnApplicationEvent()

Подписывается на события жизненного цикла приложения.

```go
func (em *EventManager) OnApplicationEvent(eventType ApplicationEventType, callback func(*ApplicationEvent)) func()
```

**Пример:**

```go
// Подписка на событие завершения работы
app.Event.OnApplicationEvent(application.EventApplicationShutdown, func(e *application.ApplicationEvent) {
    fmt.Println("Application shutting down")
    // Очистка ресурсов
})
```

## Методы диалогов

Диалоги доступны через менеджер `app.Dialog`. Полный справочник см. в [API диалогов](/reference/dialogs/).

### Диалоги сообщений

```go
// Информационный диалог
app.Dialog.Info().
    SetTitle("Success").
    SetMessage("Operation completed!").
    Show()

// Диалог ошибки
app.Dialog.Error().
    SetTitle("Error").
    SetMessage("Something went wrong.").
    Show()

// Диалог предупреждения
app.Dialog.Warning().
    SetTitle("Warning").
    SetMessage("This action cannot be undone.").
    Show()
```

### Диалоги вопросов

Диалоги вопросов используют обратные вызовы кнопок для обработки ответов пользователя:

```go
dialog := app.Dialog.Question().
    SetTitle("Confirm").
    SetMessage("Continue?")

yes := dialog.AddButton("Yes")
yes.OnClick(func() {
    // Обработка ответа "Да"
})

no := dialog.AddButton("No")
no.OnClick(func() {
    // Обработка ответа "Нет"
})

dialog.SetDefaultButton(yes)
dialog.SetCancelButton(no)
dialog.Show()
```

### Диалоги файлов

```go
// Диалог открытия файла
path, err := app.Dialog.OpenFile().
    SetTitle("Select File").
    AddFilter("Images", "*.png;*.jpg").
    PromptForSingleSelection()

// Диалог сохранения файла
path, err := app.Dialog.SaveFile().
    SetTitle("Save File").
    SetFilename("document.pdf").
    AddFilter("PDF", "*.pdf").
    PromptForSingleSelection()

// Выбор папки (используйте OpenFile с параметрами каталога)
path, err := app.Dialog.OpenFile().
    SetTitle("Select Folder").
    CanChooseDirectories(true).
    CanChooseFiles(false).
    PromptForSingleSelection()
```

## Логгер

Приложение предоставляет структурированный логгер:

```go
app.Logger.Info("Message", "key", "value")
app.Logger.Error("Error occurred", "error", err)
app.Logger.Debug("Debug info")
app.Logger.Warn("Warning message")
```

**Пример:**

```go
func (s *MyService) ProcessData(data string) error {
    s.app.Logger.Info("Processing data", "length", len(data))
    
    if err := process(data); err != nil {
        s.app.Logger.Error("Processing failed", "error", err)
        return err
    }
    
    s.app.Logger.Info("Processing complete")
    return nil
}
```

## Обработка сырых сообщений

Для приложений, которым требуется прямой низкоуровневый контроль над коммуникацией между фронтендом и бэкендом, Wails предоставляет опцию `RawMessageHandler`. Она обходит стандартную систему привязок.

:::note
Сырые сообщения следует использовать только в крайнем случае. Стандартная система привязок высоко оптимизирована и подходит для почти всех приложений. Используйте сырые сообщения только если вы профилировали свое приложение и подтвердили, что привязки являются узким местом.
:::

### RawMessageHandler

Обратный вызов `RawMessageHandler` получает сырые сообщения, отправленные с фронтенда с помощью `System.invoke()`.

```go
func (a *App) RawMessageHandler(window Window, message string)
```

**Пример:**

```go
app := application.New(application.Options{
    Name: "My App",
    RawMessageHandler: func(window application.Window, message string) {
        // Обработка сырого сообщения
        fmt.Printf("Received from %s: %s\n", window.Name(), message)

        // Можно ответить с помощью событий
        window.EmitEvent("response", processMessage(message))
    },
})
```


Более подробную информацию см. в [Руководстве по сырым сообщениям](/ru/guides/raw-messages).

## Платформозависимые параметры

### Параметры Windows

Настройте поведение, специфичное для Windows, на уровне приложения:

```go
app := application.New(application.Options{
    Name: "My App",
    Windows: application.WindowsOptions{
        // Флаги браузера WebView2 (применяются ко ВСЕМ окнам)        EnabledFeatures:       []string{"msWebView2EnableDraggableRegions"},
        DisabledFeatures:      []string{"msExperimentalFeature"},
        AdditionalBrowserArgs: []string{"--remote-debugging-port=9222"},

        // Other Windows options
        WndClass:                      "MyAppClass",
        WebviewUserDataPath:           "",  // Default: %APPDATA%\[BinaryName.exe]
        WebviewBrowserPath:            "",  // Default: system WebView2
        DisableQuitOnLastWindowClosed: false,
    },
})
```

**Флаги браузера:**
- `EnabledFeatures` - флаги функций WebView2 для включения
- `DisabledFeatures` - флаги функций WebView2 для отключения
- `AdditionalBrowserArgs` - аргументы командной строки Chromium

Подробную документацию см. в разделе [Параметры окна — Параметры окон на уровне приложения](/features/windows/options#application-level-windows-options).

### Параметры для Mac

```go
app := application.New(application.Options{
    Name: "My App",
    Mac: application.MacOptions{
        ActivationPolicy: application.ActivationPolicyRegular,
        ApplicationShouldTerminateAfterLastWindowClosed: true,
    },
})
```

### Параметры для Linux

```go
app := application.New(application.Options{
    Name: "My App",
    Linux: application.LinuxOptions{
        ProgramName:                   "my-app",
        DisableQuitOnLastWindowClosed: false,
    },
})
```

## Полный пример приложения

```go
package main

import (
	"github.com/wailsapp/wails/v3/pkg/application"
)

func main() {
	app := application.New(application.Options{
		Name:        "My Application",
		Description: "A demo application",
		Mac: application.MacOptions{
			ApplicationShouldTerminateAfterLastWindowClosed: true,
		},
	})

	// Create main window
	window := app.Window.NewWithOptions(application.WebviewWindowOptions{
		Title:            "My App",
		Width:            1024,
		Height:           768,
		MinWidth:         800,
		MinHeight:        600,
		BackgroundColour: application.NewRGB(255, 255, 255),
		URL:              "http://wails.localhost/",
	})

	window.Centre()
	window.Show()

	app.Run()
}
