## Настройка среды разработки

Это руководство проведет вас через процесс настройки полной среды разработки для работы с Wails v3.

## Необходимые инструменты

### Разработка на Go

1. **Установите Go 1.25 или новее:**
   ```bash
   # Скачайте с https://go.dev/dl/
   go version  # Проверьте установку
   ```

2. **Настройте окружение Go:**
   ```bash
   # Добавьте в профиль вашей оболочки (.bashrc, .zshrc и т.д.)
   export GOPATH=$HOME/go
   export PATH=$PATH:$GOPATH/bin
   ```

3. **Установите полезные инструменты Go:**
   ```bash
   go install golang.org/x/tools/cmd/goimports@latest
   go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest
   ```

### Node.js и npm

Требуется для сборки документации и тестирования интеграций с фронтендом.

```bash
# Установите Node.js 20+ и npm
node --version  # Должно быть 20+
npm --version
```

### Платформозависимые зависимости

**macOS:**

```bash
# Установите Xcode Command Line Tools
xcode-select --install

# Проверьте установку
xcode-select -p  # Должно вывести путь
```

**Windows:**

1. Установите [MSYS2](https://www.msys2.org/) для получения окружения, похожего на Unix
2. WebView2 Runtime (предустановлен в Windows 11, [скачать](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) для Windows 10)
3. Опционально: Установите [Git for Windows](https://git-scm.com/download/win)

**Linux (Debian/Ubuntu):**

```bash
sudo apt update
sudo apt install build-essential pkg-config libgtk-3-dev libwebkit2gtk-4.0-dev
```

**Linux (Fedora/RHEL):**

```bash
sudo dnf install gcc pkg-config gtk3-devel webkit2gtk3-devel
```

**Linux (Arch):**

```bash
sudo pacman -S base-devel gtk3 webkit2gtk
```

## Настройка репозитория

### Клонирование и конфигурация

```bash
# Клонируйте вашу форк-версию
git clone https://github.com/YOUR_USERNAME/wails.git
cd wails

# Добавьте удаленный репозиторий upstream
git remote add upstream https://github.com/wailsapp/wails.git

# Проверьте удаленные репозитории
git remote -v
```

### Сборка CLI Wails

```bash
# Перейдите в директорию v3
cd v3

# Соберите CLI
go build -o ../wails3 ./cmd/wails3

# Проверьте сборку
cd ..
./wails3 version
```

### Добавление в PATH (опционально)

**Linux/macOS:**

```bash
# Добавьте в ~/.bashrc или ~/.zshrc
export PATH=$PATH:/path/to/wails
```

**Windows:**

Добавьте директорию Wails в переменную окружения PATH через свойства системы.

## Настройка IDE

### VS Code (рекомендуется)

1. **Установите VS Code:** [Скачать](https://code.visualstudio.com/)

2. **Установите расширения:**
   - Go (от Go Team at Google)
   - ESLint
   - Prettier
   - MDX (для документации)

3. **Настройте параметры рабочего пространства** (`.vscode/settings.json`):
   ```json
   {
     "go.useLanguageServer": true,
     "go.lintTool": "golangci-lint",
     "go.lintOnSave": "workspace",
     "editor.formatOnSave": true,
     "go.formatTool": "goimports"
   }
   ```

### GoLand

1. **Установите GoLand:** [Скачать](https://www.jetbrains.com/go/)

2. **Настройте:**
   - Включите поддержку Go modules
   - Настройте наблюдение за файлами для `goimports`
   - Настройте стиль кода в соответствии с соглашениями проекта

## Проверка вашей настройки

Выполните эти команды, чтобы убедиться, что всё работает:

```bash
# Проверка версии Go
go version

# Сборка Wails
cd v3
go build ./cmd/wails3

# Запуск тестов
go test ./pkg/...

# Создание тестового приложения
cd ..
./wails3 init -n mytest -t vanilla
cd mytest
../wails3 dev
```

Если тестовое приложение собирается и запускается, ваша среда готова!

## Запуск тестов

### Модульные тесты

```bash
cd v3
go test ./...
```

### Тесты конкретных пакетов

```bash
go test ./pkg/application
go test ./pkg/events -v  # Подробный вывод
```

### Запуск с покрытием кода

```bash
go test ./... -coverprofile=coverage.out
go tool cover -html=coverage.out
```

### Запуск с детектором гонок

```bash
go test ./... -race
```

## Работа с документацией

Документация Wails собирается с помощью Astro и Starlight.

```bash
cd docs

# Установите зависимости
npm install

# Запустите сервер разработки
npm run dev

# Соберите для продакшена
npm run build
```

Документация будет доступна по адресу `http://localhost:4321/`

## Отладка

### Отладка кода Go

**VS Code:**

Создайте `.vscode/launch.json`:

```json
{
  "version": "0.2.0",
  "configurations": [
    {
      "name": "Debug Wails CLI",
      "type": "go",
      "request": "launch",
      "mode": "debug",
      "program": "${workspaceFolder}/v3/cmd/wails3",
      "args": ["dev"]
    }
  ]
}
```

**Командная строка:**

```bash
# Используйте отладчик Delve
go install github.com/go-delve/delve/cmd/dlv@latest
dlv debug ./cmd/wails3 -- dev
```

### Отладка платформенного кода

Платформозависимая отладка требует инструментов платформы:

- **macOS:** Xcode Instruments
- **Windows:** Visual Studio Debugger
- **Linux:** GDB

## Распространенные проблемы

### "command not found: wails3"

Добавьте директорию Wails в ваш PATH или используйте `./wails3` из корня проекта.

### "webkit2gtk not found" (Linux)

Установите пакеты разработки WebKit2GTK:

```bash
sudo apt install libwebkit2gtk-4.0-dev  # Debian/Ubuntu
```

### Ошибки сборки с ошибками Go модулей

```bash
cd v3
go mod tidy
go mod download
```

### Ошибки "CGO_ENABLED" на Windows

Убедитесь, что у вас есть C-компилятор (MinGW-w64 через MSYS2) в вашем PATH.
