Wails v3 — это **полнофункциональный фреймворк для десктопных приложений**, состоящий из среды выполнения Go, моста JavaScript, инструмента, управляемого задачами, и набора шаблонов, которые позволяют вам выпускать нативные приложения, работающие на основе современных веб-технологий.

На этой странице представлена *общая картина* в четырёх диаграммах:

1. **Общая архитектура** – как подключаются все подсистемы
2. **Поток выполнения во время работы** – что происходит, когда JS вызывает Go и наоборот
3. **Разработка против продакшена** – два режима сервера активов
4. **Реализации для платформ** – где находится код, специфичный для ОС

Ключевые моменты:

* **Нет HTTP / IPC** – мост использует встроенный в память канал нативного WebView
* **Идентификаторы методов** – детерминированный хеш FNV обеспечивает поиск O(1) в Go
* **Промисы** – ошибки распространяются как отклонения с кодом и стеком вызовов

**Файлы среды выполнения для каждой ОС**

**[Заглушка для диаграммы разделения платформ]**

Каждая функция следует этому шаблону:

1. **Общий интерфейс** в `pkg/application`
2. **Точка входа процессора сообщений** в `pkg/application/messageprocessor_*.go`
3. **Реализация** для каждой ОС в `internal/runtime/*.go`, защищённая тегами сборки

Отсутствующая функциональность для конкретной ОС должна возвращать `ErrCapability` и регистрировать доступность через `internal/capabilities`.
