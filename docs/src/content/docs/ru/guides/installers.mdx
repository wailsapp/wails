## Обзор

Создавайте профессиональные установщики для вашего приложения Wails на всех платформах.

## Платформенные установщики

<Tabs syncKey="platform">
  <TabItem label="Windows" icon="seti:windows">
    ### NSIS установщик
    
    ```bash
    # Install NSIS
    # Download from: https://nsis.sourceforge.io/
    
    # Create installer script (installer.nsi)
    makensis installer.nsi
    ```
    
    **installer.nsi:**
    ```nsis
    !define APPNAME "MyApp"
    !define VERSION "1.0.0"
    
    Name "${APPNAME}"
    OutFile "MyApp-Setup.exe"
    InstallDir "$PROGRAMFILES\${APPNAME}"
    
    Section "Install"
        SetOutPath "$INSTDIR"
        File "build\bin\myapp.exe"
        CreateShortcut "$DESKTOP\${APPNAME}.lnk" "$INSTDIR\myapp.exe"
    SectionEnd
    ```
    
    ### WiX Toolset
    
    Альтернатива для MSI-установщиков.
  </TabItem>

  <TabItem label="macOS" icon="apple">
    ### Создание DMG
    
    ```bash
    # Create DMG
    hdiutil create -volname "MyApp" -srcfolder build/bin/MyApp.app -ov -format UDZO MyApp.dmg
    ```
    
    ### Подпись кода
    
    ```bash
    # Sign application
    codesign --deep --force --verify --verbose --sign "Developer ID" MyApp.app
    
    # Notarize
    xcrun notarytool submit MyApp.dmg --apple-id "email" --password "app-password"
    ```
    
    ### App Store
    
    Используйте Xcode для распространения через App Store.
  </TabItem>

  <TabItem label="Linux" icon="linux">
    ### DEB-пакет
    
    ```bash
    # Create package structure
    mkdir -p myapp_1.0.0/DEBIAN
    mkdir -p myapp_1.0.0/usr/bin
    
    # Copy binary
    cp build/bin/myapp myapp_1.0.0/usr/bin/
    
    # Create control file
    cat > myapp_1.0.0/DEBIAN/control << EOF
    Package: myapp
    Version: 1.0.0
    Architecture: amd64
    Maintainer: Your Name
    Description: My Application
    EOF
    
    # Build package
    dpkg-deb --build myapp_1.0.0
    ```
    
    ### RPM-пакет
    
    Используйте `rpmbuild` для дистрибутивов на базе RPM.
    
    ### AppImage
    
    ```bash
    # Use appimagetool
    appimagetool myapp.AppDir
    ```
  </TabItem>
</Tabs>

## Автоматизированная упаковка

### Использование GoReleaser

```yaml
# .goreleaser.yml
project_name: myapp

builds:
  - binary: myapp
    goos:
      - windows
      - darwin
      - linux
    goarch:
      - amd64
      - arm64

archives:
  - format: zip
    name_template: "{{ .ProjectName }}_{{ .Version }}_{{ .Os }}_{{ .Arch }}"

nfpms:
  - formats:
      - deb
      - rpm
    vendor: Your Company
    homepage: https://example.com
    description: My Application
```

## Лучшие практики

### ✅ Что делать

- Подписывайте код на всех платформах
- Включайте информацию о версии
- Создавайте программы удаления
- Тестируйте процесс установки
- Предоставляйте четкую документацию

### ❌ Чего не делать

- Не пропускайте подпись кода
- Не забывайте о ассоциациях файлов
- Не используйте жестко заданные пути
- Не пропускайте тестирование
