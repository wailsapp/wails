## Возможности

Wails предоставляет нативные возможности для десктопных приложений:

- [Windows](/features/windows/basics) - несколько окон, окна без рамки, позиционирование
- [Меню](/features/menus/application) - меню приложения, контекстные меню, системный трей
- [Диалоги](/features/dialogs/file) - открытие/сохранение файлов, диалоги сообщений
- [События](/features/events/system) - взаимодействие между компонентами
- [Bindings](/features/bindings/methods) - типобезопасные вызовы Go ↔ JavaScript
- [Буфер обмена](/features/clipboard) - операции копирования/вставки
- [Drag & Drop](/features/drag-and-drop/files) - перетаскивание файлов
- [Клавиатура](/features/keyboard) - глобальные горячие клавиши
