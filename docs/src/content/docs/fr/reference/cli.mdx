## Vue d'ensemble

La CLI Wails (`wails3`) fournit des commandes pour créer, développer, construire et gérer des applications Wails.

## Bientôt disponible

Cette section est en cours de construction. Pour l'instant, utilisez `wails3 --help` ou `wails3 [commande] --help` pour la documentation des commandes.

**Commandes courantes :**

```bash
wails3 init          # Create a new project
wails3 dev           # Run in development mode
wails3 build         # Build for production
wails3 generate      # Generate bindings or runtime
wails3 doctor        # Check development environment
```
