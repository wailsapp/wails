## Vue d'ensemble

L'API des boîtes de dialogue fournit des méthodes pour afficher des boîtes de dialogue de fichiers natives et des boîtes de dialogue de message. Accédez aux boîtes de dialogue via le gestionnaire `app.Dialog`.

**Types de boîtes de dialogue :**
- **Boîtes de dialogue de fichiers** - Boîtes de dialogue Ouvrir et Enregistrer
- **Boîtes de dialogue de message** - Boîtes de dialogue Info, Erreur, Avertissement et Question

Toutes les boîtes de dialogue sont des **boîtes de dialogue natives du système d'exploitation** qui correspondent au look and feel de la plateforme.

## Accès aux boîtes de dialogue

Les boîtes de dialogue sont accessibles via le gestionnaire `app.Dialog` :

```go
app.Dialog.OpenFile()
app.Dialog.SaveFile()
app.Dialog.Info()
app.Dialog.Question()
app.Dialog.Warning()
app.Dialog.Error()
```

## Boîtes de dialogue de fichiers

### OpenFile()

Crée une boîte de dialogue d'ouverture de fichier.

```go
func (dm *DialogManager) OpenFile() *OpenFileDialogStruct
```

**Exemple :**
```go
dialog := app.Dialog.OpenFile()
```

### Méthodes de OpenFileDialogStruct

#### SetTitle()

Définit le titre de la boîte de dialogue.

```go
func (d *OpenFileDialogStruct) SetTitle(title string) *OpenFileDialogStruct
```

**Exemple :**
```go
dialog.SetTitle("Sélectionner une image")
```

#### AddFilter()

Ajoute un filtre de type de fichier.

```go
func (d *OpenFileDialogStruct) AddFilter(displayName, pattern string) *OpenFileDialogStruct
```

**Paramètres :**
- `displayName` - Description du filtre affichée à l'utilisateur (par exemple, "Images", "Documents")
- `pattern` - Liste d'extensions séparées par des points-virgules (par exemple, "*.png;*.jpg")

**Exemple :**
```go
dialog.AddFilter("Images", "*.png;*.jpg;*.gif").
    AddFilter("Documents", "*.pdf;*.docx").
    AddFilter("Tous les fichiers", "*.*")
```

#### SetDirectory()

Définit le répertoire initial.

```go
func (d *OpenFileDialogStruct) SetDirectory(directory string) *OpenFileDialogStruct
```

**Exemple :**
```go
homeDir, _ := os.UserHomeDir()
dialog.SetDirectory(homeDir)
```

#### CanChooseDirectories()

Active ou désactive la sélection de répertoires.

```go
func (d *OpenFileDialogStruct) CanChooseDirectories(canChooseDirectories bool) *OpenFileDialogStruct
```

**Exemple (sélection de dossier) :**
```go
// Sélectionner des dossiers au lieu de fichiers
path, err := app.Dialog.OpenFile().
    SetTitle("Sélectionner un dossier").
    CanChooseDirectories(true).
    CanChooseFiles(false).
    PromptForSingleSelection()
```

#### CanChooseFiles()

Active ou désactive la sélection de fichiers.

```go
func (d *OpenFileDialogStruct) CanChooseFiles(canChooseFiles bool) *OpenFileDialogStruct
```

#### CanCreateDirectories()

Active ou désactive la création de nouveaux répertoires.

```go
func (d *OpenFileDialogStruct) CanCreateDirectories(canCreateDirectories bool) *OpenFileDialogStruct
```

#### ShowHiddenFiles()

Affiche ou masque les fichiers cachés.

```go
func (d *OpenFileDialogStruct) ShowHiddenFiles(showHiddenFiles bool) *OpenFileDialogStruct
```

#### AttachToWindow()

Attache la boîte de dialogue à une fenêtre spécifique.

```go
func (d *OpenFileDialogStruct) AttachToWindow(window Window) *OpenFileDialogStruct
```

#### PromptForSingleSelection()

Affiche la boîte de dialogue et retourne le fichier sélectionné.

```go
func (d *OpenFileDialogStruct) PromptForSingleSelection() (string, error)
```

**Retourne :**
- `string` - Chemin du fichier sélectionné (vide si annulé)
- `error` - Erreur si la boîte de dialogue a échoué

**Exemple :**
```go
path, err := app.Dialog.OpenFile().
    SetTitle("Sélectionner une image").
    AddFilter("Images", "*.png;*.jpg;*.gif").
    PromptForSingleSelection()

if err != nil {
    // Utilisateur a annulé ou une erreur s'est produite
    return
}

// Utiliser le fichier sélectionné
processFile(path)
```

#### PromptForMultipleSelection()

Affiche la boîte de dialogue et retourne plusieurs fichiers sélectionnés.

```go
func (d *OpenFileDialogStruct) PromptForMultipleSelection() ([]string, error)
```

**Retourne :**
- `[]string` - Tableau des chemins des fichiers sélectionnés
- `error` - Erreur si la boîte de dialogue a échoué

**Exemple :**
```go
paths, err := app.Dialog.OpenFile().
    SetTitle("Sélectionner des images").
    AddFilter("Images", "*.png;*.jpg").
    PromptForMultipleSelection()

if err != nil {
    return
}

for _, path := range paths {
    processFile(path)
}
```

## Boîtes de dialogue de message

Toutes les boîtes de dialogue de message retournent `*MessageDialog` et partagent les mêmes méthodes.

### Info()

Crée une boîte de dialogue d'information.

```go
func (dm *DialogManager) Info() *MessageDialog
```

**Exemple :**
```go
app.Dialog.Info().
    SetTitle("Succès").
    SetMessage("Fichier enregistré avec succès !").
    Show()
```

### Error()

Crée une boîte de dialogue d'erreur.

```go
func (dm *DialogManager) Error() *MessageDialog
```

**Exemple :**
```go
app.Dialog.Error().
    SetTitle("Erreur").
    SetMessage("Échec de l'enregistrement du fichier : " + err.Error()).
    Show()
```

### Warning()

Crée une boîte de dialogue d'avertissement.

```go
func (dm *DialogManager) Warning() *MessageDialog
```

**Exemple :**
```go
app.Dialog.Warning().
    SetTitle("Avertissement").
    SetMessage("Cette action ne peut pas être annulée.").
    Show()
```

### Question()

Crée une boîte de dialogue de question avec des boutons personnalisés.

```go
func (dm *DialogManager) Question() *MessageDialog
```

**Exemple :**
```go
dialog := app.Dialog.Question().
    SetTitle("Confirmer").
    SetMessage("Voulez-vous enregistrer les modifications ?")

save := dialog.AddButton("Enregistrer")
save.OnClick(func() {
    saveDocument()
})

dontSave := dialog.AddButton("Ne pas enregistrer")
dontSave.OnClick(func() {
    // Continuer sans enregistrer
})

cancel := dialog.AddButton("Annuler")
cancel.OnClick(func() {
    // Ne rien faire
})

dialog.SetDefaultButton(save)
dialog.SetCancelButton(cancel)
dialog.Show()
```

### Méthodes de MessageDialog

#### SetTitle()

Définit le titre de la boîte de dialogue.

```go
func (d *MessageDialog) SetTitle(title string) *MessageDialog
```

#### SetMessage()

Définit le message de la boîte de dialogue.

```go
func (d *MessageDialog) SetMessage(message string) *MessageDialog
```

#### SetIcon()

Définit une icône personnalisée pour la boîte de dialogue.

```go
func (d *MessageDialog) SetIcon(icon []byte) *MessageDialog
```#### AddButton()

Ajoute un bouton à la boîte de dialogue et renvoie l'instance du bouton pour configuration.

```go
func (d *MessageDialog) AddButton(label string) *Button
```

**Retour :** `*Button` - L'instance du bouton pour une configuration ultérieure

**Exemple :**
```go
button := dialog.AddButton("OK")
button.OnClick(func() {
    // Gérer le clic
})
```

#### SetDefaultButton()

Définit quel bouton est le bouton par défaut (activé en appuyant sur Entrée).

```go
func (d *MessageDialog) SetDefaultButton(button *Button) *MessageDialog
```

**Exemple :**
```go
yes := dialog.AddButton("Yes")
no := dialog.AddButton("No")
dialog.SetDefaultButton(yes)
```

#### SetCancelButton()

Définit quel bouton est le bouton d'annulation (activé en appuyant sur Échap).

```go
func (d *MessageDialog) SetCancelButton(button *Button) *MessageDialog
```

**Exemple :**
```go
ok := dialog.AddButton("OK")
cancel := dialog.AddButton("Cancel")
dialog.SetCancelButton(cancel)
```

#### AttachToWindow()

Attache la boîte de dialogue à une fenêtre spécifique.

```go
func (d *MessageDialog) AttachToWindow(window Window) *MessageDialog
```

#### Show()

Affiche la boîte de dialogue. Les rappels des boutons gèrent les réponses de l'utilisateur.

```go
func (d *MessageDialog) Show()
```

**Remarque :** `Show()` ne renvoie aucune valeur. Utilisez les rappels des boutons pour gérer les réponses de l'utilisateur.

### Méthodes des boutons

#### SetAsDefault()

Marque ce bouton comme le bouton par défaut.

```go
func (b *Button) SetAsDefault() *Button
```

#### SetAsCancel()

Marque ce bouton comme le bouton d'annulation.

```go
func (b *Button) SetAsCancel() *Button
```

### Traitement de plusieurs fichiers

```go
func (s *Service) ProcessMultipleFiles(app *application.App) error {
    // Select multiple files
    paths, err := app.Dialog.OpenFile().
        SetTitle("Select Files to Process").
        AddFilter("Images", "*.png;*.jpg").
        PromptForMultipleSelection()

    if err != nil {
        return err
    }

    if len(paths) == 0 {
        app.Dialog.Info().
            SetTitle("No Files Selected").
            SetMessage("Please select at least one file.").
            Show()
        return nil
    }

    // Process files
    for _, path := range paths {
        err := processFile(path)
        if err != nil {
            app.Dialog.Error().
                SetTitle("Processing Error").
                SetMessage(fmt.Sprintf("Failed to process %s: %v", path, err)).
                Show()
            continue
        }
    }

    // Show completion
    app.Dialog.Info().
        SetTitle("Complete").
        SetMessage(fmt.Sprintf("Successfully processed %d files", len(paths))).
        Show()

    return nil
}
```

### Gestion des erreurs avec les boîtes de dialogue

```go
func (s *Service) SaveFile(app *application.App, data []byte) error {
    // Select save location
    path, err := app.Dialog.SaveFile().
        SetTitle("Save File").
        SetFilename("data.json").
        AddFilter("JSON File", "*.json").
        PromptForSingleSelection()

    if err != nil {
        // User cancelled - not an error
        return nil
    }

    // Attempt to save
    err = os.WriteFile(path, data, 0644)
    if err != nil {
        // Show error dialog
        app.Dialog.Error().
            SetTitle("Save Failed").
            SetMessage(fmt.Sprintf("Could not save file: %v", err)).
            Show()
        return err
    }

    // Show success
    app.Dialog.Info().
        SetTitle("Success").
        SetMessage("File saved successfully!").
        Show()

    return nil
}
```

import (

func (s *Service) GetDefaultDirectory() string {
    homeDir, _ := os.UserHomeDir()

    switch runtime.GOOS {
    case "windows":
        return filepath.Join(homeDir, "Documents")
    case "darwin":
        return filepath.Join(homeDir, "Documents")
    case "linux":
        return filepath.Join(homeDir, "Documents")

    }

## Bonnes pratiques

### À faire

- **Utiliser des boîtes de dialogue natives** - Elles correspondent à l'apparence et au ressenti de la plateforme
- **Fournir des titres clairs** - Aider les utilisateurs à comprendre l'objectif
- **Définir des filtres appropriés** - Guider les utilisateurs vers les types de fichiers corrects
- **Gérer l'annulation** - Vérifier les erreurs (l'utilisateur peut annuler)
- **Demander une confirmation pour les actions destructrices** - Utiliser des boîtes de dialogue de type Question
- **Fournir des retours** - Utiliser des boîtes de dialogue Info pour les messages de succès
- **Définir des valeurs par défaut raisonnables** - Répertoire par défaut, nom de fichier, etc.
- **Utiliser des rappels pour les actions des boutons** - Gérer correctement les réponses de l'utilisateur

### À ne pas faire

- **Ne pas ignorer les erreurs** - L'annulation par l'utilisateur renvoie une erreur
- **Ne pas utiliser d'étiquettes de boutons ambiguës** - Être spécifique : "Enregistrer"/"Annuler"
- **Ne pas abuser des boîtes de dialogue** - Elles interrompent le flux de travail
- **Ne pas afficher d'erreurs pour une annulation** - C'est une action normale
- **Ne pas oublier les filtres de fichiers** - Aider les utilisateurs à trouver les bons fichiers
- **Ne pas coder en dur les chemins** - Utiliser os.UserHomeDir() ou similaire

## Types de boîtes de dialogue par plateforme

### macOS

- Les boîtes de dialogue glissent depuis la barre de titre
- Style "Sheet" attaché à la fenêtre parente
- Apparence native macOS

### Windows

- Boîtes de dialogue Windows standard
- Suit les lignes directrices de design Windows
- Apparence moderne Windows 10/11

### Linux

- Boîtes de dialogue GTK sur les systèmes basés sur GTK
- Boîtes de dialogue Qt sur les systèmes basés sur Qt
- Correspond à l'environnement de bureau--------|------|------|-------|
| `ShowHiddenFiles()` | ✅ Fonctionne | ❌ Aucun effet | L'utilisateur contrôle via le basculeur d'interface de la boîte de dialogue (Ctrl+H ou menu) |
| `CanCreateDirectories()` | ✅ Fonctionne | ❌ Aucun effet | Toujours activé dans le portail |
| `ResolvesAliases()` | ✅ Fonctionne | ❌ Aucun effet | Le portail gère la résolution des liens symboliques |
| `SetButtonText()` | ✅ Fonctionne | ✅ Fonctionne | Le texte personnalisé du bouton d'acceptation fonctionne |

**Pourquoi ces limitations existent :** Les boîtes de dialogue basées sur le portail de GTK4 délèguent le contrôle de l'interface utilisateur à
l'environnement de bureau (GNOME, KDE, etc.). C'est intentionnel - le portail offre une expérience utilisateur cohérente entre
les applications et respecte les préférences de l'utilisateur.

:::note
Le tableau ci-dessus montre le comportement de GTK4 lors de la compilation avec `-tags gtk4`. Par défaut, Wails utilise GTK3
qui offre un contrôle total sur ces options de boîte de dialogue.
:::

## Modèles courants

### Modèle « Enregistrer sous »

```go
func (s *Service) SaveAs(app *application.App, currentPath string) (string, error) {
    // Extract filename from current path
    filename := filepath.Base(currentPath)

    // Show save dialog
    path, err := app.Dialog.SaveFile().
        SetTitle("Save As").
        SetFilename(filename).
        PromptForSingleSelection()

    if err != nil {
        return "", err
    }

    return path, nil
}
```

### Modèle « Ouvrir récent »

```go
func (s *Service) OpenRecent(app *application.App, recentPath string) error {
    // Check if file still exists
    if _, err := os.Stat(recentPath); os.IsNotExist(err) {
        dialog := app.Dialog.Question().
            SetTitle("File Not Found").
            SetMessage("The file no longer exists. Remove from recent files?")

        remove := dialog.AddButton("Remove")
        remove.OnClick(func() {
            s.removeFromRecent(recentPath)
        })

        cancel := dialog.AddButton("Cancel")
        dialog.SetCancelButton(cancel)
        dialog.Show()

        return err
    }

    return s.openFile(recentPath)
}
