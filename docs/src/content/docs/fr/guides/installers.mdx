## Vue d'ensemble

Créez des installateurs professionnels pour votre application Wails sur toutes les plateformes.

## Installateurs par plateforme

<Tabs syncKey="platform">
  <TabItem label="Windows" icon="seti:windows">
    ### Installateur NSIS
    
    ```bash
    # Install NSIS
    # Download from: https://nsis.sourceforge.io/
    
    # Create installer script (installer.nsi)
    makensis installer.nsi
    ```
    
    **installer.nsi:**
    ```nsis
    !define APPNAME "MyApp"
    !define VERSION "1.0.0"
    
    Name "${APPNAME}"
    OutFile "MyApp-Setup.exe"
    InstallDir "$PROGRAMFILES\${APPNAME}"
    
    Section "Install"
        SetOutPath "$INSTDIR"
        File "build\bin\myapp.exe"
        CreateShortcut "$DESKTOP\${APPNAME}.lnk" "$INSTDIR\myapp.exe"
    SectionEnd
    ```
    
    ### WiX Toolset
    
    Alternative pour les installateurs MSI.
  </TabItem>

  <TabItem label="macOS" icon="apple">
    ### Création de DMG
    
    ```bash
    # Create DMG
    hdiutil create -volname "MyApp" -srcfolder build/bin/MyApp.app -ov -format UDZO MyApp.dmg
    ```
    
    ### Signature de code
    
    ```bash
    # Sign application
    codesign --deep --force --verify --verbose --sign "Developer ID" MyApp.app
    
    # Notarize
    xcrun notarytool submit MyApp.dmg --apple-id "email" --password "app-password"
    ```
    
    ### App Store
    
    Utilisez Xcode pour la distribution sur l'App Store.
  </TabItem>

  <TabItem label="Linux" icon="linux">
    ### Package DEB
    
    ```bash
    # Create package structure
    mkdir -p myapp_1.0.0/DEBIAN
    mkdir -p myapp_1.0.0/usr/bin
    
    # Copy binary
    cp build/bin/myapp myapp_1.0.0/usr/bin/
    
    # Create control file
    cat > myapp_1.0.0/DEBIAN/control << EOF
    Package: myapp
    Version: 1.0.0
    Architecture: amd64
    Maintainer: Your Name
    Description: My Application
    EOF
    
    # Build package
    dpkg-deb --build myapp_1.0.0
    ```
    
    ### Package RPM
    
    Utilisez `rpmbuild` pour les distributions basées sur RPM.
    
    ### AppImage
    
    ```bash
    # Use appimagetool
    appimagetool myapp.AppDir
    ```
  </TabItem>
</Tabs>

## Empaquetage automatisé

### Utilisation de GoReleaser

```yaml
# .goreleaser.yml
project_name: myapp

builds:
  - binary: myapp
    goos:
      - windows
      - darwin
      - linux
    goarch:
      - amd64
      - arm64

archives:
  - format: zip
    name_template: "{{ .ProjectName }}_{{ .Version }}_{{ .Os }}_{{ .Arch }}"

nfpms:
  - formats:
      - deb
      - rpm
    vendor: Your Company
    homepage: https://example.com
    description: My Application
```

## Bonnes pratiques

### ✅ À faire

- Signer le code sur toutes les plateformes
- Inclure les informations de version
- Créer des désinstallateurs
- Tester le processus d'installation
- Fournir une documentation claire

### ❌ À ne pas faire

- Ne pas sauter la signature de code
- Ne pas oublier les associations de fichiers
- Ne pas coder en dur les chemins
- Ne pas sauter les tests
