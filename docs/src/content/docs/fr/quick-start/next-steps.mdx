## Fonctionnalités

Wails offre des capacités de bureau natif :

- [Windows](/features/windows/basics) - Fenêtres multiples, fenêtres sans bordure, positionnement
- [Menus](/features/menus/application) - Menus d'application, menus contextuels, barre d'état système
- [Boîtes de dialogue](/features/dialogs/file) - Ouverture/sauvegarde de fichiers, boîtes de dialogue de message
- [Événements](/features/events/system) - Communication entre composants
- [Liaisons](/features/bindings/methods) - Appels Go ↔ JavaScript typés
- [Presse-papiers](/features/clipboard) - Opérations de copie/coller
- [Glisser-déposer](/features/drag-and-drop/files) - Glisser-déposer de fichiers
- [Clavier](/features/keyboard) - Raccourcis globaux
