## Fenêtres sans bordure

Wails propose **la prise en charge des fenêtres sans bordure** avec des zones de glissement basées sur CSS et un comportement natif à la plateforme. Supprimez la barre de titre native de la plateforme pour avoir un contrôle total sur l'apparence de la fenêtre, des designs personnalisés et des expériences utilisateur uniques, tout en conservant les fonctionnalités essentielles telles que le déplacement, le redimensionnement et les contrôles système.

## Démarrage rapide

```go
window := app.Window.NewWithOptions(application.WebviewWindowOptions{
    Title:     "Frameless App",
    Width:     800,
    Height:    600,
    Frameless: true,
})
```

**CSS pour la barre de titre déplaçable :**

```css
.titlebar {
    --wails-draggable: drag;
    height: 40px;
    background: #333;
}

.titlebar button {
    --wails-draggable: no-drag;
}
```

**HTML :**

```html
<div class="titlebar">
    <span>My Application</span>
    <button onclick="window.close()">×</button>
</div>
```

**C'est tout !** Vous avez une barre de titre personnalisée.

## Création de fenêtres sans bordure

### Fenêtre sans bordure de base

```go
window := app.Window.NewWithOptions(application.WebviewWindowOptions{
    Frameless: true,
    Width:     800,
    Height:    600,
})
```

**Ce que vous obtenez :**
- Pas de barre de titre
- Pas de bordures de fenêtre
- Pas de boutons système
- Arrière-plan transparent (optionnel)

**Ce que vous devez implémenter :**
- Zone déplaçable
- Boutons Fermer/Réduire/Agrandir
- Poignées de redimensionnement (si redimensionnable)

### Avec arrière-plan transparent

```go
window := app.Window.NewWithOptions(application.WebviewWindowOptions{
    Frameless:      true,
    BackgroundType: application.BackgroundTypeTransparent,
})
```

**Cas d'utilisation :**
- Coins arrondis
- Formes personnalisées
- Fenêtres superposées
- Écrans de démarrage

## Zones de glissement

### Glissement basé sur CSS

Utilisez la propriété CSS `--wails-draggable` :

```css
/* Zone déplaçable */
.titlebar {
    --wails-draggable: drag;
}

/* Éléments non déplaçables dans la zone déplaçable */
.titlebar button {
    --wails-draggable: no-drag;
}
```

**Valeurs :**
- `drag` - La zone est déplaçable
- `no-drag` - La zone n'est pas déplaçable (même si le parent l'est)


## Boutons système

### Implémentation de Fermer/Réduire/Agrandir

**Côté Go :**

```go
type WindowControls struct {
    window *application.WebviewWindow
}

func (wc *WindowControls) Minimise() {
    wc.window.Minimise()
}

func (wc *WindowControls) Maximise() {
    if wc.window.IsMaximised() {
        wc.window.UnMaximise()
    } else {
        wc.window.Maximise()
    }
}

func (wc *WindowControls) Close() {
    wc.window.Close()
}
```

**Côté JavaScript :**

```javascript
import { Minimise, Maximise, Close } from './bindings/WindowControls'

document.querySelector('.minimize').addEventListener('click', Minimise)
document.querySelector('.maximize').addEventListener('click', Maximise)
document.querySelector('.close').addEventListener('click', Close)
```

**Ou utilisez les méthodes du runtime :**

```javascript
import { Window } from '@wailsio/runtime'

document.querySelector('.minimize').addEventListener('click', () => Window.Minimise())
document.querySelector('.maximize').addEventListener('click', () => Window.Maximise())
document.querySelector('.close').addEventListener('click', () => Window.Close())
```

### Basculer l'état Agrandir

Suivez l'état d'agrandissement pour l'icône du bouton :

```javascript
import { Window } from '@wailsio/runtime'

async function toggleMaximise() {
    const isMaximised = await Window.IsMaximised()

    if (isMaximised) {
        await Window.Restore()
    } else {
        await Window.Maximise()
    }

    updateMaximiseButton()
}

async function updateMaximiseButton() {
    const isMaximised = await Window.IsMaximised()
    const button = document.querySelector('.maximize')
    button.textContent = isMaximised ? '❐' : '□'
}
```

## Poignées de redimensionnement

### Redimensionnement basé sur CSS

Wails fournit des poignées de redimensionnement automatiques pour les fenêtres sans bordure :

```css
/* Activer le redimensionnement sur tous les bords */
body {
    --wails-resize: all;
}

/* Ou sur des bords spécifiques */
.resize-top {
    --wails-resize: top;
}

.resize-bottom {
    --wails-resize: bottom;
}

.resize-left {
    --wails-resize: left;
}

.resize-right {
    --wails-resize: right;
}

/* Coins */
.resize-top-left {
    --wails-resize: top-left;
}

.resize-top-right {
    --wails-resize: top-right;
}

.resize-bottom-left {
    --wails-resize: bottom-left;
}

.resize-bottom-right {
    --wails-resize: bottom-right;
}
```

**Valeurs :**
- `all` - Redimensionnement depuis tous les bords
- `top`, `bottom`, `left`, `right` - Bords spécifiques
- `top-left`, `top-right`, `bottom-left`, `bottom-right` - Coins
- `none` - Pas de redimensionnement

### Exemple de poignée de redimensionnement

```html
<div class="window">
    <div class="titlebar">...</div>
    <div class="content">...</div>
    <div class="resize-handle resize-bottom-right"></div>
</div>
```

```css
.resize-handle {
    position: absolute;
    width: 16px;
    height: 16px;
}

.resize-bottom-right {
    --wails-resize: bottom-right;
    bottom: 0;
    right: 0;
    cursor: nwse-resize;
}
```

## Exemple complet

Voici une fenêtre sans bordure prête pour la production :

**Go :**

```go
package main

import (
    _ "embed"
    "github.com/wailsapp/wails/v3/pkg/application"
)

//go:embed frontend/dist
var assets embed.FS

func main() {
    app := application.New(application.Options{
        Name: "Frameless App",
    })

    window := app.Window.NewWithOptions(application.WebviewWindowOptions{
        Title:     "Frameless Application",
        Width:     1000,
        Height:    700,
        MinWidth:  800,
        MinHeight: 600,
        Frameless: true,
        
        Assets: application.AssetOptions{
            Handler: application.AssetFileServerFS(assets),
        },
        
        Mac: application.MacOptions{
            TitleBarAppearsTransparent: true,
            InvisibleTitleBarHeight:    40,
        },
        
        Windows: application.WindowsOptions{
            DisableFramelessWindowDecorations: false,
        },
    })

    window.Center()
    window.Show()

    app.Run()
}
```

## Bonnes pratiques

### ✅ À faire

- **Fournir une zone de glissement** - Les utilisateurs ont besoin de pouvoir déplacer la fenêtre
- **Implémenter les boutons système** - Fermer, minimiser, maximiser
- **Définir une taille minimale** - Éviter des mises en page inutilisables
- **Tester sur toutes les plateformes** - Le comportement varie
- **Utiliser CSS pour les zones de glissement** - Flexible et maintenable
- **Fournir un retour visuel** - États au survol sur les boutons

### ❌ À ne pas faire

- **Ne pas oublier les poignées de redimensionnement** - Si la fenêtre est redimensionnable
- **Ne pas rendre toute la fenêtre déplaçable** - Cela empêche l'interaction
- **Ne pas oublier `no-drag` sur les boutons** - Ils ne fonctionneront pas---
