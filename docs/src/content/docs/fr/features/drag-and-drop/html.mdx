Le glisser-déposer HTML5 permet aux utilisateurs de faire glisser des éléments dans l'interface utilisateur de votre application - par exemple, pour réorganiser une liste ou déplacer des éléments entre des colonnes. Il s'agit d'une fonctionnalité web standard qui fonctionne dans Wails sans configuration spéciale.

## Rendre un élément glissable

Par défaut, la plupart des éléments ne peuvent pas être déplacés par glisser-déposer. Pour rendre un élément glissable, ajoutez `draggable="true"` :

```html
<div class="item" draggable="true">Glissez-moi</div>
```

L'élément affichera désormais un aperçu du glissement lorsque l'utilisateur clique et fait glisser.

## Définir une zone de dépôt

Les éléments n'acceptent pas les dépôts par défaut. Pour qu'un élément accepte les dépôts, vous devez annuler le comportement par défaut sur `dragover` :

```html
<div class="drop-zone" id="target">Déposez ici</div>

<script>
const target = document.getElementById('target');

target.addEventListener('dragover', (e) => {
    e.preventDefault(); // Autoriser le dépôt
});

target.addEventListener('drop', (e) => {
    e.preventDefault();
    // Gérer le dépôt
});
</script>
```

Appeler `preventDefault()` sur `dragover` est requis - cela signale que cet élément accepte les dépôts. Sans cela, l'événement de dépôt ne se déclenchera pas.

## Styliser le survol de glissement

Pour montrer aux utilisateurs où ils peuvent déposer des éléments, ajoutez un retour visuel lors du survol d'une zone de dépôt. L'événement `dragenter` se déclenche lorsqu'un élément entre dans la zone, et `dragleave` se déclenche lorsqu'il en sort :

```css
.drop-zone {
    border: 2px dashed #ccc;
    padding: 40px;
    transition: all 0.2s ease;
}

.drop-zone.drag-over {
    border-color: #007bff;
    background-color: rgba(0, 123, 255, 0.1);
}
```

```javascript
const target = document.getElementById('target');

target.addEventListener('dragenter', () => {
    target.classList.add('drag-over');
});

target.addEventListener('dragleave', () => {
    target.classList.remove('drag-over');
});

target.addEventListener('drop', (e) => {
    e.preventDefault();
    target.classList.remove('drag-over');
    // Gérer le dépôt
});
```

Remarque : `dragleave` se déclenche également lors de l'entrée dans un élément enfant, ce qui peut provoquer des clignotements. L'exemple complet ci-dessous montre comment gérer ce cas.

## Exemple complet

Une liste de tâches où les éléments peuvent être déplacés entre des colonnes de priorité. Cela suit l'élément en cours de glissement dans une variable, ce qui est l'approche la plus simple lorsque tout se trouve sur la même page :

```html
<div class="tasks">
    <div class="item" draggable="true">Corriger le bug de connexion</div>
    <div class="item" draggable="true">Mettre à jour la documentation</div>
    <div class="item" draggable="true">Ajouter le mode sombre</div>
</div>

<div class="columns">
    <div class="drop-zone" data-priority="high">
        <h3>Priorité élevée</h3>
        <ul></ul>
    </div>
    <div class="drop-zone" data-priority="low">
        <h3>Priorité faible</h3>
        <ul></ul>
    </div>
</div>

<script>
let draggedItem = null;

// Suivre quel élément est en cours de glissement
document.querySelectorAll('.item').forEach(item => {
    item.addEventListener('dragstart', () => {
        draggedItem = item;
        item.classList.add('dragging');
    });
    
    item.addEventListener('dragend', () => {
        item.classList.remove('dragging');
    });
});

// Gérer les dépôts sur chaque zone
document.querySelectorAll('.drop-zone').forEach(zone => {
    zone.addEventListener('dragover', (e) => {
        e.preventDefault();
    });
    
    zone.addEventListener('dragenter', () => {
        zone.classList.add('drag-over');
    });
    
    zone.addEventListener('dragleave', (e) => {
        // Ne retirer la classe que si on quitte complètement la zone,
        // pas seulement en entrant dans un élément enfant
        if (!zone.contains(e.relatedTarget)) {
            zone.classList.remove('drag-over');
        }
    });
    
    zone.addEventListener('drop', (e) => {
        e.preventDefault();
        zone.classList.remove('drag-over');
        
        if (draggedItem) {
            const li = document.createElement('li');
            li.textContent = draggedItem.textContent;
            zone.querySelector('ul').appendChild(li);
            draggedItem.remove();
        }
    });
});
</script>

<style>
.item {
    padding: 12px 16px;
    background: #f0f0f0;
    margin: 8px 0;
    border-radius: 8px;
    cursor: grab;
}

.item.dragging {
    opacity: 0.5;
}

.drop-zone {
    min-height: 150px;
    border: 2px dashed #ccc;
    border-radius: 8px;
    padding: 15px;
    transition: all 0.2s ease;
}

.drop-zone.drag-over {
    border-color: #007bff;
    background: rgba(0, 123, 255, 0.1);
}
</style>
```

## Combiner avec le dépôt de fichiers

Si votre application utilise à la fois le glisser-déposer HTML et le [dépôt de fichiers](./files), vos zones de dépôt HTML recevront également des événements lorsque les utilisateurs feront glisser des fichiers depuis le système d'exploitation. Pour éviter toute confusion, filtrez les glissements de fichiers externes dans vos gestionnaires :

```javascript
zone.addEventListener('dragenter', (e) => {
    // Ignorer les glissements de fichiers externes
    if (e.dataTransfer?.types.includes('Files')) return;
    
    zone.classList.add('drag-over');
});

zone.addEventListener('dragover', (e) => {
    // Ignorer les glissements de fichiers externes
    if (e.dataTransfer?.types.includes('Files')) return;
    
    e.preventDefault();
});

zone.addEventListener('drop', (e) => {
    // Ignorer les glissements de fichiers externes
    if (e.dataTransfer?.types.includes('Files')) return;
    
    e.preventDefault();
    zone.classList.remove('drag-over');
    // Gérer le dépôt interne
});
```

Le tableau `dataTransfer.types` contient `'Files'` lorsque l'utilisateur fait glisser des fichiers depuis le système d'exploitation, mais contient des types comme `'text/plain'` pour les glissements HTML internes. Cela vous permet de distinguer les deux cas.

## Transmettre des données avec dataTransfer

L'exemple ci-dessus suit l'élément en cours de glissement dans une variable JavaScript. Cela fonctionne bien lorsque tout se trouve sur la même page. Mais si vous devez glisser entre des iframes ou transmettre des données qui ne sont pas liées à un élément DOM, utilisez l'API `dataTransfer` :

```javascript
// Lorsque le glissement commence, stocker les données
item.addEventListener('dragstart', (e) => {
    e.dataTransfer.setData('text/plain', item.id);
});

// Lorsque le dépôt a lieu, récupérer les données
target.addEventListener('drop', (e) => {
    e.preventDefault();
    const itemId = e.dataTransfer.getData('text/plain');
    const item = document.getElementById(itemId);
    // Déplacer ou copier l'élément
});
```

Les données sont stockées sous forme de chaînes, vous devrez donc sérialiser les objets avec `JSON.stringify()` si nécessaire.
