Wails propose un puissant système de liaison de touches qui vous permet d'enregistrer des raccourcis clavier globaux fonctionnant sur toutes les fenêtres de votre application. Cela permet aux utilisateurs d'accéder rapidement aux fonctionnalités sans avoir à naviguer dans les menus.

## Accéder au gestionnaire de liaisons de touches

Le gestionnaire de liaisons de touches est accessible via la propriété `KeyBindings` de votre instance d'application :

```go
app := application.New(application.Options{
    Name: "Keyboard Shortcuts Demo",
})

// Access the key binding manager
keyBindings := app.KeyBinding
```

## Ajouter des liaisons de touches

### Liaison de touches de base

Enregistrez un raccourci clavier simple :

```go
app.KeyBinding.Add("Ctrl+S", func(window application.Window) {
    // Handle save action
    app.Logger.Info("Save shortcut triggered")
    // Perform save operation...
})
```

### Liaisons de touches multiples

Enregistrez plusieurs raccourcis pour des opérations courantes :

```go
// File operations
app.KeyBinding.Add("Ctrl+N", func(window application.Window) {
    // New file
    window.EmitEvent("file:new", nil)
})

app.KeyBinding.Add("Ctrl+O", func(window application.Window) {
    // Open file
    dialog := app.Dialog.OpenFile()
    if file, err := dialog.PromptForSingleSelection(); err == nil {
        window.EmitEvent("file:open", file)
    }
})

app.KeyBinding.Add("Ctrl+S", func(window application.Window) {
    // Save file
    window.EmitEvent("file:save", nil)
})

// Edit operations
app.KeyBinding.Add("Ctrl+Z", func(window application.Window) {
    // Undo
    window.EmitEvent("edit:undo", nil)
})

app.KeyBinding.Add("Ctrl+Y", func(window application.Window) {
    // Redo (Windows/Linux)
    window.EmitEvent("edit:redo", nil)
})

app.KeyBinding.Add("Cmd+Shift+Z", func(window application.Window) {
    // Redo (macOS)
    window.EmitEvent("edit:redo", nil)
})
```

## Accélérateurs de liaison de touches

### Format de l'accélérateur

Les liaisons de touches utilisent un format d'accélérateur standard avec des modificateurs et des touches :

```go
// Modifier keys
"Ctrl+S"        // Control + S
"Cmd+S"         // Command + S (macOS)
"Alt+F4"        // Alt + F4
"Shift+Ctrl+Z"  // Shift + Control + Z

// Function keys
"F1"            // F1 key
"Ctrl+F5"       // Control + F5

// Special keys
"Escape"        // Escape key
"Enter"         // Enter key
"Space"         // Spacebar
"Tab"           // Tab key
"Backspace"     // Backspace key
"Delete"        // Delete key

// Arrow keys
"Up"            // Up arrow
"Down"          // Down arrow
"Left"          // Left arrow
"Right"         // Right arrow
```

### Accélérateurs spécifiques à la plateforme

Gérez les différences de plateforme pour les raccourcis courants :

```go
import "runtime"

// Cross-platform save shortcut
if runtime.GOOS == "darwin" {
    app.KeyBinding.Add("Cmd+S", saveHandler)
} else {
    app.KeyBinding.Add("Ctrl+S", saveHandler)
}

// Or register both
app.KeyBinding.Add("Ctrl+S", saveHandler)
app.KeyBinding.Add("Cmd+S", saveHandler)
```

## Gérer les liaisons de touches

### Supprimer des liaisons de touches

Supprimez les liaisons de touches lorsqu'elles ne sont plus nécessaires :

```go
// Remove a specific key binding
app.KeyBinding.Remove("Ctrl+S")

// Example: Temporary key binding for a modal
app.KeyBinding.Add("Escape", func(window application.Window) {
    // Close modal
    window.EmitEvent("modal:close", nil)
    // Remove this temporary binding
    app.KeyBinding.Remove("Escape")
})
```

### Obtenir toutes les liaisons de touches

Récupérez toutes les liaisons de touches enregistrées :

```go
allBindings := app.KeyBinding.GetAll()
for _, binding := range allBindings {
    app.Logger.Info("Key binding", "accelerator", binding.Accelerator)
}
```

## Utilisation avancée

### Liaisons de touches contextuelles

Rendez les liaisons de touches sensibles au contexte en vérifiant l'état de l'application :

```go
app.KeyBinding.Add("Ctrl+S", func(window application.Window) {
    // Check current application state
    if isEditMode() {
        // Save document
        saveDocument()
    } else if isInSettings() {
        // Save settings
        saveSettings()
    } else {
        app.Logger.Info("Save not available in current context")
    }
})
```

### Actions spécifiques à la fenêtre

Les liaisons de touches reçoivent la fenêtre active, permettant un comportement spécifique à la fenêtre :

```go
app.KeyBinding.Add("F11", func(window application.Window) {
    // Toggle fullscreen for the active window
    if window.Fullscreen() {
        window.SetFullscreen(false)
    } else {
        window.SetFullscreen(true)
    }
})

app.KeyBinding.Add("Ctrl+W", func(window application.Window) {
    // Close the active window
    window.Close()
})
```

### Gestion dynamique des liaisons de touches

Ajoutez et supprimez dynamiquement des liaisons de touches en fonction de l'état de l'application :

```go
func enableEditMode() {
    // Add edit-specific key bindings
    app.KeyBinding.Add("Ctrl+B", func(window application.Window) {
        window.EmitEvent("format:bold", nil)
    })
    
    app.KeyBinding.Add("Ctrl+I", func(window application.Window) {
        window.EmitEvent("format:italic", nil)
    })
    
    app.KeyBinding.Add("Ctrl+U", func(window application.Window) {
        window.EmitEvent("format:underline", nil)
    })
}

func disableEditMode() {
    // Remove edit-specific key bindings
    app.KeyBinding.Remove("Ctrl+B")
    app.KeyBinding.Remove("Ctrl+I")
    app.KeyBinding.Remove("Ctrl+U")
}
```

## Bonnes pratiques

1. **Utilisez des raccourcis standards** : Suivez les conventions de la plateforme pour les opérations courantes :
   ```go
   // Cross-platform save
   if runtime.GOOS == "darwin" {app.KeyBinding.Add("Cmd+S", saveHandler)
   } else {
       app.KeyBinding.Add("Ctrl+S", saveHandler)
   }
   ```

2. **Fournir un retour visuel** : Informez les utilisateurs lorsque des raccourcis sont déclenchés :
   ```go
   app.KeyBinding.Add("Ctrl+S", func(window application.Window) {
       saveDocument()
       // Afficher une notification brève
       window.EmitEvent("notification:show", "Document enregistré")
   })
   ```

3. **Gérer les conflits** : Faites attention à ne pas écraser les raccourcis système importants :
   ```go
   // Éviter d'écraser les raccourcis système tels que :
   // Ctrl+Alt+Del (Windows)
   // Cmd+Espace (Spotlight sur macOS)
   // Alt+Tab (Changement de fenêtre)
   ```

4. **Documenter les raccourcis** : Fournissez une aide ou une documentation pour les raccourcis disponibles :
   ```go
   app.KeyBinding.Add("F1", func(window application.Window) {
       // Afficher la boîte de dialogue d'aide avec les raccourcis disponibles
       showKeyboardShortcutsHelp()
   })
   ```

5. **Nettoyer** : Supprimez les liaisons de touches temporaires lorsqu'elles ne sont plus nécessaires :
   ```go
   func enterEditMode() {
       app.KeyBinding.Add("Escape", exitEditModeHandler)
   }
   
   func exitEditModeHandler(window application.Window) {
       exitEditMode()
       app.KeyBinding.Remove("Escape") // Nettoyage de la liaison temporaire
   }
   ```

## Exemple complet

Voici un exemple complet d'un éditeur de texte avec des raccourcis clavier :

```go
package main

import (
    "runtime"
    "github.com/wailsapp/wails/v3/pkg/application"
)

func main() {
    app := application.New(application.Options{
        Name: "Text Editor with Shortcuts",
    })

    // Opérations sur les fichiers
    if runtime.GOOS == "darwin" {
        app.KeyBinding.Add("Cmd+N", func(window application.Window) {
            window.EmitEvent("file:new", nil)
        })
        app.KeyBinding.Add("Cmd+O", func(window application.Window) {
            openFile(app, window)
        })
        app.KeyBinding.Add("Cmd+S", func(window application.Window) {
            window.EmitEvent("file:save", nil)
        })
    } else {
        app.KeyBinding.Add("Ctrl+N", func(window application.Window) {
            window.EmitEvent("file:new", nil)
        })
        app.KeyBinding.Add("Ctrl+O", func(window application.Window) {
            openFile(app, window)
        })
        app.KeyBinding.Add("Ctrl+S", func(window application.Window) {
            window.EmitEvent("file:save", nil)
        })
    }

    // Opérations d'affichage
    app.KeyBinding.Add("F11", func(window application.Window) {
        window.SetFullscreen(!window.Fullscreen())
    })

    app.KeyBinding.Add("F1", func(window application.Window) {
        showKeyboardShortcuts(window)
    })

    // Créer la fenêtre principale
    window := app.Window.New()
    window.SetTitle("Text Editor")

    err := app.Run()
    if err != nil {
        panic(err)
    }
}

func openFile(app *application.App, window application.Window) {
    dialog := app.Dialog.OpenFile()
    dialog.AddFilter("Text Files", "*.txt;*.md")
    
    if file, err := dialog.PromptForSingleSelection(); err == nil {
        window.EmitEvent("file:open", file)
    }
}

func showKeyboardShortcuts(window application.Window) {
    shortcuts := `
Keyboard Shortcuts:
- Ctrl/Cmd+N: New file
- Ctrl/Cmd+O: Open file
- Ctrl/Cmd+S: Save file
- F11: Toggle fullscreen
- F1: Show this help
`
    window.EmitEvent("help:show", shortcuts)
}
```

:::tip[Conseil de pro]
Testez vos liaisons de touches sur toutes les plateformes cibles pour vous assurer qu'elles fonctionnent correctement et ne entrent pas en conflit avec les raccourcis système.
:::

:::danger[Attention]
Faites attention à ne pas écraser les raccourcis système critiques. Certaines combinaisons de touches sont réservées par le système d'exploitation et ne peuvent pas être capturées par les applications.
:::
