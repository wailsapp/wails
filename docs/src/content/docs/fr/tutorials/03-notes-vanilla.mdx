Dans ce tutoriel, vous allez construire une application de notes qui démontre les opérations sur les fichiers, les boîtes de dialogue natives et les modèles d'applications de bureau modernes en utilisant Wails v3.

![Capture d'écran de l'application Notes](../../../../assets/notes-app.png)

## Ce que vous allez construire

- Une application de notes complète avec des fonctionnalités de création, de modification et de suppression
- Des boîtes de dialogue natives pour enregistrer/ouvrir des fichiers afin d'importer et d'exporter des notes
- Un enregistrement automatique lors de la frappe avec un debounce pour réduire les mises à jour inutiles
- Une mise en page professionnelle à deux colonnes (barre latérale + éditeur) imitant Apple Notes

## Ce que vous allez apprendre

- Utilisation des boîtes de dialogue natives dans Wails (`SaveFileDialog`, `OpenFileDialog`, `InfoDialog`)
- Travail avec des fichiers JSON pour la persistance des données
- Mise en œuvre de modèles d'enregistrement automatique avec debounce
- Construction d'interfaces utilisateur de bureau professionnelles avec du CSS moderne
- Sérialisation JSON correcte des structs Go

2. **Créer le NotesService**

   Créez un nouveau fichier `notesservice.go` à la racine du projet :

   ```go
   package main

   import (
   	"encoding/json"
   	"errors"
   	"os"
   	"time"

   	"github.com/wailsapp/wails/v3/pkg/application"
   )

   type Note struct {
   	ID        string    `json:"id"`
   	Title     string    `json:"title"`
   	Content   string    `json:"content"`
   	CreatedAt time.Time `json:"createdAt"`
   	UpdatedAt time.Time `json:"updatedAt"`
   }

   type NotesService struct {
   	notes []Note
   }

   func NewNotesService() *NotesService {
   	return &NotesService{
   		notes: make([]Note, 0),
   	}
   }

   // GetAll retourne toutes les notes
   func (n *NotesService) GetAll() []Note {
   	return n.notes
   }

   // Create crée une nouvelle note
   func (n *NotesService) Create(title, content string) Note {
   	note := Note{
   		ID:        generateID(),
   		Title:     title,
   		Content:   content,
   		CreatedAt: time.Now(),
   		UpdatedAt: time.Now(),
   	}
   	n.notes = append(n.notes, note)
   	return note
   }

   // Update met à jour une note existante
   func (n *NotesService) Update(id, title, content string) error {
   	for i := range n.notes {
   		if n.notes[i].ID == id {
   			n.notes[i].Title = title
   			n.notes[i].Content = content
   			n.notes[i].UpdatedAt = time.Now()
   			return nil
   		}
   	}
   	return errors.New("note not found")
   }

   // Delete supprime une note
   func (n *NotesService) Delete(id string) error {
   	for i := range n.notes {
   		if n.notes[i].ID == id {
   			n.notes = append(n.notes[:i], n.notes[i+1:]...)
   			return nil
   		}
   	}
   	return errors.New("note not found")
   }

   // SaveToFile enregistre les notes dans un fichier
   func (n *NotesService) SaveToFile() error {
   	path, err := application.Get().Dialog.SaveFile().
   		SetFilename("notes.json").
   		AddFilter("JSON Files", "*.json").
   		PromptForSingleSelection()

   	if err != nil {
   		return err
   	}

   	data, err := json.MarshalIndent(n.notes, "", "  ")
   	if err != nil {
   		return err
   	}

   	if err := os.WriteFile(path, data, 0644); err != nil {
   		return err
   	}

   	application.Get().Dialog.Info().
   		SetTitle("Success").
   		SetMessage("Notes saved successfully!").
   		Show()

   	return nil
   }

   // LoadFromFile charge les notes depuis un fichier
   func (n *NotesService) LoadFromFile() error {
   	path, err := application.Get().Dialog.OpenFile().
   		AddFilter("JSON Files", "*.json").
   		PromptForSingleSelection()

   	if err != nil {
   		return err
   	}

   	data, err := os.ReadFile(path)
   	if err != nil {
   		return err
   	}

   	var notes []Note
   	if err := json.Unmarshal(data, &notes); err != nil {
   		return err
   	}

   	n.notes = notes

   	application.Get().Dialog.Info().
   		SetTitle("Success").
   		SetMessage("Notes loaded successfully!").
   		Show()

   	return nil
   }

   func generateID() string {
   	return time.Now().Format("20060102150405")
   }
   ```

   **Ce qui se passe ici :**

   - **Struct Note** : Définit la structure de données avec des balises JSON (minuscules) pour une sérialisation correcte
   - **Opérations CRUD** : GetAll, Create, Update et Delete pour gérer les notes en mémoire
   - **Boîtes de dialogue de fichiers** : Utilise `application.Get().Dialog.SaveFile()` et `application.Get().Dialog.OpenFile()` pour accéder aux boîtes de dialogue natives
   - **Boîtes de dialogue Info** : Affiche des messages de succès en utilisant `application.Get().Dialog.Info()`
   - **Génération d'ID** : Générateur d'ID simple basé sur l'horodatage

   let notes = []

   // Load notes on startup
   async function loadNotes() {
       notes = await NotesService.GetAll()
       renderNotesList()

   function renderNotesList() {
       const notesList = document.getElementById('notes-list')

       if (notes.length === 0) {
           notesList.innerHTML = '<div style="padding: 20px; text-align: center; color: #999;">No notes yet</div>'
           return

               <h3>${note.title || 'Untitled'}</h3>
               <p>${note.content || 'No content'}</p>
           </div>
       `).join('')

       // Add click handlers
       document.querySelectorAll('.note-item').forEach(item => {
           item.addEventListener('click', () => {
               const id = item.dataset.id
               selectNote(id)
           })
       })
   }

   // Select a note
   function selectNote(id) {
       currentNote = notes.find(n => n.id === id)
       if (currentNote) {
           document.getElementById('empty-state').style.display = 'none'
           document.getElementById('note-editor').style.display = 'flex'
           document.getElementById('note-title').value = currentNote.title
           document.getElementById('note-content').value = currentNote.content

               `Last updated: ${new Date(currentNote.updatedAt).toLocaleString()}`
           renderNotesList()
       }

       notes.push(note)
       selectNote(note.id)
       // Focus the title input and select all text so user can immediately type

       titleInput.focus()
       titleInput.select()

                   note.updatedAt = new Date().toISOString()

## Concepts clés

### Fonctions de dialogue de niveau package

Dans Wails v3, les boîtes de dialogue de fichiers et de messages sont des fonctions de niveau package, et non des méthodes de l'instance de l'application :

```go
// Correct - fonction de niveau package
path, err := application.SaveFileDialog().
    SetFilename("notes.json").
    AddFilter("Fichiers JSON", "*.json").
    PromptForSingleSelection()

// Incorrect - non disponible
path, err := app.Dialog.SaveFile() // Cela n'existe pas
```

### Mappage des balises JSON

Les balises JSON des structures Go doivent être en minuscules pour correspondre à l'accès aux propriétés JavaScript :

```go
type Note struct {
    ID string `json:"id"` // Doit être en minuscules
}
```

```javascript
// JavaScript accède avec des minuscules
const noteId = note.id // Correct
const noteId = note.ID // Serait undefined
```

### Sauvegarde automatique avec debounce

Le debounce de 500 ms réduit les appels inutiles au backend :

```javascript
let updateTimeout
function scheduleUpdate() {
    clearTimeout(updateTimeout) // Annule le timer précédent
    updateTimeout = setTimeout(async () => {
        // Sauvegarde uniquement si l'utilisateur arrête de taper pendant 500 ms
        await NotesService.Update(currentNote.id, title, content)
    }, 500)
}
```
