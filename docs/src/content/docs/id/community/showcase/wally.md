[Wally](https://ergodox-ez.com/pages/wally) adalah flasher firmware resmi untuk
keyboard [Ergodox](https://ergodox-ez.com/). Tampilannya bagus dan merupakan contoh
fantastis dari apa yang dapat Anda capai dengan Wails: kemampuan untuk menggabungkan kekuatan
Go dan alat grafis kaya dari dunia pengembangan web.
