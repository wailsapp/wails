[Packsmith](https://github.com/sqot0/packsmith) adalah aplikasi open source untuk mengelola modpack Minecraft yang memungkinkan Anda:

- Mengelola modpack dengan mod loader Forge, Fabric, Neoforge, atau Quilt.
- Mencari dan mengunduh mod dari Modrinth dan CurseForge.
- Memisahkan mod untuk client dan server.
- Memperbarui, mengubah versi, atau menghapus mod dari modpack Anda.
