[LTools](https://github.com/lian-yang/ltools) adalah toolbox desktop cross-platform berbasis plugin yang dibangun dengan Wails v3, terinspirasi oleh uTools. Menyediakan hub terpadu utilitas developer melalui arsitektur plugin modular.

Fitur search bar global (gaya Spotlight), integrasi system tray, dan plugin bawaan termasuk clipboard manager, tool screenshot dengan anotasi, editor JSON, kalkulator, system monitor, dan lainnya. Tool baru dapat ditambahkan melalui sistem plugin yang extensible.
