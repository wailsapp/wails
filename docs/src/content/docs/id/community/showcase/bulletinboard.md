Aplikasi [BulletinBoard](https://github.com/raguay/BulletinBoard) adalah
papan pesan serbaguna untuk pesan statis atau dialog untuk mendapatkan informasi dari
pengguna untuk script. Memiliki TUI untuk membuat dialog baru yang nantinya dapat
digunakan untuk mendapatkan informasi dari pengguna. Desainnya adalah tetap berjalan di
sistem Anda dan menampilkan informasi sesuai kebutuhan lalu bersembunyi. Saya memiliki proses
untuk memantau file di sistem saya dan mengirim kontennya ke BulletinBoard saat
berubah. Berfungsi sangat baik dengan workflow saya. Ada juga
[Alfred workflow](https://github.com/raguay/MyAlfred/blob/master/Alfred%205/EmailIt.alfredworkflow)
untuk mengirim informasi ke program. Workflow ini juga untuk bekerja dengan
[EmailIt](https://github.com/raguay/EmailIt).
