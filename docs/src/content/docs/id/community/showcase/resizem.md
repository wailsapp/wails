[Resizem](https://github.com/barats/resizem) - adalah aplikasi yang dirancang untuk proses gambar
massal. Sangat berguna bagi pengguna yang perlu mengubah ukuran, mengonversi, dan
mengelola file gambar dalam jumlah besar sekaligus.
