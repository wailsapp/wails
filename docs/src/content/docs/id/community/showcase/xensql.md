**[XenSQL](https://github.com/Bare7a/XenSQL)** adalah **workbench SQL desktop local-first yang cepat** dibangun dengan **Go, Wails, dan React**. Aplikasi ini menggabungkan PostgreSQL, MySQL/MariaDB, dan SQLite dalam satu antarmuka yang bersih dan terasa native — tanpa cloud, tanpa telemetri, dan tanpa akun.

## Sorotan Utama

- **Editor SQL yang Powerful** — Berbasis Monaco dengan autocomplete cerdas yang aware schema, eksekusi multi-statement, hasil streaming, dan tab hasil per-statement
- **Penampil Data Lanjutan** — Inspektor JSON interaktif, editor sel aware syntax (JSON, XML, HTML, teks), pengeditan inline, dan inspeksi record penuh
- **Pengeditan Data Seamless** — Jelajahi tabel, stage perubahan inline, operasi bulk, dan `INSERT`/`UPDATE`/`DELETE` aman dengan dukungan `RETURNING`
- **Fitur Produktivitas** — Penjelajah schema, query tersimpan, riwayat query, pencarian cepat (`Ctrl+P`), dan alur kerja keyboard-first
- **Opsi Ekspor** — CSV, JSON, Markdown, SQL INSERTs

Sepenuhnya offline dan portabel. Semuanya disimpan secara lokal dalam satu folder `XenSQL-data/` yang ikut dengan aplikasi.

**Database yang Didukung**: PostgreSQL, MySQL, MariaDB, dan SQLite (dengan mode read-only dan opsi transport aman).

Dirancang untuk developer yang menginginkan kecepatan, kejelasan, dan kontrol tanpa bloat alat SQL tradisional.
