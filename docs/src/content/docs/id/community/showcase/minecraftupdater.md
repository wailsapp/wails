[Minecraft Updater](https://github.com/Gurkengewuerz/MinecraftModUpdater) adalah
tool utilitas untuk memperbarui dan mensinkronkan mod Minecraft untuk userbase Anda. Dibangun
menggunakan Wails2 dan React dengan [antd](https://ant.design/) sebagai frontend
framework.
