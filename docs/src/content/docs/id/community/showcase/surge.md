[Surge](https://getsurge.io/) adalah aplikasi berbagi file p2p yang dirancang untuk memanfaatkan
teknologi blockchain guna memungkinkan transfer file 100% anonim. Surge
dienkripsi end-to-end, terdesentralisasi, dan open source.
