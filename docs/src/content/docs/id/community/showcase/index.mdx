## Cara menambahkan proyek saya di showcase

<Steps>

1. Fork repositori.
2. Tambahkan gambar di folder `docs/src/assets/showcase-images`.
3. Salin file `_template.md` di folder
   `docs/src/content/docs/community/showcase`.
4. Rename file yang disalin ke nama proyek Anda. (Nama tidak boleh diawali
   dengan `_`)
5. Perbarui title, image, link dan konten file.
6. Tambahkan ke daftar di atas di
   `docs/src/content/docs/community/showcase/index.mdx`.
7. Kirim PR.

</Steps>
