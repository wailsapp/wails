2. **建立 TODO 服務**

   TODO 服務將管理我們的應用程式狀態，並提供用於 CRUD 操作的方法。與每個請求都隔離的網頁伺服器不同，桌面應用程式可能同時進行多個操作，因此我們需要執行緒安全的狀態管理。

   刪除 `greetservice.go` 並建立新檔案 `todoservice.go`：

   ```go title="todoservice.go"
   package main

   import (
       "errors"
       "sync"
   )

   type Todo struct {
       ID        int    `json:"id"`
       Title     string `json:"title"`
       Completed bool   `json:"completed"`
   }

   type TodoService struct {
       todos  []Todo
       nextID int
       mu     sync.RWMutex
   }

   func NewTodoService() *TodoService {
       return &TodoService{
           todos:  []Todo{},
           nextID: 1,
       }
   }

   func (t *TodoService) GetAll() []Todo {
       t.mu.RLock()
       defer t.mu.RUnlock()
       return t.todos
   }

   func (t *TodoService) Add(title string) (*Todo, error) {
       if title == "" {
           return nil, errors.New("title cannot be empty")
       }

       t.mu.Lock()
       defer t.mu.Unlock()

       todo := Todo{
           ID:        t.nextID,
           Title:     title,
           Completed: false,
       }
       t.todos = append(t.todos, todo)
       t.nextID++

       return &todo, nil
   }

   func (t *TodoService) Toggle(id int) error {
       t.mu.Lock()
       defer t.mu.Unlock()

       for i := range t.todos {
           if t.todos[i].ID == id {
               t.todos[i].Completed = !t.todos[i].Completed
               return nil
           }
       }
       return errors.New("todo not found")
   }

   func (t *TodoService) Delete(id int) error {
       t.mu.Lock()
       defer t.mu.Unlock()

       for i, todo := range t.todos {
           if todo.ID == id {
               t.todos = append(t.todos[:i], t.todos[i+1:]...)
               return nil
           }
       }
       return errors.New("todo not found")
   }
   ```

   **這裡發生了什麼：**

   **`Todo` 結構體：**
   - 定義了我們資料的結構，包含 ID、Title 和 Completed 欄位
   - `json:` 標籤告訴 Go 如何將此結構體轉換為 JSON 以傳遞給前端
   - 每個欄位都已匯出（大寫開頭），以便繫結生成器能夠識別

   **`TodoService` 結構體：**
   - `todos []Todo` - 儲存所有 TODO 項目的切片
   - `nextID int` - 追蹤下一個要分配的 ID（模擬自動遞增）
   - `mu sync.RWMutex` - 用於執行緒安全存取的讀寫互斥鎖

   **使用 `sync.RWMutex` 實現執行緒安全：**
   - 桌面應用程式可能從 UI 發起多個並行操作
   - `RLock()` 允許同時進行多個讀取（例如，多次 `GetAll` 呼叫）
   - `Lock()` 提供寫入的排他性存取（例如，`Add`、`Toggle`、`Delete`）
   - `defer` 確保即使函數提前返回或發生恐慌，鎖也會被釋放

   **這些方法：**
   - `GetAll()` - 返回所有 TODO 項目（使用讀取鎖，因為我們不修改資料）
   - `Add(title)` - 建立新的 TODO 項目，驗證輸入，遞增 ID
   - `Toggle(id)` - 翻轉 TODO 項目的完成狀態
   - `Delete(id)` - 從切片中移除一個 TODO 項目

   **錯誤處理：**
   - 我們遵循 Go 慣例，將 `error` 作為最後一個值返回
   - 空的標題會被拒絕
   - 對不存在的 TODO 項目的操作會返回錯誤
   - 這些錯誤會在前端變成 JavaScript 異常

4. **建立前端 UI**

   現在讓我們構建前端。這裡我們將呼叫 Go 方法並顯示 UI。我們使用原生 JavaScript 以保持簡單，並向您展示繫結是如何直接工作的。

   替換 `frontend/src/main.js`：

   ```javascript title="frontend/src/main.js"
   import {TodoService} from "../bindings/changeme";

   async function loadTodos() {
       const todos = await TodoService.GetAll();
       const list = document.getElementById('todo-list');

       list.innerHTML = todos.map(todo => `
           <div class="todo ${todo.completed ? 'completed' : ''}">
               <input type="checkbox"
                      ${todo.completed ? 'checked' : ''}
                      onchange="toggleTodo(${todo.id})">
               <span>${todo.title}</span>
               <button onclick="deleteTodo(${todo.id})">Delete</button>
           </div>
       `).join('');
   }

   window.addTodo = async () => {
       const input = document.getElementById('todo-input');
       const title = input.value.trim();

       if (title) {
           await TodoService.Add(title);
           input.value = '';
           await loadTodos();
       }
   }

   window.toggleTodo = async (id) => {
       await TodoService.Toggle(id);
       await loadTodos();
   }

   window.deleteTodo = async (id) => {
       await TodoService.Delete(id);
       await loadTodos();
   }

   // Load todos on startup
   loadTodos();
   ```

   **這裡發生了什麼：**

   **匯入繫結：**
   - `import {TodoService} from "../bindings/changeme"` - 引入自動生成的 Go 繫結
   - 注意：`changeme` 將是您 `go.mod` 中的實際模組名稱

   **`loadTodos()` 函數：**
   - 呼叫 `TodoService.GetAll()` 以從 Go 獲取所有 TODO 項目
   - 使用模板字面量為每個 TODO 項目構建 HTML
   - 動態添加/移除 `completed` 類別以進行樣式設定
   - 使用 `onclick` 屬性將按鈕連接到我們的函數
   - 將所有 HTML 連接起來並注入 DOM

   **CRUD 函數：**
   - `addTodo()` - 驗證輸入，呼叫 Go 的 `Add` 方法，刷新列表
   - `toggleTodo(id)` - 呼叫 Go 的 `Toggle` 方法，刷新列表
   - `deleteTodo(id)` - 呼叫 Go 的 `Delete` 方法，刷新列表
   - 所有函數都是異步的，因為 Go 呼叫返回 Promise**為什麼要繫結到視窗：**
- `window.addTodo = ...` 讓函式可以從 HTML 的 `onclick` 屬性中存取
- 這是一個針對原生 JavaScript 的簡單模式（框架會以不同方式處理此問題）
- 在生產環境中，您可能應該使用正確的事件委派（event delegation）

**重新載入模式：**
- 在每次狀態變更（新增/切換/刪除）之後，我們會再次呼叫 `loadTodos()`
- 這確保了 UI 與 Go 狀態保持同步
- 替代方案：讓 Go 方法回傳新狀態，以避免第二次呼叫

## 運作原理

### 執行緒安全的狀態管理

`sync.RWMutex` 提供安全的並發存取：

```go
func (t *TodoService) GetAll() []Todo {
    t.mu.RLock()  // 讀取鎖 — 允許多個讀取器
    defer t.mu.RUnlock()
    return t.todos
}

func (t *TodoService) Add(title string) (*Todo, error) {
    t.mu.Lock()  // 寫入鎖 — 獨佔存取
    defer t.mu.Unlock()
    // ... 變更操作
}
```

**為什麼這很重要：**
- 多個前端呼叫可能同時發生
- 讀取操作不會互相阻塞
- 寫入操作獲得獨佔存取權
- `defer` 確保鎖始終被釋放

### 錯誤處理

服務會針對無效操作回傳錯誤：

```go
func (t *TodoService) Add(title string) (*Todo, error) {
    if title == "" {
        return nil, errors.New("title cannot be empty")
    }
    // ...
}
```

在前端，您可以捕獲這些錯誤：

```javascript
try {
    await TodoService.Add(title);
} catch (err) {
    alert('Error: ' + err);
}
```

### 狀態同步

每次變更後，我們會重新載入完整列表：

```javascript
window.addTodo = async () => {
    await TodoService.Add(title);  // 變更操作
    await loadTodos();             // 重新整理
}
```

**替代方法：** 從每個方法回傳更新後的列表，以避免第二次呼叫。

## 增強功能

### 新增統計資訊

將以下程式碼加入 `todoservice.go`：

```go
type TodoStats struct {
    Total     int `json:"total"`
    Completed int `json:"completed"`
    Active    int `json:"active"`
}

func (t *TodoService) GetStats() TodoStats {
    t.mu.RLock()
    defer t.mu.RUnlock()

    stats := TodoStats{
        Total: len(t.todos),
    }

    for _, todo := range t.todos {
        if todo.Completed {
            stats.Completed++
        } else {
            stats.Active++
        }
    }

    return stats
}
```

在前端顯示：

```javascript
async function loadTodos() {
    const [todos, stats] = await Promise.all([
        TodoService.GetAll(),
        TodoService.GetStats()
    ]);

    // 顯示統計資訊
    document.getElementById('stats').textContent =
        `${stats.active} active, ${stats.completed} completed`;

    // ... 渲染 todos
}
```

### 新增「清除已完成」

```go
func (t *TodoService) ClearCompleted() int {
    t.mu.Lock()
    defer t.mu.Unlock()

    removed := 0
    newTodos := []Todo{}

    for _, todo := range t.todos {
        if !todo.Completed {
            newTodos = append(newTodos, todo)
        } else {
            removed++
        }
    }

    t.todos = newTodos
    return removed
}
```
