## 簡介

Wails 為桌面應用程式提供跨平台的 Dock 服務。此服務允許您：

- 隱藏和顯示 macOS Dock 中的應用程式圖示
- 在應用程式圖示或 Dock/工作列圖示上顯示徽章（macOS 和 Windows）

## 基本用法

### 建立服務

首先，初始化 Dock 服務：

```go
import "github.com/wailsapp/wails/v3/pkg/application"
import "github.com/wailsapp/wails/v3/pkg/services/dock"

// 建立新的 Dock 服務
dockService := dock.New()

// 將服務註冊到應用程式
app := application.New(application.Options{
    Services: []application.Service{
        application.NewService(dockService),
    },
})
```

### 使用自訂徽章選項建立服務（僅限 Windows）

在 Windows 上，您可以透過各種選項自訂徽章的外觀：

```go
import "github.com/wailsapp/wails/v3/pkg/application"
import "github.com/wailsapp/wails/v3/pkg/services/dock"
import "image/color"

// 使用自訂徽章選項建立 Dock 服務
options := dock.BadgeOptions{
    TextColour:       color.RGBA{255, 255, 255, 255}, // 白色文字
    BackgroundColour: color.RGBA{0, 0, 255, 255},     // 藍色背景
    FontName:         "consolab.ttf",                 // 粗體 Consolas 字型
    FontSize:         20,                             // 單一字元的字型大小
    SmallFontSize:    14,                             // 多個字元的字型大小
}

dockService := dock.NewWithOptions(options)

// 將服務註冊到應用程式
app := application.New(application.Options{
    Services: []application.Service{
        application.NewService(dockService),
    },
})
```

## Dock 操作

### 隱藏 Dock 應用程式圖示

從 macOS Dock 中隱藏應用程式圖示：

```go
// 隱藏應用程式圖示
dockService.HideAppIcon()
```

### 顯示 Dock 應用程式圖示

在 macOS Dock 中顯示應用程式圖示：

```go
// 顯示應用程式圖示
dockService.ShowAppIcon()
```

## 徽章操作

### 設定徽章

在應用程式圖示/Dock 圖示上設定徽章：

```go
// 設定預設徽章
dockService.SetBadge("")

// 設定數字徽章
dockService.SetBadge("3")

// 設定文字徽章
dockService.SetBadge("New")
```

### 設定自訂徽章（僅限 Windows）

套用一次性選項來設定徽章：

```go
options := dock.BadgeOptions{
    BackgroundColour: color.RGBA{0, 255, 255, 255},
    FontName:         "arialb.ttf", // 系統字型
    FontSize:         16,
    SmallFontSize:    10,
    TextColour:       color.RGBA{0, 0, 0, 255},
}

// 設定預設徽章
dockService.SetCustomBadge("", options)

// 設定數字徽章
dockService.SetCustomBadge("3", options)

// 設定文字徽章
dockService.SetCustomBadge("New", options)
```

### 移除徽章

從應用程式圖示中移除徽章：

```go
dockService.RemoveBadge()
```

### 取得設定的徽章
```go
dockService.GetBadge()
``` 

## 平台注意事項

<Tabs>
  <TabItem label="macOS" icon="fa-brands:apple">
    
    在 macOS 上：
    
    - Dock 圖示可以**隱藏**和**顯示**
    - 徽章直接顯示在 Dock 圖示上
    - 徽章選項**不可自訂**（傳遞給 `NewWithOptions`/`SetCustomBadge` 的選項會被忽略）
    - 使用標準 macOS Dock 徽章樣式，並自動適應外觀
    - 標籤溢位由系統處理
    - 提供空標籤會顯示預設徽章 "●"
    
  </TabItem>
  
  <TabItem label="Windows" icon="fa-brands:windows">
    
    在 Windows 上：
    
    - 此服務目前不支援隱藏/顯示工作列圖示
    - 徽章作為工作列上的疊加圖示顯示
    - 徽章支援文字值
    - 徽章外觀可透過 `BadgeOptions` 自訂
    - 應用程式必須有視窗才能顯示徽章
    - 多字元標籤會自動使用較小的字型大小
    - 不處理標籤溢位
    - 自訂選項：
      - **TextColour**: 文字顏色（預設：白色）
      - **BackgroundColour**: 徽章背景顏色（預設：紅色）
      - **FontName**: 字型檔案名稱（預設： "segoeuib.ttf"）
      - **FontSize**: 單一字元的字型大小（預設：18）
      - **SmallFontSize**: 多個字元的字型大小（預設：14）
    
  </TabItem>
  
  <TabItem label="Linux" icon="fa-brands:linux">
    
    在 Linux 上：
    
    - 不支援 Dock 圖示可見性和徽章功能
    
  </TabItem>
</Tabs>

## 最佳實踐

1. **隱藏 Dock 圖示時（macOS）：**
   - 確保使用者仍能存取您的應用程式（例如，透過 [系統匣](https://v3.wails.io/learn/systray/)）
   - 在替代 UI 中包含「退出」選項
   - 應用程式不會出現在 Command+Tab 切換器中
   - 開啟的視窗仍保持可見且可操作
   - 關閉所有視窗可能不會結束應用程式（macOS 行為因版本而異）
   - 使用者會失去透過 Dock 右鍵選單退出的標準方式

2. **謹慎使用徽章：**
   - 過多的徽章更新會分散使用者注意力
   - 僅將徽章保留給重要的通知

3. **保持徽章文字簡短：**
   - 數字徽章最有效
   - 在 macOS 上，文字徽章應簡短

4. **對於 Windows 徽章自訂：**
   - 確保文字和背景顏色之間有高對比度
   - 測試不同長度的文字，因為字型大小會隨長度縮小
   - 使用常見系統字型以確保可用性


## API 參考

### 服務管理
| 方法                                     | 描述                                           |
|------|--------------------------|----|------------------|-------------|------|
| `New()`                                    | 建立新的 Dock 服務                            |
| `NewWithOptions(options BadgeOptions)`     | 使用自訂徽章選項建立新的 Dock 服務（僅限 Windows；在 macOS 和 Linux 上選項會被忽略） |

### Dock 操作
| 方法                                       | 描述                                                |
|------|--------------------------|------------------------------------|----------|------|
| `HideAppIcon()`                | 從 macOS Dock 中隱藏應用程式圖示（僅限 macOS）         |
| `ShowAppIcon()`                | 在 macOS Dock 中顯示應用程式圖示（僅限 macOS）           |

### 徽章操作
| 方法                                            | 描述                                                |
|------|------------|------------|---------|------|-----|----------|-----------------------------|----------|
| `SetBadge(label string) error`                    | 設定具有指定標籤的徽章                      |
| `SetCustomBadge(label string, options BadgeOptions) error` | 設定具有指定標籤和自訂樣式選項的徽章（僅限 Windows） |
| `RemoveBadge() error`                             | 從應用程式圖示中移除徽章                |
| `GetBadge() *string`                             | 取得目前的徽章                |

### 結構與型別

```go
// 用於自訂徽章外觀的選項（僅限 Windows）
type BadgeOptions struct {
    TextColour       color.RGBA  // 徽章文字的顏色
    BackgroundColour color.RGBA  // 徽章背景的顏色
    FontName         string      // 字型檔案名稱（例如，"segoeuib.ttf"）
    FontSize         int         // 單一字元的字型大小
    SmallFontSize    int         // 多個字元的字型大小
}
```
