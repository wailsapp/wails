## 功能

Wails 提供原生桌面功能：

- [視窗](/features/windows/basics) - 多視窗、無邊框視窗、位置設定
- [選單](/features/menus/application) - 應用程式選單、右鍵選單、系統匣
- [對話框](/features/dialogs/file) - 檔案開啟/儲存、訊息對話框
- [事件](/features/events/system) - 元件間的通訊
- [繫結](/features/bindings/methods) - 類型安全的 Go ↔ JavaScript 呼叫
- [剪貼簿](/features/clipboard) - 複製/貼上操作
- [拖放](/features/drag-and-drop/files) - 檔案拖放
- [鍵盤](/features/keyboard) - 全域快捷鍵
