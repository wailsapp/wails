## 目前狀態：Alpha

請查看 [更新日誌](./changelog) 以獲取最新狀態。

我們的目標是達到 Beta 狀態。本路線圖概述了我們在過渡到 Beta 之前需要實作的主要功能與改進。請注意，這是一份持續更新的文件，可能會隨著優先級調整或新見解的出現而進行更新。

## 如何貢獻

- 測試最新的 Alpha 版本並回報任何問題
- 貢獻文件與範例
- 參與討論並對擬議的功能提供反饋
- 提交程式碼以修復錯誤或新增功能

我們歡迎社群的貢獻。如果您願意協助達成上述目標，或對路線圖有任何建議，請開啟一個 Issue 或加入我們的社群討論。

## 回饋與更新

本路線圖可能會根據社群回饋與專案優先級進行變更。我們將定期更新此文件，以反映我們的進度與方向的任何調整。您的意見對於塑造 Wails 的未來至關重要！本路線圖是一份持續更新的文件，可能會隨時變更。如果您有任何建議，請開啟一個 Issue。每個里程碑都將有一組我們旨在達成的目標，這些目標可能會有所調整。
