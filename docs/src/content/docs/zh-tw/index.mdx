## 快速開始

<Tabs syncKey="os">
  <TabItem label="macOS" icon="apple">
```bash
# 安裝 Wails
go install github.com/wailsapp/wails/v3/cmd/wails3@latest

# 建立您的應用程式
wails3 init -n myapp -t vanilla

# 使用熱重載執行
cd myapp && wails3 dev
```
  </TabItem>
  
  <TabItem label="Windows" icon="seti:windows">
```powershell
# 安裝 Wails
go install github.com/wailsapp/wails/v3/cmd/wails3@latest

# 建立您的應用程式
wails3 init -n myapp -t vanilla

# 使用熱重載執行
cd myapp; wails3 dev
```
  </TabItem>
  
  <TabItem label="Linux" icon="linux">
```bash
# 安裝 Wails
go install github.com/wailsapp/wails/v3/cmd/wails3@latest

# 建立您的應用程式
wails3 init -n myapp -t vanilla

# 使用熱重載執行
cd myapp && wails3 dev
```
  </TabItem>
</Tabs>

**您的應用程式現在正在執行**，具備熱重載功能以及類型安全的 Go 至 JS 綁定。


## 為什麼選擇 Wails？

<CardGrid>
  <Card title="用戶可感知的效能" icon="rocket">
    - 二進制檔案約 15MB，對比 Electron 的 150MB
    - 基礎記憶體約 10MB，對比 100MB+
    - 啟動時間 <0.5 秒，對比 2-3 秒
    - 使用作業系統原生的 WebView 進行渲染
    - 無需捆綁瀏覽器帶來的額外負荷
  </Card>
  
  <Card title="開發者體驗" icon="setting">
    - 單一 Go 程式碼庫適用於所有平台
    - 支援任何網頁框架 - React, Vue, Svelte
    - 開發期間支援熱重載
    - 自動生成綁定，輕鬆從 JavaScript 呼叫 Go
    - 記憶體內 IPC，無需網路連接埠
  </Card>
  
  <Card title="生產環境就緒" icon="approve-check">
    - 支援多個視窗及其生命週期
    - 原生選單和系統匣
    - 平台原生的檔案對話框
    - 系統整合與快捷鍵
    - 程式碼簽署與封裝工具
  </Card>
  
  <Card title="跨平台原生體驗" icon="laptop">
    - 單一程式碼庫適用於 Windows, macOS, Linux
    - 在需要時使用平台特定功能
    - 不犧牲使用者體驗
    - 從單一建置部署至所有平台
    - 行動裝置支援即將推出...
  </Card>
</CardGrid>
