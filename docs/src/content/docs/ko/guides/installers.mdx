## 개요

모든 플랫폼에서 Wails 애플리케이션을 위한 전문적인 인스톨러를 생성하세요.

## 플랫폼별 인스톨러

<Tabs syncKey="platform">
  <TabItem label="Windows" icon="seti:windows">
    ### NSIS 인스톨러
    
    ```bash
    # Install NSIS
    # Download from: https://nsis.sourceforge.io/
    
    # Create installer script (installer.nsi)
    makensis installer.nsi
    ```
    
    **installer.nsi:**
    ```nsis
    !define APPNAME "MyApp"
    !define VERSION "1.0.0"
    
    Name "${APPNAME}"
    OutFile "MyApp-Setup.exe"
    InstallDir "$PROGRAMFILES\${APPNAME}"
    
    Section "Install"
        SetOutPath "$INSTDIR"
        File "build\bin\myapp.exe"
        CreateShortcut "$DESKTOP\${APPNAME}.lnk" "$INSTDIR\myapp.exe"
    SectionEnd
    ```
    
    ### WiX Toolset
    
    MSI 인스톨러를 위한 대안입니다.
  </TabItem>

  <TabItem label="macOS" icon="apple">
    ### DMG 생성
    
    ```bash
    # Create DMG
    hdiutil create -volname "MyApp" -srcfolder build/bin/MyApp.app -ov -format UDZO MyApp.dmg
    ```
    
    ### 코드 서명
    
    ```bash
    # Sign application
    codesign --deep --force --verify --verbose --sign "Developer ID" MyApp.app
    
    # Notarize
    xcrun notarytool submit MyApp.dmg --apple-id "email" --password "app-password"
    ```
    
    ### 앱 스토어
    
    앱 스토어 배포를 위해 Xcode를 사용하세요.
  </TabItem>

  <TabItem label="Linux" icon="linux">
    ### DEB 패키지
    
    ```bash
    # Create package structure
    mkdir -p myapp_1.0.0/DEBIAN
    mkdir -p myapp_1.0.0/usr/bin
    
    # Copy binary
    cp build/bin/myapp myapp_1.0.0/usr/bin/
    
    # Create control file
    cat > myapp_1.0.0/DEBIAN/control << EOF
    Package: myapp
    Version: 1.0.0
    Architecture: amd64
    Maintainer: Your Name
    Description: My Application
    EOF
    
    # Build package
    dpkg-deb --build myapp_1.0.0
    ```
    
    ### RPM 패키지
    
    RPM 기반 배포판에는 `rpmbuild`를 사용하세요.
    
    ### AppImage
    
    ```bash
    # Use appimagetool
    appimagetool myapp.AppDir
    ```
  </TabItem>
</Tabs>

## 자동 패키징

### GoReleaser 사용

```yaml
# .goreleaser.yml
project_name: myapp

builds:
  - binary: myapp
    goos:
      - windows
      - darwin
      - linux
    goarch:
      - amd64
      - arm64

archives:
  - format: zip
    name_template: "{{ .ProjectName }}_{{ .Version }}_{{ .Os }}_{{ .Arch }}"

nfpms:
  - formats:
      - deb
      - rpm
    vendor: Your Company
    homepage: https://example.com
    description: My Application
```

## 모범 사례

### ✅ 수행할 사항

- 모든 플랫폼에서 코드 서명하기
- 버전 정보 포함하기
- 제거 프로그램(uninstaller) 생성하기
- 설치 과정 테스트하기
- 명확한 문서 제공하기

### ❌ 수행하지 말아야 할 사항

- 코드 서명을 건너뛰지 않기
- 파일 연결을 잊지 않기
- 경로를 하드코딩하지 않기
- 테스트를 건너뛰지 않기
