## 개요

Wails CLI(`wails3`)는 Wails 애플리케이션을 생성, 개발, 빌드 및 관리하기 위한 명령어를 제공합니다.

## 곧 출시 예정

이 섹션은 현재 준비 중입니다. 현재는 명령어 문서를 위해 `wails3 --help` 또는 `wails3 [command] --help`를 사용하세요.

**일반 명령어:**

```bash
wails3 init          # 새 프로젝트 생성
wails3 dev           # 개발 모드에서 실행
wails3 build         # 프로덕션을 위한 빌드
wails3 generate      # 바인딩 또는 런타임 생성
wails3 doctor        # 개발 환경 확인
```
