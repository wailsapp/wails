## 기능

Wails는 네이티브 데스크톱 기능을 제공합니다:

- [Windows](/features/windows/basics) - 여러 창, 프레임리스 창, 위치 지정
- [메뉴](/features/menus/application) - 애플리케이션 메뉴, 컨텍스트 메뉴, 시스템 트레이
- [다이얼로그](/features/dialogs/file) - 파일 열기/저장, 메시지 다이얼로그
- [이벤트](/features/events/system) - 컴포넌트 간 통신
- [바인딩](/features/bindings/methods) - 타입 안전한 Go ↔ JavaScript 호출
- [클립보드](/features/clipboard) - 복사/붙여넣기 작업
- [드래그 앤 드롭](/features/drag-and-drop/files) - 파일 드래그 앤 드롭
- [키보드](/features/keyboard) - 글로벌 단축키
