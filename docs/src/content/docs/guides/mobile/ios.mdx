## Prerequisites

- **macOS** 12.0+
- **Xcode** 14.0+
- **Go** 1.21+ with CGO

```bash
xcode-select --install
```

## Building

```bash
# Build for iOS
task ios:build

# Run on simulator
task ios:run

# Build for device (requires code signing)
task ios:build:device
```

## Configuration

```go
app := application.New(application.Options{
    Name: "My App",
    iOS: application.IOSOptions{
        DisableBounce: true,
        EnableZoom:    false,
        UserAgent:     "MyApp/1.0",
        BackgroundColour: application.NewRGB(255, 255, 255),
    },
})
```

## iOS-Specific Features

<Aside type="tip" title="Multiple API Options">
iOS features are available through two APIs:
- **Unified API** (recommended): `app.Mobile.HapticFeedback()` - works cross-platform
- **Platform Manager**: `application.IOS.HapticsImpact()` - iOS-specific manager

See the [MobileManager API](/guides/mobile#mobilemanager-api) section for details on the unified API.
</Aside>

These features are available on iOS via the `application.IOS` manager or the unified `app.Mobile` API.

### Haptic Feedback

Trigger native iOS haptic feedback using the Taptic Engine:

<Tabs>
<TabItem label="Unified API (Recommended)">
```go
// Available haptic styles (cross-platform)
app.Mobile.HapticFeedback(application.HapticLight)   // Button taps
app.Mobile.HapticFeedback(application.HapticMedium)  // Default
app.Mobile.HapticFeedback(application.HapticHeavy)   // Important actions
app.Mobile.HapticFeedback(application.HapticSoft)    // Gentle feedback
app.Mobile.HapticFeedback(application.HapticRigid)   // Sharp feedback

// Or use Vibrate (uses medium haptic on iOS)
app.Mobile.Vibrate(50) // Duration ignored on iOS
```
</TabItem>
<TabItem label="Platform Manager">
```go
// Available impact styles:
// - "light"  - Subtle feedback
// - "medium" - Moderate feedback
// - "heavy"  - Strong feedback
// - "soft"   - Soft, gentle feedback
// - "rigid"  - Sharp, precise feedback

application.IOS.HapticsImpact("medium")

// Use for UI interactions
application.IOS.HapticsImpact("light")  // Button taps
application.IOS.HapticsImpact("heavy")  // Important actions
```
</TabItem>
</Tabs>

### Scroll Control

Control WKWebView scrolling behavior at runtime:

<Tabs>
<TabItem label="Unified API (Recommended)">
```go
// Disable scrolling entirely (for fixed-layout UIs)
app.Mobile.SetScrollEnabled(false)

// Re-enable scrolling
app.Mobile.SetScrollEnabled(true)

// Hide scroll indicators
app.Mobile.SetScrollIndicatorsEnabled(false)
```
</TabItem>
<TabItem label="Platform Manager">
```go
// Disable scrolling entirely (for fixed-layout UIs)
application.IOS.SetScrollEnabled(false)

// Re-enable scrolling
application.IOS.SetScrollEnabled(true)

// Hide scroll indicators
application.IOS.SetScrollIndicatorsEnabled(false)
```
</TabItem>
</Tabs>

### Bounce Effect

Control the iOS rubber-band bounce effect when scrolling past content bounds:

<Tabs>
<TabItem label="Unified API (Recommended)">
```go
// Disable bounce effect for more native-app feel
app.Mobile.SetBounceEnabled(false)

// Re-enable bounce
app.Mobile.SetBounceEnabled(true)
```
</TabItem>
<TabItem label="Platform Manager">
```go
// Disable bounce effect for more native-app feel
application.IOS.SetBounceEnabled(false)

// Re-enable bounce
application.IOS.SetBounceEnabled(true)
```
</TabItem>
</Tabs>

<Aside type="tip">
Disabling bounce is useful for apps with fixed layouts or when you want to prevent overscroll interference with custom gestures.
</Aside>

### Custom User Agent

Set a custom User-Agent string for the WebView:

<Tabs>
<TabItem label="Unified API (Recommended)">
```go
// Set custom user agent
app.Mobile.SetCustomUserAgent("MyApp/1.0 (iOS)")

// Include version info
version := "2.1.0"
app.Mobile.SetCustomUserAgent(fmt.Sprintf("MyApp/%s WKWebView", version))
```
</TabItem>
<TabItem label="Platform Manager">
```go
// Set custom user agent
application.IOS.SetCustomUserAgent("MyApp/1.0 (iOS)")

// Include version info
version := "2.1.0"
application.IOS.SetCustomUserAgent(fmt.Sprintf("MyApp/%s WKWebView", version))
```
</TabItem>
</Tabs>

### Device Information

Get iOS device information:

<Tabs>
<TabItem label="Unified API (Recommended)">
```go
// Returns typed DeviceInfo struct
info := app.Mobile.DeviceInfo()
fmt.Println(info.Platform)     // "ios"
fmt.Println(info.Manufacturer) // "Apple"
fmt.Println(info.Model)        // e.g., "iPhone 15"
fmt.Println(info.OSVersion)    // e.g., "17.0"

// Check if running on mobile
if app.Mobile.IsMobile() {
    // iOS or Android specific logic
}
```
</TabItem>
<TabItem label="Platform Manager">
```go
// Get device info struct
info := application.IOS.DeviceInfo()
fmt.Println(info.Model)         // e.g., "iPhone 15"
fmt.Println(info.SystemName)    // "iOS"
fmt.Println(info.SystemVersion) // e.g., "17.0"
fmt.Println(info.IsSimulator)   // true/false
```
</TabItem>
</Tabs>

### Native Tabs

Control native iOS tab bar:

```go
// Enable native tab bar
application.IOS.SetNativeTabsEnabled(true)

// Check if native tabs are enabled
if application.IOS.NativeTabsIsEnabled() {
    // Select a specific tab
    application.IOS.SelectNativeTab(2)
}
```

## Troubleshooting

**Code signing required**
- Physical devices require an Apple Developer account
- Simulators don't need signing

**Command Line Tools not found**
```bash
xcode-select --install
```
