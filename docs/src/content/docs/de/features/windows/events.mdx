## Fensterereignisse

Wails bietet **umfassende Ereignis-Hooks** für den Fensterlebenszyklus und Zustandsänderungen: Erstellung, Fokus/Verlust des Fokus, Größe verschieben/verschieben, minimieren/maximieren und Schließen. Registrieren Sie Callbacks, behandeln Sie Ereignisse und koordinieren Sie Fenster mit einfachen, konsistenten APIs.

## Lebenszyklus-Ereignisse

### OnCreate

Wird aufgerufen, wenn ein Fenster erstellt wird:

```go
app.OnWindowCreation(func(window *application.WebviewWindow) {
    fmt.Printf("Fenster erstellt: %s (ID: %d)\n", window.Name(), window.ID())
    
    // Konfigurieren Sie alle neuen Fenster
    window.SetMinSize(400, 300)
    
    // Registrieren Sie fensterspezifische Handler
    window.OnClose(func() bool {
        return confirmClose()
    })
})
```

**Anwendungsfälle:**
- Konfigurieren Sie alle Fenster konsistent
- Registrieren Sie Ereignis-Handler
- Verfolgen Sie die Fenstererstellung
- Initialisieren Sie fensterspezifische Ressourcen

### OnClose

Wird aufgerufen, wenn der Benutzer versucht, das Fenster zu schließen:

```go
window.OnClose(func() bool {
    // Geben Sie false zurück, um das Schließen zu verhindern
    // Geben Sie true zurück, um das Schließen zu erlauben
    
    if hasUnsavedChanges() {
        result := showConfirmdialog("Nicht gespeicherte Änderungen. Trotzdem schließen?")
        return result == "yes"
    }
    return true
})
```

**Wichtig:**
- Wird nur durch Benutzeraktionen ausgelöst (Klicken auf die X-Schaltfläche)
- Wird NICHT durch `window.Destroy()` ausgelöst
- Kann das Schließen verhindern, indem `false` zurückgegeben wird

**Anwendungsfälle:**
- Bestätigung vor dem Schließen
- Zustand speichern
- Unbeabsichtigtes Schließen verhindern
- Aufräumen vor dem Schließen

**Beispiel mit Dialog:**

```go
window.OnClose(func() bool {
    if !hasUnsavedChanges() {
        return true
    }
    
    // Bestätigungsdialog anzeigen
    dialog := app.Window.NewWithOptions(application.WebviewWindowOptions{
        Title:       "Schließen bestätigen",
        Width:       400,
        Height:      150,
        Parent:      window,
        AlwaysOnTop: true,
    })
    
    // Auf Benutzerantwort warten
    result := waitFordialogResult(dialog)
    
    return result == "yes"
})
```

### OnDestroy

Wird aufgerufen, wenn das Fenster zerstört wird:

```go
window.OnDestroy(func() {
    fmt.Printf("Fenster zerstört: %s\n", window.Name())
    
    // Ressourcen aufräumen
    closeDatabase()
    
 // Aus der Verfolgung entfernen
    removeWindowFromRegistry(window.ID())
    
    // Anwendungsstatus aktualisieren
    updateWindowCount()
})
```

**Wichtig:**
- Wird immer aufgerufen, wenn das Fenster zerstört wird
- Kann nicht verhindert werden
- Letzte Chance zum Aufräumen

**Anwendungsfälle:**
- Ressourcen freigeben
- Verbindungen schließen
- Anwendungsstatus aktualisieren
- Aus der Verfolgung entfernen

**Beispiel mit Ressourcen-Aufräumen:**

```go
type ManagedWindow struct {
    window    *application.WebviewWindow
    db        *sql.DB
    listeners []func()
}

func (mw *ManagedWindow) Setup() {
    mw.window.OnDestroy(func() {
        // Datenbank schließen
        if mw.db != nil {
            mw.db.Close()
        }
        
        // Ereignis-Listener entfernen
        for _, listener := range mw.listeners {
            listener()
        }
        
        // Referenzen löschen
        mw.db = nil
        mw.listeners = nil
    })
}
```

## Fokus-Ereignisse

### OnFocus

Wird aufgerufen, wenn das Fenster den Fokus erhält:

```go
window.OnFocus(func() {
    fmt.Println("Fenster hat Fokus erhalten")
    
    // UI aktualisieren
    updateTitleBar(true)
    
    // Daten aktualisieren
    refreshContent()
    
    // Andere Fenster benachrichtigen
    app.Event.Emit("window-focused", window.ID())
})
```

**Anwendungsfälle:**
- UI-Darstellung aktualisieren
- Daten aktualisieren
- Operationen fortsetzen
- Mit anderen Fenstern koordinieren

### OnBlur

Wird aufgerufen, wenn das Fenster den Fokus verliert:

```go
window.OnBlur(func() {
    fmt.Println("Fenster hat Fokus verloren")
    
    // UI aktualisieren
    updateTitleBar(false)
    
    // Operationen pausieren
    pauseAnimations()
    
    // Zustand speichern
    saveCurrentState()
})
```

**Anwendungsfälle:**
- UI-Darstellung aktualisieren
- Operationen pausieren
- Zustand speichern
- Ressourcennutzung reduzieren

**Beispiel: Fokus-bewusste UI:**

```go
type FocusAwareWindow struct {
    window  *application.WebviewWindow
    focused bool
}

func (fw *FocusAwareWindow) Setup() {
    fw.window.OnFocus(func() {
        fw.focused = true
        fw.updateAppearance()
    })
    
    fw.window.OnBlur(func() {
        fw.focused = false
        fw.updateAppearance()
    })
}

func (fw *FocusAwareWindow) updateAppearance() {
    if fw.focused {
        fw.window.EmitEvent("update-theme", "active")
    } else {
        fw.window.EmitEvent("update-theme", "inactive")
    }
}
```

## Zustandsänderungs-Ereignisse

### OnMinimise / OnUnMinimise

Wird aufgerufen, wenn das Fenster minimiert oder wiederhergestellt wird:

```go
window.OnMinimise(func() {
    fmt.Println("Fenster minimiert")
    
    // Teure Operationen pausieren
    pauseRendering()
    
    // Zustand speichern
    saveWindowState()
})

window.OnUnMinimise(func() {
    fmt.Println("Fenster aus minimiertem Zustand wiederhergestellt")
    
    // Operationen fortsetzen
    resumeRendering()
    
    // Daten aktualisieren
    refreshContent()
})
```

**Anwendungsfälle:**
- Operationen pausieren/fortsetzen
- Zustand speichern/wiederherstellen
- Ressourcennutzung reduzieren
- UI aktualisieren

### OnMaximise / OnUnMaximise

Wird aufgerufen, wenn das Fenster maximiert oder wiederhergestellt wird:

```go
window.OnMaximise(func() {
    fmt.Println("Fenster maximiert")
    
    // Layout anpassen
    window.EmitEvent("layout-mode", "maximised")
    
    // Schaltflächensymbol aktualisieren
    updateMaximiseButton("restore")
})

window.OnUnMaximise(func() {
    fmt.Println("Fenster aus maximiertem Zustand wiederhergestellt")
    
    // Layout anpassen
    window.EmitEvent("layout-mode", "normal")
    
    // Schaltflächensymbol aktualisieren
    updateMaximiseButton("maximise")
})
```

**Anwendungsfälle:**
- Layout anpassen
- UI aktualisieren
- Fensterzustand speichern
- Mit anderen Fenstern koordinieren

### OnFullscreen / OnUnFullscreen

Wird aufgerufen, wenn das Fenster den Vollbildmodus betritt oder verlässt:

```go
window.OnFullscreen(func() {
    fmt.Println("Fenster hat Vollbildmodus betreten")
    
    // UI-Chrome ausblenden
    window.EmitEvent("chrome-visibility", false)
    
    // Layout anpassen
    window.EmitEvent("layout-mode", "fullscreen")
})

window.OnUnFullscreen(func() {
    fmt.Println("Fenster hat Vollbildmodus verlassen")
    
    // UI-Chrome einblenden
    window.EmitEvent("chrome-visibility", true)
    
    // Layout wiederherstellen
    window.EmitEvent("layout-mode", "normal")
})
```

**Anwendungsfälle:**
- UI-Elemente ein-/ausblenden
- Layout anpassen
- Steuerelemente aktualisieren
- Einstellungen speichern

## Positions- und Größen-Ereignisse

### OnMove

Wird aufgerufen, wenn das Fenster verschoben wird:

```go
window.OnMove(func(x, y int) {
    fmt.Printf("Fenster verschoben nach: %d, %d\n", x, y)
    
    // Position speichern
    saveWindowPosition(x, y)
    
    // Verwandte Fenster aktualisieren
    updateRelatedWindowPositions(x, y)
})
```

**Anwendungsfälle:**
- Fensterposition speichern
- Verwandte Fenster aktualisieren
- An Kanten andocken
- Multi-Monitor-Handling

### OnResize

Wird aufgerufen, wenn das Fenster in der Größe geändert wird:

```go
window.OnResize(func(width, height int) {
    fmt.Printf("Fenstergröße geändert auf: %dx%d\n", width, height)
    
    // Größe speichern
    saveWindowSize(width, height)
    
    // Layout anpassen
    window.EmitEvent("window-size", map[string]int{
        "width":  width,
        "height": height,
    })
})
```

**Anwendungsfälle:**
- Fenstergröße speichern
- Layout anpassen
- UI aktualisieren
- Responsives Design

**Beispiel: Responsives Layout:**

```go
window.OnResize(func(width, height int) {
    var layout string
    
    if width < 600 {
        layout = "compact"
    } else if width < 1200 {
        layout = "normal"
    } else {
        layout = "wide"
    }
    
    window.EmitEvent("layout-changed", layout)
})
```

    "encoding/json"
    "fmt"
    "os"

)

type WindowState struct {
    X          int  `json:"x"`
    Y          int  `json:"y"`
    Width      int  `json:"width"`
    Height     int  `json:"height"`
    Maximised  bool `json:"maximised"`
    Fullscreen bool `json:"fullscreen"`
}

type ManagedWindow struct {
    app     *application.Application
}
