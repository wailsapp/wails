## Dateidialoge

Wails bietet **native Dateidialoge** mit plattformgerechtem Erscheinungsbild zum Öffnen von Dateien, Speichern von Dateien und Auswählen von Ordnern. Einfache API mit Dateityp-Filterung, Unterstützung für Mehrfachauswahl und Standardordnern.

## Erstellen von Dateidialogen

Dateidialoge werden über den `app.Dialog`-Manager angesprochen:

```go
app.Dialog.OpenFile()
app.Dialog.SaveFile()
```

## Dialog zum Öffnen von Dateien

Wählen Sie Dateien zum Öffnen aus:

```go
path, err := app.Dialog.OpenFile().
    SetTitle("Bild auswählen").
    AddFilter("Bilder", "*.png;*.jpg;*.gif").
    AddFilter("Alle Dateien", "*.*").
    PromptForSingleSelection()

if err != nil || path == "" {
    return
}

openFile(path)
```

**Verwendungszwecke:**
- Dokumente öffnen
- Dateien importieren
- Bilder laden
- Konfigurationsdateien auswählen

### Einzelauswahl einer Datei

```go
path, err := app.Dialog.OpenFile().
    SetTitle("Dokument öffnen").
    AddFilter("Textdateien", "*.txt").
    AddFilter("Alle Dateien", "*.*").
    PromptForSingleSelection()

if err != nil || path == "" {
    // Benutzer hat abgebrochen oder ein Fehler ist aufgetreten
    return
}

// Ausgewählte Datei verwenden
data, _ := os.ReadFile(path)
```

### Mehrfachauswahl von Dateien

```go
paths, err := app.Dialog.OpenFile().
    SetTitle("Bilder auswählen").
    AddFilter("Bilder", "*.png;*.jpg;*.jpeg;*.gif").
    PromptForMultipleSelection()

if err != nil {
    return
}

// Alle ausgewählten Dateien verarbeiten
for _, path := range paths {
    processFile(path)
}
```

### Mit Standardordner

```go
path, err := app.Dialog.OpenFile().
    SetTitle("Datei öffnen").
    SetDirectory("/Users/me/Documents").
    PromptForSingleSelection()
```

## Dialog zum Speichern von Dateien

Wählen Sie den Speicherort aus:

```go
path, err := app.Dialog.SaveFile().
    SetFilename("document.txt").
    AddFilter("Textdateien", "*.txt").
    AddFilter("Alle Dateien", "*.*").
    PromptForSingleSelection()

if err != nil || path == "" {
    return
}

saveFile(path, data)
```

**Verwendungszwecke:**
- Dokumente speichern
- Daten exportieren
- Neue Dateien erstellen
- Speichern unter...

### Mit Standarddateiname

```go
path, err := app.Dialog.SaveFile().
    SetFilename("export.csv").
    AddFilter("CSV-Dateien", "*.csv").
    PromptForSingleSelection()
```

### Mit Standardordner

```go
path, err := app.Dialog.SaveFile().
    SetDirectory("/Users/me/Documents").
    SetFilename("untitled.txt").
    PromptForSingleSelection()
```

### Überschreibungsbestätigung

```go
path, err := app.Dialog.SaveFile().
    SetFilename("document.txt").
    PromptForSingleSelection()

if err != nil || path == "" {
    return
}

// Prüfen, ob die Datei bereits existiert
if _, err := os.Stat(path); err == nil {
    dialog := app.Dialog.Question().
        SetTitle("Überschreiben bestätigen").
        SetMessage("Datei existiert bereits. Überschreiben?")

    overwriteBtn := dialog.AddButton("Überschreiben")
    overwriteBtn.OnClick(func() {
        saveFile(path, data)
    })

    cancelBtn := dialog.AddButton("Abbrechen")
    dialog.SetDefaultButton(cancelBtn)
    dialog.SetCancelButton(cancelBtn)
    dialog.Show()
    return
}

saveFile(path, data)
```

## Dialog zur Ordnerauswahl

Wählen Sie ein Verzeichnis mit dem Dialog zum Öffnen von Dateien unter Aktivierung der Ordnerauswahl aus:

```go
path, err := app.Dialog.OpenFile().
    SetTitle("Ausgabeordner auswählen").
    CanChooseDirectories(true).
    CanChooseFiles(false).
    PromptForSingleSelection()

if err != nil || path == "" {
    return
}

exportToFolder(path)
```

**Verwendungszwecke:**
- Ausgabeordner auswählen
- Arbeitsbereich auswählen
- Sicherungsort auswählen
- Installationsverzeichnis auswählen

### Mit Standardordner

```go
path, err := app.Dialog.OpenFile().
    SetTitle("Ordner auswählen").
    SetDirectory("/Users/me/Documents").
    CanChooseDirectories(true).
    CanChooseFiles(false).
    PromptForSingleSelection()
```

## Dateifilter

Verwenden Sie die Methode `AddFilter()`, um Dateityp-Filter zu Dialogen hinzuzufügen. Jeder Aufruf fügt eine neue Filteroption hinzu.

### Einfache Filter

```go
path, _ := app.Dialog.OpenFile().
    AddFilter("Textdateien", "*.txt").
    AddFilter("Alle Dateien", "*.*").
    PromptForSingleSelection()
```

### Mehrere Erweiterungen

Verwenden Sie Semikolons, um mehrere Erweiterungen in einem einzelnen Filter anzugeben:

```go
dialog := app.Dialog.OpenFile().
    AddFilter("Bilder", "*.png;*.jpg;*.jpeg;*.gif;*.bmp").
    AddFilter("Dokumente", "*.txt;*.doc;*.docx;*.pdf").
    AddFilter("Alle Dateien", "*.*")
```

### Musterformat

Verwenden Sie **Semikolons**, um mehrere Erweiterungen in einem einzelnen Filter zu trennen:

```go
// Mehrere Erweiterungen, durch Semikolons getrennt
AddFilter("Bilder", "*.png;*.jpg;*.gif")
```

## Vollständige Beispiele

### Bilddatei öffnen

```go
func openImage(app *application.App) (image.Image, error) {
    path, err := app.Dialog.OpenFile().
        SetTitle("Bild auswählen").
        AddFilter("Bilder", "*.png;*.jpg;*.jpeg;*.gif;*.bmp").
        PromptForSingleSelection()

    if err != nil {
        return nil, err
    }

    if path == "" {
        return nil, errors.New("keine Datei ausgewählt")
    }

    // Bild öffnen und decodieren
    file, err := os.Open(path)
    if err != nil {
        app.Dialog.Error().
            SetTitle("Öffnen fehlgeschlagen").
            SetMessage(err.Error()).
            Show()
        return nil, err
    }
    defer file.Close()

    img, _, err := image.Decode(file)
    if err != nil {
        app.Dialog.Error().
            SetTitle("Ungültiges Bild").
            SetMessage("Bilddatei konnte nicht decodiert werden.").
            Show()
        return nil, err
    }

    return img, nil
}
```

### Dokument mit Validierung speichern

```go
func saveDocument(app *application.App, content string) {
    path, err := app.Dialog.SaveFile().
        SetFilename("document.txt").
        AddFilter("Textdateien", "*.txt").
        AddFilter("Markdown-Dateien", "*.md").
        AddFilter("Alle Dateien", "*.*").
        PromptForSingleSelection()

    if err != nil || path == "" {
        return
    }

    // Erweiterung validieren
    ext := filepath.Ext(path)
    if ext != ".txt" && ext != ".md" {
        dialog := app.Dialog.Question().
            SetTitle("Erweiterung bestätigen").
            SetMessage(fmt.Sprintf("Als %s-Datei speichern?", ext))

        saveBtn := dialog.AddButton("Speichern")
        saveBtn.OnClick(func() {
            doSave(app, path, content)
        })

        cancelBtn := dialog.AddButton("Abbrechen")
        dialog.SetDefaultButton(cancelBtn)
        dialog.SetCancelButton(cancelBtn)
        dialog.Show()
        return
    }

    doSave(app, path, content)
}

func doSave(app *application.App, path, content string) {
    if err := os.WriteFile(path, []byte(content), 0644); err != nil {
        app.Dialog.Error().
            SetTitle("Speichern fehlgeschlagen").
            SetMessage(err.Error()).
            Show()
        return
    }

    app.Dialog.Info().
        SetTitle("Gespeichert").
        SetMessage("Dokument erfolgreich gespeichert!").
        Show()
}
```
