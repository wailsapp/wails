## Nachrichten-Dialoge

Wails bietet **native Nachrichten-Dialoge** mit plattformgerechtem Erscheinungsbild: Info-, Warn-, Fehler- und Frage-Dialoge mit anpassbaren Titeln, Nachrichten und Schaltflächen. Einfache API, natives Verhalten, standardmäßig barrierefrei.

## Erstellen von Dialogen

Nachrichten-Dialoge werden über den `app.Dialog`-Manager zugänglich gemacht:

```go
app.Dialog.Info()
app.Dialog.Question()
app.Dialog.Warning()
app.Dialog.Error()
```

Alle Methoden geben ein `*MessageDialog` zurück, das über Methodenkettung konfiguriert werden kann.

## Informationsdialog

Zeigt Informationsmeldungen an:

```go
app.Dialog.Info().
    SetTitle("Erfolg").
    SetMessage("Datei erfolgreich gespeichert!").
    Show()
```

**Verwendungszwecke:**
- Erfolgsbestätigungen
- Abschlussmitteilungen
- Informationsmeldungen
- Statusaktualisierungen

**Beispiel - Speicherbestätigung:**

```go
func saveFile(app *application.App, path string, data []byte) error {
    if err := os.WriteFile(path, data, 0644); err != nil {
        return err
    }

    app.Dialog.Info().
        SetTitle("Datei gespeichert").
        SetMessage(fmt.Sprintf("Gespeichert in %s", filepath.Base(path))).
        Show()

    return nil
}
```

## Warnungsdialog

Zeigt Warnungen an:

```go
app.Dialog.Warning().
    SetTitle("Warnung").
    SetMessage("Diese Aktion kann nicht rückgängig gemacht werden.").
    Show()
```

**Verwendungszwecke:**
- Nicht-kritische Warnungen
- Deprecation-Hinweise
- Vorsichtsmeldungen
- Potenzielle Probleme

**Beispiel - Festplattenspeicher-Warnung:**

```go
func checkDiskSpace(app *application.App) {
    available := getDiskSpace()

    if available < 100*1024*1024 { // Weniger als 100MB
        app.Dialog.Warning().
            SetTitle("Niedriger Festplattenspeicher").
            SetMessage(fmt.Sprintf("Nur noch %d MB verfügbar.", available/(1024*1024))).
            Show()
    }
}
```

## Fehlerdialog

Zeigt Fehler an:

```go
app.Dialog.Error().
    SetTitle("Fehler").
    SetMessage("Verbindung zum Server fehlgeschlagen.").
    Show()
```

**Verwendungszwecke:**
- Fehlermeldungen
- Fehlschlag-Benachrichtigungen
- Ausnahmebehandlung
- Kritische Probleme

**Beispiel - Netzwerkfehler:**

```go
func fetchData(app *application.App, url string) ([]byte, error) {
    resp, err := http.Get(url)
    if err != nil {
        app.Dialog.Error().
            SetTitle("Netzwerkfehler").
            SetMessage(fmt.Sprintf("Verbindung fehlgeschlagen: %v", err)).
            Show()
        return nil, err
    }
    defer resp.Body.Close()

    return io.ReadAll(resp.Body)
}
```

## Frage-Dialog

Stellt dem Benutzer Fragen und verarbeitet Antworten über Schaltflächen-Rückrufe:

```go
dialog := app.Dialog.Question().
    SetTitle("Bestätigen").
    SetMessage("Änderungen speichern vor dem Schließen?")

save := dialog.AddButton("Speichern")
save.OnClick(func() {
    saveChanges()
})

dontSave := dialog.AddButton("Nicht speichern")
dontSave.OnClick(func() {
    // Ohne Speichern fortfahren
})

cancel := dialog.AddButton("Abbrechen")
cancel.OnClick(func() {
    // Nicht schließen
})

dialog.SetDefaultButton(save)
dialog.SetCancelButton(cancel)
dialog.Show()
```

**Verwendungszwecke:**
- Aktionen bestätigen
- Ja/Nein-Fragen
- Multiple-Choice
- Benutzerentscheidungen

**Beispiel - Ungespeicherte Änderungen:**

```go
func closeDocument(app *application.App) {
    if !hasUnsavedChanges() {
        doClose()
        return
    }

    dialog := app.Dialog.Question().
        SetTitle("Ungespeicherte Änderungen").
        SetMessage("Möchten Sie Ihre Änderungen speichern?")

    save := dialog.AddButton("Speichern")
    save.OnClick(func() {
        if saveDocument() {
            doClose()
        }
    })

    dontSave := dialog.AddButton("Nicht speichern")
    dontSave.OnClick(func() {
        doClose()
    })

    cancel := dialog.AddButton("Abbrechen")
    // Abbrechen-Schaltfläche hat keinen Rückruf - schließt nur den Dialog

    dialog.SetDefaultButton(save)
    dialog.SetCancelButton(cancel)
    dialog.Show()
}
```

## Dialog-Optionen

### Titel und Nachricht

```go
dialog := app.Dialog.Info().
    SetTitle("Vorgang abgeschlossen").
    SetMessage("Alle Dateien wurden erfolgreich verarbeitet.")
```

**Best Practices:**
- **Titel:** Kurz, beschreibend (2-5 Wörter)
- **Nachricht:** Klar, spezifisch, handlungsorientiert
- **Jargon vermeiden:** Einfache Sprache verwenden

### Schaltflächen

**Einzelne Schaltfläche (Info/Warnung/Fehler):**

Info-, Warn- und Fehler-Dialoge zeigen standardmäßig eine "OK"-Schaltfläche an:

```go
app.Dialog.Info().
    SetMessage("Fertig!").
    Show()
```

Sie können auch benutzerdefinierte Schaltflächen hinzufügen:

```go
dialog := app.Dialog.Info().
    SetMessage("Fertig!")
ok := dialog.AddButton("Verstanden!")
dialog.SetDefaultButton(ok)
dialog.Show()
```

**Mehrere Schaltflächen (Frage):**

Verwenden Sie `AddButton()`, um Schaltflächen hinzuzufügen, die ein `*Button` zurückgeben, das Sie konfigurieren können:

```go
dialog := app.Dialog.Question().
    SetMessage("Wählen Sie eine Aktion")

option1 := dialog.AddButton("Option 1")
option1.OnClick(func() {
    handleOption1()
})

option2 := dialog.AddButton("Option 2")
option2.OnClick(func() {
    handleOption2()
})

option3 := dialog.AddButton("Option 3")
option3.OnClick(func() {
    handleOption3()
})

dialog.Show()
```

**Standard- und Abbrechen-Schaltflächen:**

Verwenden Sie `SetDefaultButton()`, um anzugeben, welche Schaltfläche hervorgehoben und durch Eingabe (Enter) ausgelöst wird.
Verwenden Sie `SetCancelButton()`, um anzugeben, welche Schaltfläche durch Escape ausgelöst wird.

```go
dialog := app.Dialog.Question().
    SetMessage("Datei löschen?")

deleteBtn := dialog.AddButton("Löschen")
deleteBtn.OnClick(func() {
    performDelete()
})

cancelBtn := dialog.AddButton("Abbrechen")
// Kein Rückruf erforderlich - verwirft nur den Dialog

dialog.SetDefaultButton(cancelBtn)  // Sichere Option als Standard
dialog.SetCancelButton(cancelBtn)   // Escape löst Abbrechen aus
dialog.Show()
```

Sie können auch die fluent-Methoden `SetAsDefault()` und `SetAsCancel()` auf Schaltflächen verwenden:

```go
dialog := app.Dialog.Question().
    SetMessage("Datei löschen?")

dialog.AddButton("Löschen").OnClick(func() {
    performDelete()
})

dialog.AddButton("Abbrechen").SetAsDefault().SetAsCancel()

dialog.Show()
```

**Best Practices:**
- **1-3 Schaltflächen:** Benutzer nicht überfordern
- **Klare Beschriftungen:** "Speichern" statt "OK"
- **Sichere Standardaktion:** Nicht-destruktive Aktion
- **Reihenfolge ist wichtig:** Wahrscheinlichste Aktion zuerst (außer Abbrechen)

### Benutzerdefiniertes Symbol

Legen Sie ein benutzerdefiniertes Symbol für den Dialog fest:

```go
app.Dialog.Info().
    SetTitle("Beispiel für benutzerdefiniertes Symbol").
    SetMessage("Verwendung eines benutzerdefinierten Symbols").
    SetIcon(myIconBytes).
    Show()
```

### Fensteranbindung

An ein bestimmtes Fenster anbinden:

```go
dialog := app.Dialog.Question().
    SetMessage("Fensterspezifische Frage").
    AttachToWindow(window)

dialog.AddButton("OK")
dialog.Show()
```

**Vorteile:**
- Dialog erscheint im richtigen Fenster
- Elternfenster während der Anzeige deaktiviert
- Besseres Multi-Fenster-Erlebnis

## Vollständige Beispiele

### Zerstörerische Aktion bestätigen

```go
func deleteFiles(app *application.App, paths []string) {
    // Löschung bestätigen
    message := fmt.Sprintf("%d Datei(en) löschen?", len(paths))
    if len(paths) == 1 {
        message = fmt.Sprintf("%s löschen?", filepath.Base(paths[0]))
    }

    dialog := app.Dialog.Question().
        SetTitle("Löschung bestätigen").
        SetMessage(message)

    deleteBtn := dialog.AddButton("Löschen")
    deleteBtn.OnClick(func() {
        // Löschung durchführen
        var errs []error
        for _, path := range paths {
            if err := os.Remove(path); err != nil {
                errs = append(errs, err)
            }
        }

        // Ergebnis anzeigen
        if len(errs) > 0 {
            app.Dialog.Error().
                SetTitle("Löschen fehlgeschlagen").
                SetMessage(fmt.Sprintf("Löschen von %d Datei(en) fehlgeschlagen", len(errs))).
                Show()
        } else {
            app.Dialog.Info().
                SetTitle("Löschen abgeschlossen").
                SetMessage(fmt.Sprintf("%d Datei(en) gelöscht", len(paths))).
                Show()
        }
    })

    cancelBtn := dialog.AddButton("Abbrechen")
    dialog.SetDefaultButton(cancelBtn)
    dialog.SetCancelButton(cancelBtn)
    dialog.Show()
}
```

### Beenden bestätigen

```go
// Beenden mit Bestätigungsdialog
result := app.Dialog.Question().
    SetTitle("Beenden bestätigen").
    SetMessage("Möchten Sie die Anwendung wirklich beenden?").
    AddButton("Beenden").
    AddButton("Abbrechen").
    Show()

if result == "Beenden" {
    app.Quit()
}
```
