## Native Dialoge

Wails bietet **native Systemdialoge**, die plattformübergreifend funktionieren: Nachrichten-Dialoge (Info, Warnung, Fehler, Frage), Dateidialoge (Öffnen, Speichern, Ordner) und benutzerdefinierte Dialogfenster mit plattformnativer Erscheinung und Verhalten.

## Schnellstart

```go
// Informationsdialog
app.Dialog.Info().
    SetTitle("Erfolg").
    SetMessage("Datei erfolgreich gespeichert!").
    Show()

// Fragen-Dialog mit Button-Callbacks
dialog := app.Dialog.Question().
    SetTitle("Bestätigen").
    SetMessage("Diese Datei löschen?")

deleteBtn := dialog.AddButton("Löschen")
deleteBtn.OnClick(func() {
    deleteFile()
})

cancelBtn := dialog.AddButton("Abbrechen")
dialog.SetDefaultButton(cancelBtn)
dialog.SetCancelButton(cancelBtn)
dialog.Show()

// Datei-Öffnen-Dialog
path, _ := app.Dialog.OpenFile().
    SetTitle("Bild auswählen").
    AddFilter("Bilder", "*.png;*.jpg").
    PromptForSingleSelection()
```

**Das war's!** Native Dialoge mit minimalem Code.

## Zugriff auf Dialoge

Dialoge werden über den `app.Dialog`-Manager angesprochen:

```go
app.Dialog.Info()
app.Dialog.Question()
app.Dialog.Warning()
app.Dialog.Error()
app.Dialog.OpenFile()
app.Dialog.SaveFile()
```

## Dialog-Typen

### Informationsdialog

Zeigen Sie einfache Nachrichten an:

```go
app.Dialog.Info().
    SetTitle("Willkommen").
    SetMessage("Willkommen in unserer Anwendung!").
    Show()
```

**Verwendungszwecke:**
- Erfolgsmeldungen
- Informationshinweise
- Bestätigungen der Fertigstellung

### Warnungsdialog

Zeigen Sie Warnungen an:

```go
app.Dialog.Warning().
    SetTitle("Warnung").
    SetMessage("Diese Aktion kann nicht rückgängig gemacht werden.").
    Show()
```

**Verwendungszwecke:**
- Nicht-kritische Warnungen
- Deprecation-Hinweise
- Vorsichtsmeldungen

### Fehlerdialog

Zeigen Sie Fehler an:

```go
app.Dialog.Error().
    SetTitle("Fehler").
    SetMessage("Speichern der Datei fehlgeschlagen: " + err.Error()).
    Show()
```

**Verwendungszwecke:**
- Fehlermeldungen
- Fehlschlag-Benachrichtigungen
- Ausnahmebehandlung

### Fragen-Dialog

Stellen Sie Benutzern Fragen und verarbeiten Sie Antworten über Button-Callbacks:

```go
dialog := app.Dialog.Question().
    SetTitle("Löschen bestätigen").
    SetMessage("Sind Sie sicher, dass Sie diese Datei löschen möchten?")

deleteBtn := dialog.AddButton("Löschen")
deleteBtn.OnClick(func() {
    deleteFile()
})

cancelBtn := dialog.AddButton("Abbrechen")
dialog.SetDefaultButton(cancelBtn)
dialog.SetCancelButton(cancelBtn)
dialog.Show()
```

**Verwendungszwecke:**
- Aktionen bestätigen
- Ja/Nein-Fragen
- Multiple Choice

## Dateidialoge

### Datei-Öffnen-Dialog

Wählen Sie Dateien zum Öffnen aus:

```go
path, err := app.Dialog.OpenFile().
    SetTitle("Bild auswählen").
    AddFilter("Bilder", "*.png;*.jpg;*.gif").
    AddFilter("Alle Dateien", "*.*").
    PromptForSingleSelection()

if err == nil && path != "" {
    openFile(path)
}
```

**Mehrfachauswahl:**

```go
paths, err := app.Dialog.OpenFile().
    SetTitle("Bilder auswählen").
    AddFilter("Bilder", "*.png;*.jpg").
    PromptForMultipleSelection()

if err == nil {
    for _, path := range paths {
        processFile(path)
    }
}
```

### Datei-Speichern-Dialog

Wählen Sie den Speicherort:

```go
path, err := app.Dialog.SaveFile().
    SetFilename("dokument.txt").
    AddFilter("Textdateien", "*.txt").
    AddFilter("Alle Dateien", "*.*").
    PromptForSingleSelection()

if err == nil && path != "" {
    saveFile(path)
}
```

### Ordner-Auswahl-Dialog

Wählen Sie ein Verzeichnis, indem Sie den Datei-Öffnen-Dialog mit aktivierter Ordnerauswahl verwenden:

```go
path, err := app.Dialog.OpenFile().
    SetTitle("Ausgabeordner auswählen").
    CanChooseDirectories(true).
    CanChooseFiles(false).
    PromptForSingleSelection()

if err == nil && path != "" {
    exportToFolder(path)
}
```

## Dialog-Optionen

### Titel und Nachricht

```go
dialog := app.Dialog.Info().
    SetTitle("Erfolg").
    SetMessage("Vorgang erfolgreich abgeschlossen!")
```

### Buttons

**Standard-Button für einfache Dialoge:**

Info-, Warnungs- und Fehlerdialoge zeigen einen Standard-"OK"-Button an:

```go
app.Dialog.Info().
    SetMessage("Fertig!").
    Show()
```

**Benutzerdefinierte Buttons für Fragen-Dialoge:**

Verwenden Sie `AddButton()`, um Buttons hinzuzufügen, die ein `*Button` zurückgeben, das Sie mit Callbacks konfigurieren können:

```go
dialog := app.Dialog.Question().
    SetMessage("Aktion wählen")

save := dialog.AddButton("Speichern")
save.OnClick(func() {
    saveDocument()
})

dontSave := dialog.AddButton("Nicht speichern")
dontSave.OnClick(func() {
    discardChanges()
})

cancel := dialog.AddButton("Abbrechen")
// Kein Callback nötig - schließt nur den Dialog

dialog.SetDefaultButton(save)
dialog.SetCancelButton(cancel)
dialog.Show()
```

**Standard- und Abbrechen-Buttons:**

Verwenden Sie `SetDefaultButton()`, um anzugeben, welcher Button hervorgehoben und durch Eingabe (Enter) ausgelöst wird.
Verwenden Sie `SetCancelButton()`, um anzugeben, welcher Button durch Escape ausgelöst wird.

```go
dialog := app.Dialog.Question().
    SetMessage("Datei löschen?")

deleteBtn := dialog.AddButton("Löschen")
deleteBtn.OnClick(func() {
    performDelete()
})

cancelBtn := dialog.AddButton("Abbrechen")
dialog.SetDefaultButton(cancelBtn)  // Sichere Option standardmäßig hervorgehoben
dialog.SetCancelButton(cancelBtn)   // Escape löst Abbrechen aus
dialog.Show()
```

### Fensteranbindung

Binden Sie den Dialog an ein bestimmtes Fenster:

```go
dialog := app.Dialog.Info().
    SetMessage("Fensterspezifische Nachricht").
    AttachToWindow(window)

dialog.Show()
```

**Verhalten:**
- Dialog erscheint zentriert über dem Elternfenster
- Elternfenster ist während der Anzeige des Dialogs deaktiviert
- Dialog bewegt sich mit dem Elternfenster (macOS)

## Häufige Muster

### Bestätigung vor zerstörerischer Aktion

```go
func deleteFile(app *application.App, path string) {
    dialog := app.Dialog.Question().
        SetTitle("Löschen bestätigen").
        SetMessage(fmt.Sprintf("%s löschen?", filepath.Base(path)))

    deleteBtn := dialog.AddButton("Löschen")
    deleteBtn.OnClick(func() {
        if err := os.Remove(path); err != nil {
            app.Dialog.Error().
                SetTitle("Löschen fehlgeschlagen").
                SetMessage(err.Error()).
                Show()
        }
    })

    cancelBtn := dialog.AddButton("Abbrechen")
    dialog.SetDefaultButton(cancelBtn)
    dialog.SetCancelButton(cancelBtn)
    dialog.Show()
}
```
