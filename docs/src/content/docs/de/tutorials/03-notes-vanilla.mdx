In diesem Tutorial erstellen Sie eine Notizenanwendung, die Dateioperationen, native Dialoge und moderne Desktop-App-Muster mit Wails v3 demonstriert.

![Screenshot der Notizen-App](../../../../assets/notes-app.png)

## Was Sie erstellen werden

- Eine vollständige Notizen-App mit Funktionen zum Erstellen, Bearbeiten und Löschen
- Native Dateispeicher-/Dateioffen-Dialoge zum Importieren und Exportieren von Notizen
- Automatische Speicherung beim Tippen mit Debounce, um unnötige Updates zu reduzieren
- Professionelles zweispaltiges Layout (Seitenleiste + Editor), das Apple Notes nachahmt

## Was Sie lernen werden

- Verwendung nativer Dateidialoge in Wails (`SaveFileDialog`, `OpenFileDialog`, `InfoDialog`)
- Arbeiten mit JSON-Dateien zur Datenspeicherung
- Implementierung von Debounce-Auto-Save-Mustern
- Aufbau professioneller Desktop-UIs mit modernem CSS
- Korrekte JSON-Serialisierung von Go-Strukturen

2. **Erstellen Sie den NotesService**

   Erstellen Sie eine neue Datei `notesservice.go` im Projektstamm:

   ```go
   package main

   import (
   	"encoding/json"
   	"errors"
   	"os"
   	"time"

   	"github.com/wailsapp/wails/v3/pkg/application"
   )

   type Note struct {
   	ID        string    `json:"id"`
   	Title     string    `json:"title"`
   	Content   string    `json:"content"`
   	CreatedAt time.Time `json:"createdAt"`
   	UpdatedAt time.Time `json:"updatedAt"`
   }

   type NotesService struct {
   	notes []Note
   }

   func NewNotesService() *NotesService {
   	return &NotesService{
   		notes: make([]Note, 0),
   	}
   }

   // GetAll gibt alle Notizen zurück
   func (n *NotesService) GetAll() []Note {
   	return n.notes
   }

   // Create erstellt eine neue Notiz
   func (n *NotesService) Create(title, content string) Note {
   	note := Note{
   		ID:        generateID(),
   		Title:     title,
   		Content:   content,
   		CreatedAt: time.Now(),
   		UpdatedAt: time.Now(),
   	}
   	n.notes = append(n.notes, note)
   	return note
   }

   // Update aktualisiert eine vorhandene Notiz
   func (n *NotesService) Update(id, title, content string) error {
   	for i := range n.notes {
   		if n.notes[i].ID == id {
   			n.notes[i].Title = title
   			n.notes[i].Content = content
   			n.notes[i].UpdatedAt = time.Now()
   			return nil
   		}
   	}
   	return errors.New("note not found")
   }

   // Delete löscht eine Notiz
   func (n *NotesService) Delete(id string) error {
   	for i := range n.notes {
   		if n.notes[i].ID == id {
   			n.notes = append(n.notes[:i], n.notes[i+1:]...)
   			return nil
   		}
   	}
   	return errors.New("note not found")
   }

   // SaveToFile speichert Notizen in einer Datei
   func (n *NotesService) SaveToFile() error {
   	path, err := application.Get().Dialog.SaveFile().
   		SetFilename("notes.json").
   		AddFilter("JSON Files", "*.json").
   		PromptForSingleSelection()

   	if err != nil {
   		return err
   	}

   	data, err := json.MarshalIndent(n.notes, "", "  ")
   	if err != nil {
   		return err
   	}

   	if err := os.WriteFile(path, data, 0644); err != nil {
   		return err
   	}

   	application.Get().Dialog.Info().
   		SetTitle("Success").
   		SetMessage("Notes saved successfully!").
   		Show()

   	return nil
   }

   // LoadFromFile lädt Notizen aus einer Datei
   func (n *NotesService) LoadFromFile() error {
   	path, err := application.Get().Dialog.OpenFile().
   		AddFilter("JSON Files", "*.json").
   		PromptForSingleSelection()

   	if err != nil {
   		return err
   	}

   	data, err := os.ReadFile(path)
   	if err != nil {
   		return err
   	}

   	var notes []Note
   	if err := json.Unmarshal(data, &notes); err != nil {
   		return err
   	}

   	n.notes = notes

   	application.Get().Dialog.Info().
   		SetTitle("Success").
   		SetMessage("Notes loaded successfully!").
   		Show()

   	return nil
   }

   func generateID() string {
   	return time.Now().Format("20060102150405")
   }
   ```

   **Was hier passiert:**

   - **Note struct**: Definiert die Datenstruktur mit JSON-Tags (kleingeschrieben) für die korrekte Serialisierung
   - **CRUD-Operationen**: GetAll, Create, Update und Delete zum Verwalten von Notizen im Speicher
   - **Dateidialoge**: Verwendet `application.Get().Dialog.SaveFile()` und `application.Get().Dialog.OpenFile()`, um auf native Dialoge zuzugreifen
   - **Info-Dialoge**: Zeigt Erfolgsmeldungen mit `application.Get().Dialog.Info()` an
   - **ID-Generierung**: Einfacher, auf Zeitstempeln basierender ID-Generator

3. **Aktualisieren Sie main.go**

   Ersetzen Sie den Inhalt von `main.go`:

   import (
   	"embed"
   	_ "embed"
   	"log"

   //go:embed all:frontend/dist

   		URL:              "/",
   	})

   	if err != nil {
   		log.Fatal(err)
   	}
   }

6. **Implementieren Sie die JavaScript-Logik**

   Ersetzen Sie `frontend/src/main.js`:

   ```javascript
   import { NotesService } from '../bindings/changeme'

   let notes = []
   let currentNote = null

   // Load notes on startup
   async function loadNotes() {
       notes = await NotesService.GetAll()
       renderNotesList()
   }

   // Render notes list
   function renderNotesList() {
       const notesList = document.getElementById('notes-list')

       if (notes.length === 0) {
           notesList.innerHTML = '<div style="padding: 20px; text-align: center; color: #999;">No notes yet</div>'
           return
       }

       notesList.innerHTML = notes.map(note => `
           <div class="note-item ${currentNote?.id === note.id ? 'active' : ''}" data-id="${note.id}">
               <h3>${note.title || 'Untitled'}</h3>
               <p>${note.content || 'No content'}</p>
           </div>
       `).join('')

       // Add click handlers
       document.querySelectorAll('.note-item').forEach(item => {
           item.addEventListener('click', () => {
               const id = item.dataset.id
               selectNote(id)
           })
       })
   }

   // Select a note
   function selectNote(id) {
       currentNote = notes.find(n => n.id === id)
       if (currentNote) {
           document.getElementById('empty-state').style.display = 'none'
           document.getElementById('note-editor').style.display = 'flex'
           document.getElementById('note-title').value = currentNote.title
           document.getElementById('note-content').value = currentNote.content
           document.getElementById('last-updated').textContent =
               `Last updated: ${new Date(currentNote.updatedAt).toLocaleString()}`
           renderNotesList()
       }
   }

   // Create new note
   document.getElementById('new-note-btn').addEventListener('click', async () => {
       const note = await NotesService.Create('Untitled', '')
       notes.push(note)
       selectNote(note.id)
       // Focus the title input and select all text so user can immediately type
       const titleInput = document.getElementById('note-title')
       titleInput.focus()
       titleInput.select()
   })

   // Update note on input
   let updateTimeout
   function scheduleUpdate() {
       clearTimeout(updateTimeout)
       updateTimeout = setTimeout(async () => {
           if (currentNote) {
               const title = document.getElementById('note-title').value
               const content = document.getElementById('note-content').value

               await NotesService.Update(currentNote.id, title, content)

               // Update local copy
               const note = notes.find(n => n.id === currentNote.id)
               if (note) {
                   note.title = title
                   note.content = content
                   note.updatedAt = new Date().toISOString()
               }

               renderNotesList()
               document.getElementById('last-updated').textContent =
                   `Last updated: ${new Date().toLocaleString()}`
           }
       }, 500)
   }

   document.getElementById('note-title').addEventListener('input', scheduleUpdate)
   document.getElementById('note-content').addEventListener('input', scheduleUpdate)

   // Delete note
   document.getElementById('delete-btn').addEventListener('click', async () => {
       if (!currentNote) return

       try {
           await NotesService.Delete(currentNote.id)
           notes = notes.filter(n => n.id !== currentNote.id)
           currentNote = null
           document.getElementById('empty-state').style.display = 'flex'
           document.getElementById('note-editor').style.display = 'none'
           renderNotesList()
       } catch (error) {
           console.error('Delete failed:', error)
       }
   })

   // Save to file
   document.getElementById('save-btn').addEventListener('click', async () => {
       try {
           await NotesService.SaveToFile()
       } catch (error) {
           if (error) console.error('Save failed:', error)
       }
   })```go
   // Aus Datei laden

           renderNotesList()
       } catch (error) {

   - **Auto-Save**: Ein Debounce von 500ms verhindert übermäßige Aufrufe an das Backend während der Eingabe

## Schlüsselkonzepte

### Dialogfunktionen auf Paketebene

In Wails v3 sind Datei- und Nachrichten-Dialoge Funktionen auf Paketebene, keine Methoden der App-Instanz:

```go
// Korrekt - Funktion auf Paketebene
path, err := application.SaveFileDialog().
    SetFilename("notes.json").
    AddFilter("JSON Files", "*.json").
    PromptForSingleSelection()

// Falsch - nicht verfügbar
path, err := app.Dialog.SaveFile() // Existiert nicht
```

### JSON-Tag-Zuordnung

Go-Struktur-JSON-Tags müssen kleingeschrieben sein, um dem JavaScript-Eigenschaftszugriff zu entsprechen:

```go
type Note struct {
    ID string `json:"id"` // Muss kleingeschrieben sein
}
```

```javascript
// JavaScript greift mit kleingeschriebenen Namen zu
const noteId = note.id // Korrekt
const noteId = note.ID // Wäre undefined
```

### Debounced Auto-Save

Der 500ms-Debounce reduziert unnötige Aufrufe an das Backend:

```javascript
let updateTimeout
function scheduleUpdate() {
    clearTimeout(updateTimeout) // Vorherigen Timer abbrechen
    updateTimeout = setTimeout(async () => {
        // Speichert nur, wenn der Benutzer 500ms lang nicht tippt
        await NotesService.Update(currentNote.id, title, content)
    }, 500)
}
```
