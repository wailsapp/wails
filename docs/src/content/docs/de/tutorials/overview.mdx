### QR-Code-Dienst

<Image src={qr1} alt="QR-Code-Beispiel" style="max-width: 400px; margin: 1rem auto; display: block;" />

Lernen Sie die Grundlagen von Wails-Diensten kennen, indem Sie einen QR-Code-Generator erstellen. Dieses Tutorial führt Sie in die Kernkonzepte ein, Ihre Anwendungslogik in wiederverwendbare Dienste zu organisieren.

**Was Sie lernen werden:**
- Wie man einen Wails-Dienst erstellt und strukturiert
- Verwaltung externer Go-Abhängigkeiten
- Bindung von Go-Methoden an Ihre Frontend-Oberfläche
- Datenübertragung zwischen Go und JavaScript
- Organisation des Codes für bessere Wartbarkeit

**Am besten geeignet für:** Wails-Einsteiger, die die Dienstarchitektur verstehen möchten

### TODO-Liste

<Image src={todoApp} alt="TODO-Listen-Anwendung" style="max-width: 400px; margin: 1rem auto; display: block;" />

Erstellen Sie eine vollständige TODO-Listen-Anwendung mit einer schönen, modernen Benutzeroberfläche. Dieses praktische Tutorial vermittelt Ihnen die wichtigsten Wails-Muster anhand einer realen Anwendung mit Vanilla JavaScript.

**Was Sie lernen werden:**
- Dienstbasierte Architektur mit threadsicherem State-Management
- CRUD-Operationen (Create, Read, Update, Delete)
- Typsichere Bindungen zwischen Go und JavaScript
- Erstellung moderner Benutzeroberflächen ohne Framework-Komplexität
- Korrekte Fehlerbehandlung und Validierungsmuster

**Dauer bis zur Fertigstellung:** ca. 20 Minuten

**Am besten geeignet für:** Ihre erste vollständige Wails-Anwendung – ideal, um die Grundlagen zu verstehen, bevor Sie Framework-Komplexität hinzufügen

### Notizen

<Image src={notesApp} alt="Notizen-Anwendung" style="max-width: 600px; margin: 1rem auto; display: block;" />

Erstellen Sie eine Anwendung im Stil von Apple Notes mit nativen Dateidialogen und Auto-Save-Funktionalität. Dieses Tutorial demonstriert desktop-spezifische Funktionen wie Dateioperationen, native Dialoge und professionelle UI-Muster.

**Was Sie lernen werden:**
- Native Dateidialoge (Speichern, Öffnen, Informationen)
- JSON-basierte Datenpersistenz
- Debounced Auto-Save-Muster
- Professionelle zweispaltige Desktop-Layouts
- Arbeit mit Dateisystemoperationen in Go

**Dauer bis zur Fertigstellung:** ca. 30 Minuten

**Am besten geeignet für:** Das Erlernen desktop-spezifischer Funktionen wie Dateioperationen und native OS-Dialoge
