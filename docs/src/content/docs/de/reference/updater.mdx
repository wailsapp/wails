# Referenz zur Updater-API

Die Updater-API bietet automatische Update-Funktionalität für Wails-Anwendungen, einschließlich Überprüfung auf Updates, Herunterladen und Installieren.

## Go-API

### CreateUpdaterService

Erstellt einen neuen Updater-Dienst.

```go
func CreateUpdaterService(currentVersion string, opts ...UpdaterOption) (*UpdaterService, error)
```

**Parameter:**
- `currentVersion` - Die aktuelle Version Ihrer Anwendung (z. B. "1.0.0")
- `opts` - Konfigurationsoptionen

**Rückgabewerte:**
- `*UpdaterService` - Die Instanz des Updater-Dienstes
- `error` - Fehler, falls die Erstellung fehlschlägt

**Beispiel:**
```go
updater, err := application.CreateUpdaterService(
    "1.0.0",
    application.WithUpdateURL("https://updates.example.com/myapp/"),
)
```

### Funktionen für UpdaterOption

#### WithUpdateURL

Legt die Basis-URL für Update-Manifeste fest.

```go
func WithUpdateURL(url string) UpdaterOption
```

**Erforderlich.** Die URL sollte auf das Verzeichnis verweisen, das Ihr `update.json`-Manifest enthält.

#### WithCheckInterval

Legt das Intervall für automatische Hintergrund-Updatesprüchungen fest.

```go
func WithCheckInterval(interval time.Duration) UpdaterOption
```

Auf `0` setzen, um die automatische Überprüfung zu deaktivieren. Der Standardwert ist `0` (deaktiviert).

#### WithAllowPrerelease

Ermöglicht, dass Pre-Release-Versionen als Updates berücksichtigt werden.

```go
func WithAllowPrerelease(allow bool) UpdaterOption
```

Der Standardwert ist `false`.

#### WithChannel

Legt den Update-Kanal fest (z. B. "stable", "beta", "canary").

```go
func WithChannel(channel string) UpdaterOption
```

Der Kanal wird an den Pfad der Update-URL angehängt.

#### WithPublicKey

Legt den öffentlichen Schlüssel zur Signaturenüberprüfung fest.

```go
func WithPublicKey(key string) UpdaterOption
```

Der Schlüssel sollte ein base64-codierter Ed25519-publiksschlüssel sein.

#### WithRequireSignature

Fordert erfordert, dass alle Updates signiert sind.

```go
func WithRequireSignature(require bool) UpdaterOption
```

Der Standardwert ist `false`.

### Methoden von UpdaterService

#### GetCurrentVersion

Gibt die aktuelle Anwendungsversion zurück.

```go
func (u *UpdaterService) GetCurrentVersion() string
```

#### CheckForUpdate

Überprüft, ob eine neue Version verfügbar ist.

```go
func (u *UpdaterService) CheckForUpdate() (*UpdateInfo, error)
```

**Rückgabewerte:**
- `*UpdateInfo` - Update-Informationen, oder `nil`, wenn kein Update verfügbar ist
- `error` - Fehler, falls die Überprüfung fehlschlägt

#### DownloadUpdate

Lädt das verfügbare Update herunter.

```go
func (u *UpdaterService) DownloadUpdate() error
```

Sendet `updater:progress`-Ereignisse während des Downloads.

#### ApplyUpdate

Wendet das heruntergeladene Update an.

```go
func (u *UpdaterService) ApplyUpdate() error
```

Dies startet die Anwendung neu.

#### DownloadAndApply

Lädt ein Update herunter und wendet es in einem Aufruf an.

```go
func (u *UpdaterService) DownloadAndApply() error
```

#### GetState

Gibt den aktuellen Updater-Status zurück.

```go
func (u *UpdaterService) GetState() string
```

**Mögliche Werte:**
- `"idle"` - Kein Update läuft
- `"checking"` - Wird nach Updates gesucht
- `"available"` - Update verfügbar
- `"downloading"` - Update wird heruntergeladen
- `"ready"` - Update heruntergeladen, bereit zur Installation
- `"installing"` - Update wird installiert
- `"error"` - Ein Fehler ist aufgetreten

#### GetUpdateInfo

Gibt Informationen über das verfügbare Update zurück.

```go
func (u *UpdaterService) GetUpdateInfo() *UpdateInfo
```

#### GetLastError

Gibt die letzte Fehlermeldung zurück.

```go
func (u *UpdaterService) GetLastError() string
```

#### Reset

Setzt den Updater-Status zurück.

```go
func (u *UpdaterService) Reset()
```

### UpdateInfo

Informationen über ein verfügbares Update.

```go
type UpdateInfo struct {
    Version      string    `json:"version"`
    ReleaseDate  time.Time `json:"releaseDate"`
    ReleaseNotes string    `json:"releaseNotes"`
    Size         int64     `json:"size"`
    PatchSize    int64     `json:"patchSize,omitempty"`
    Mandatory    bool      `json:"mandatory"`
    HasPatch     bool      `json:"hasPatch"`
}
```

| Feld | Typ | Beschreibung |
|-------|--|-------------|
| `Version` | `string` | Die neue Versionsnummer |
| `ReleaseDate` | `time.Time` | Wann diese Version veröffentlicht wurde |
| `ReleaseNotes` | `string` | Versionshinweise (Markdown) |
| `Size` | `int64` | Gesamte Download-Größe in Bytes |
| `PatchSize` | `int64` | Patch-Größe in Bytes (falls verfügbar) |
| `Mandatory` | `bool` | Ob dieses Update erforderlich ist |
| `HasPatch` | `bool` | Ob ein Patch verfügbar ist |

### UpdaterConfig

Konfiguration für den Updater.

```go
type UpdaterConfig struct {
    UpdateURL        string
    CheckInterval    time.Duration
    AllowPrerelease  bool
    PublicKey        string
    RequireSignature bool
    Channel          string
}
```

### Ereignisse

#### updater:progress

Wird während des Downloads mit Fortschrittsinformationen gesendet.

```typescript
import { Events } from '@wailsio/runtime';

Events.On('updater:progress', (data: DownloadProgress) => {
  console.log(`${data.percentage}% heruntergeladen`);
});
```

## Format des Update-Manifests

Das Update-Manifest (`update.json`) muss an Ihrer Update-URL gehostet werden.

### Schema

```typescript
interface Manifest {
  version: string;
  release_date: string;      // ISO 8601
  release_notes?: string;
  platforms: {
    [platform: string]: PlatformUpdate;
  };
  minimum_version?: string;
  mandatory?: boolean;
}

interface PlatformUpdate {
  url: string;
  size: number;
  checksum: string;          // "sha256:..."
  signature?: string;
  patches?: PatchInfo[];
}

interface PatchInfo {
  from: string;              // Version, ab der dieser Patch gilt
  url: string;
  size: number;
  checksum: string;
  signature?: string;
}
```

|--|-|
| macOS (Apple Silicon) | `macos-arm64` |

| Windows (64-Bit) | `windows-amd64` |
| Linux (64-Bit) | `linux-amd64` |
| Linux (ARM64) | `linux-arm64` |

## Serviceintegration

### Registrieren des Dienstes

```go
app := application.New(application.Options{
    Name: "MyApp",
    Services: []application.Service{
        application.NewService(updater),
    },
})
```

### Dienstlebenszyklus

Der Updater-Dienst implementiert die `Service`-Schnittstelle:

```go
type Service interface {
    ServiceName() string
    ServiceStartup(ctx context.Context, options ServiceOptions) error
    ServiceShutdown() error
}
```

- **ServiceStartup**: Startet die Hintergrundprüfung auf Updates, wenn konfiguriert
- **ServiceShutdown**: Stoppt die Hintergrundprüfung

### Event-Handler

Registrieren Sie Handler in Go:

```go
updater.OnUpdateAvailable(func(info *UpdateInfo) {
    log.Printf("Update verfügbar: %s", info.Version)
})

updater.OnDownloadProgress(func(downloaded, total int64, percentage float64) {
    log.Printf("Download: %.1f%%", percentage)
})

updater.OnError(func(err string) {
    log.Printf("Update-Fehler: %s", err)
})
```

## Sicherheitsüberlegungen

### HTTPS

Verwenden Sie immer HTTPS für Update-URLs, um Man-in-the-Middle-Angriffe zu verhindern.

### Prüfsummenüberprüfung

Alle Downloads werden vor der Installation gegen SHA256-Prüfsummen überprüft.

### Signaturenüberprüfung

Aktivieren Sie die Signaturenüberprüfung für Produktionsanwendungen:

```go
updater, _ := application.CreateUpdaterService(
    "1.0.0",
    application.WithUpdateURL("https://updates.example.com/myapp/"),
    application.WithRequireSignature(true),
    application.WithPublicKey("MCowBQYDK2VwAyEA..."),
)
```

### Codesignierung

Stellen Sie sicher, dass Ihre Anwendungsbinärdateien ordnungsgemäß codiert signiert sind:
- macOS: Developer ID Application-Zertifikat
- Windows: Codesignier-Zertifikat mit Zeitstempel

Details finden Sie im [Leitfaden zur Codesignierung](/de/guides/build/signing).
