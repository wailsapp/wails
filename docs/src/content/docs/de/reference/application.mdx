## Übersicht

Die `Application` ist das Herzstück Ihrer Wails-App. Sie verwaltet Fenster, Dienste, Ereignisse und bietet Zugriff auf alle plattformspezifischen Funktionen.

## Erstellen einer Anwendung

```go
import "github.com/wailsapp/wails/v3/pkg/application"

app := application.New(application.Options{
    Name:        "My App",
    Description: "My awesome application",
    Services: []application.Service{
        application.NewService(&MyService{}),
    },
})
```

## Kernmethoden

### Run()

Startet die Ereignisschleife der Anwendung.

```go
func (a *App) Run() error
```

**Beispiel:**

```go
err := app.Run()
if err != nil {
    log.Fatal(err)
}
```

**Rückgabe:** Fehler, wenn der Start fehlschlägt

### Quit()

Fährt die Anwendung sauber herunter.

```go
func (a *App) Quit()
```

**Beispiel:**

```go
// In einem Menü-Handler
menu.Add("Quit").OnClick(func(ctx *application.Context) {
    app.Quit()
})
```

### Config()

Gibt die Anwendungskonfiguration zurück.

```go
func (a *App) Config() Options
```

**Beispiel:**

```go
config := app.Config()
fmt.Println("App name:", config.Name)
```

## Fensterverwaltung

### app.Window.New()

Erstellt ein neues WebView-Fenster mit Standardoptionen.

```go
func (wm *WindowManager) New() *WebviewWindow
```

**Beispiel:**

```go
window := app.Window.New()
window.Show()
```

### app.Window.NewWithOptions()

Erstellt ein neues WebView-Fenster mit benutzerdefinierten Optionen.

```go
func (wm *WindowManager) NewWithOptions(options WebviewWindowOptions) *WebviewWindow
```

**Beispiel:**

```go
window := app.Window.NewWithOptions(application.WebviewWindowOptions{
    Title:  "My Window",
    Width:  800,
    Height: 600,
    BackgroundColour: application.NewRGB(255, 255, 255),
})
```

### app.Window.GetByName()

Ruft ein Fenster anhand seines Namens ab.

```go
func (wm *WindowManager) GetByName(name string) Window
```

**Beispiel:**

```go
window := app.Window.GetByName("main")
if window != nil {
    window.Show()
}
```

### app.Window.GetAll()

Gibt alle Fenster der Anwendung zurück.

```go
func (wm *WindowManager) GetAll() []Window
```

**Beispiel:**

```go
windows := app.Window.GetAll()
for _, window := range windows {
    fmt.Println("Window:", window.Name())
}
```

## Manager

Die Anwendung bietet über Eigenschaften Zugriff auf verschiedene Manager:

```go
app.Window      // Fensterverwaltung
app.Menu        // Menüverwaltung
app.Dialog      // Dialogverwaltung
app.Event       // Ereignisverwaltung
app.Clipboard   // Zwischenablageoperationen
app.Screen      // Bildschirminformationen
app.SystemTray  // Systemablage
app.Browser     // Browseroperationen
app.Env         // Umgebungsvariablen
```

### Beispielverwendung

```go
// Fenster erstellen
window := app.Window.New()

// Dialog anzeigen
app.Dialog.Info().SetMessage("Hello!").Show()

// In die Zwischenablage kopieren
app.Clipboard.SetText("Copied text")

// Bildschirme abrufen
screens := app.Screen.GetAll()
```

## Dienstverwaltung

### RegisterService()

Registriert einen Dienst bei der Anwendung.

```go
func (a *App) RegisterService(service Service) error
```

**Beispiel:**

```go
type MyService struct {
    app *application.Application
}

func NewMyService(app *application.Application) *MyService {
    return &MyService{app: app}
}

// Registrierung nach der App-Erstellung
app.RegisterService(application.NewService(NewMyService(app)))
```

## Ereignisverwaltung

### app.Event.Emit()

Sendet ein benutzerdefiniertes Ereignis.

```go
func (em *EventManager) Emit(name string, data ...interface{})
```

**Beispiel:**

```go
// Ereignis mit Daten senden
app.Event.Emit("user-logged-in", map[string]interface{}{
    "username": "john",
    "timestamp": time.Now(),
})
```

### app.Event.On()

Hört auf benutzerdefinierte Ereignisse.

```go
func (em *EventManager) On(name string, callback func(*CustomEvent))
```

**Beispiel:**

```go
app.Event.On("user-logged-in", func(e *application.CustomEvent) {
    data := e.Data.(map[string]interface{})
    username := data["username"].(string)
    fmt.Println("User logged in:", username)
})
```

### app.Event.OnApplicationEvent()

Hört auf Anwendungslebenszyklusereignisse.

```go
func (em *EventManager) OnApplicationEvent(eventType ApplicationEventType, callback func(*ApplicationEvent)) func()
```

**Beispiel:**

```go
// Auf Herunterfahren lauschen
app.Event.OnApplicationEvent(application.EventApplicationShutdown, func(e *application.ApplicationEvent) {
    fmt.Println("Application shutting down")
    // Aufräumen
})
```

## Dialogmethoden

Dialoge werden über den `app.Dialog`-Manager aufgerufen. Siehe [Dialogs API](/reference/dialogs/) für die vollständige Referenz.

### Nachrichten-Dialoge

```go
// Informationsdialog
app.Dialog.Info().
    SetTitle("Success").
    SetMessage("Operation completed!").
    Show()

// Fehlerdialog
app.Dialog.Error().
    SetTitle("Error").
    SetMessage("Something went wrong.").
    Show()

// Warnungsdialog
app.Dialog.Warning().
    SetTitle("Warning").
    SetMessage("This action cannot be undone.").
    Show()
```

### Frage-Dialoge

Frage-Dialoge verwenden Schaltflächen-Callbacks, um auf Benutzereingaben zu reagieren:

```go
dialog := app.Dialog.Question().
    SetTitle("Confirm").
    SetMessage("Continue?")

yes := dialog.AddButton("Yes")
yes.OnClick(func() {
    // Ja verarbeiten
})

no := dialog.AddButton("No")
no.OnClick(func() {
    // Nein verarbeiten
})

dialog.SetDefaultButton(yes)
dialog.SetCancelButton(no)
dialog.Show()
```

### Datei-Dialoge

```go
// Datei öffnen Dialog
path, err := app.Dialog.OpenFile().
    SetTitle("Select File").
    AddFilter("Images", "*.png;*.jpg").
    PromptForSingleSelection()

// Datei speichern Dialog
path, err := app.Dialog.SaveFile().
    SetTitle("Save File").
    SetFilename("document.pdf").
    AddFilter("PDF", "*.pdf").
    PromptForSingleSelection()

// Ordnerauswahl (OpenFile mit Verzeichnisoptionen verwenden)
path, err := app.Dialog.OpenFile().
    SetTitle("Select Folder").
    CanChooseDirectories(true).
    CanChooseFiles(false).
    PromptForSingleSelection()
```

## Logger

Die Anwendung stellt einen strukturierten Logger bereit:

```go
app.Logger.Info("Message", "key", "value")
app.Logger.Error("Error occurred", "error", err)
app.Logger.Debug("Debug info")
app.Logger.Warn("Warning message")
```

**Beispiel:**

```go
func (s *MyService) ProcessData(data string) error {
    s.app.Logger.Info("Processing data", "length", len(data))
    
    if err := process(data); err != nil {
        s.app.Logger.Error("Processing failed", "error", err)
        return err
    }
    
    s.app.Logger.Info("Processing complete")
    return nil
}
```

## Raw-Nachrichten-Verarbeitung

Für Anwendungen, die eine direkte, low-level-Kontrolle über die Kommunikation vom Frontend zum Backend benötigen, bietet Wails die Option `RawMessageHandler`. Dies umgeht das Standard-Bindungssystem.

:::note
Raw-Nachrichten sollten nur als letzter Ausweg verwendet werden. Das Standard-Bindungssystem ist hochgradig optimiert und für fast alle Anwendungen ausreichend. Verwenden Sie Raw-Nachrichten nur, wenn Sie Ihre Anwendung profiliert und bestätigt haben, dass Bindungen ein Engpass sind.
:::

### RawMessageHandler

Der `RawMessageHandler`-Callback empfängt rohe Nachrichten, die vom Frontend mit `System.invoke()` gesendet werden.

```go
func (a *App) RawMessageHandler(window Window, message string)
```

**Beispiel:**

```go
app := application.New(application.Options{
    Name: "My App",
    RawMessageHandler: func(window application.Window, message string) {
        // Verarbeite die rohe Nachricht
        fmt.Printf("Received from %s: %s\n", window.Name(), message)

        // Du kannst über Ereignisse antworten
        window.EmitEvent("response", processMessage(message))
    },
})
```


Weitere Details finden Sie im [Raw Messages Guide](/de/guides/raw-messages).

## Plattformspezifische Optionen

### Windows-Optionen

Konfigurieren Sie Windows-spezifisches Verhalten auf Anwendungsebene:

```go
app := application.New(application.Options{
    Name: "My App",
    Windows: application.WindowsOptions{
        // WebView2-Browser-Flags (gelten für ALLE Fenster)        EnabledFeatures:       []string{"msWebView2EnableDraggableRegions"},
        DisabledFeatures:      []string{"msExperimentalFeature"},
        AdditionalBrowserArgs: []string{"--remote-debugging-port=9222"},

        // Other Windows options
        WndClass:                      "MyAppClass",
        WebviewUserDataPath:           "",  // Default: %APPDATA%\[BinaryName.exe]
        WebviewBrowserPath:            "",  // Default: system WebView2
        DisableQuitOnLastWindowClosed: false,
    },
})
```

**Browser-Flags:**
- `EnabledFeatures` - WebView2-Funktionsflags zum Aktivieren
- `DisabledFeatures` - WebView2-Funktionsflags zum Deaktivieren
- `AdditionalBrowserArgs` - Chromium-Befehlszeilenargumente

Siehe [Fensteroptionen - Anwendungsebene Fensteroptionen](/features/windows/options#application-level-windows-options) für detaillierte Dokumentation.

### Mac-Optionen

```go
app := application.New(application.Options{
    Name: "My App",
    Mac: application.MacOptions{
        ActivationPolicy: application.ActivationPolicyRegular,
        ApplicationShouldTerminateAfterLastWindowClosed: true,
    },
})
```

### Linux-Optionen

```go
app := application.New(application.Options{
    Name: "My App",
    Linux: application.LinuxOptions{
        ProgramName:                   "my-app",
        DisableQuitOnLastWindowClosed: false,
    },
})
```

## Vollständiges Anwendungsbeispiel

```go
package main

import (
    "github.com/wailsapp/wails/v3/pkg/application"
)

func main() {
    app := application.New(application.Options{
        Name:        "My Application",
        Description: "A demo application",
        Mac: application.MacOptions{
            ApplicationShouldTerminateAfterLastWindowClosed: true,
        },
    })

    // Create main window
    window := app.Window.NewWithOptions(application.WebviewWindowOptions{
        Title:            "My App",
        Width:            1024,
        Height:           768,
        MinWidth:         800,
        MinHeight:        600,
        BackgroundColour: application.NewRGB(255, 255, 255),
        URL:              "http://wails.localhost/",
    })

    window.Centre()
    window.Show()

    app.Run()
}
