## Übersicht

Die Wails CLI (`wails3`) stellt Befehle zum Erstellen, Entwickeln, Bauen und Verwalten von Wails-Anwendungen bereit.

## Demnächst verfügbar

Dieser Abschnitt befindet sich noch im Aufbau. Verwenden Sie vorerst `wails3 --help` oder `wails3 [command] --help` für die Dokumentation der Befehle.

**Häufig verwendete Befehle:**

```bash
wails3 init          # Create a new project
wails3 dev           # Run in development mode
wails3 build         # Build for production
wails3 generate      # Generate bindings or runtime
wails3 doctor        # Check development environment
```
