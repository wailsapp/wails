## Übersicht

Die Menü-API bietet Methoden zum Erstellen und Verwalten von Anwendungsmenüs, Kontextmenüs und Systemablaufleistenmenüs.

**Menütypen:**
- **Anwendungsmenüs** - Obere Menüleiste (Datei, Bearbeiten, usw.)
- **Kontextmenüs** - Rechtsklick-Menüs
- **Systemablaufleistenmenüs** - Menüs in der Systemablaufleiste/Benachrichtigungsfläche

## Menüs erstellen

### NewMenu()

Erstellt ein neues Menü.

```go
func (a *App) NewMenu() *Menu
```

**Beispiel:**
```go
menu := app.NewMenu()
```

## Menümethoden

### Add()

Fügt ein Menüelement zum Menü hinzu.

```go
func (m *Menu) Add(label string) *MenuItem
```

**Parameter:**
- `label` - Der für das Menüelement angezeigte Text

**Rückgabe:** Das erstellte Menüelement

**Beispiel:**
```go
item := menu.Add("Datei öffnen")
item.OnClick(func(ctx *application.Context) {
    // Klick verarbeiten
})
```

### AddSubmenu()

Fügt ein Untermenü zum Menü hinzu.

```go
func (m *Menu) AddSubmenu(label string) *Menu
```

**Parameter:**
- `label` - Das Untermenü-Label

**Rückgabe:** Das erstellte Untermenü

**Beispiel:**
```go
fileMenu := menu.AddSubmenu("Datei")
fileMenu.Add("Neu")
fileMenu.Add("Öffnen")
fileMenu.Add("Speichern")
```

### AddSeparator()

Fügt eine visuelle Trennlinie zwischen Menüelementen hinzu.

```go
func (m *Menu) AddSeparator()
```

**Beispiel:**
```go
menu.Add("Kopieren")
menu.Add("Einfügen")
menu.AddSeparator()
menu.Add("Alles auswählen")
```

**Best Practice:** Verwenden Sie Trennlinien, um zusammengehörige Menüelemente zu gruppieren.

### AddCheckbox()

Fügt ein ankreuzbares Menüelement hinzu.

```go
func (m *Menu) AddCheckbox(label string, checked bool) *MenuItem
```

**Parameter:**
- `label` - Das Label der Kontrollkästchen
- `checked` - Anfangszustand der Aktivierung

**Beispiel:**
```go
darkMode := menu.AddCheckbox("Dunkelmodus", false)
darkMode.OnClick(func(ctx *application.Context) {
    isChecked := darkMode.Checked()
    // Dunkelmodus umschalten
})
```

### AddRadio()

Fügt ein Radiomenuelement hinzu (gegenseitig ausschließende Gruppe).

```go
func (m *Menu) AddRadio(label string, checked bool) *MenuItem
```

**Parameter:**
- `label` - Das Label des Radioknopfes
- `checked` - Anfangszustand der Aktivierung

**Beispiel:**
```go
// Radiogruppe für Ansichtsmodi erstellen
viewMenu := menu.AddSubmenu("Ansicht")
listView := viewMenu.AddRadio("Listenansicht", true)
gridView := viewMenu.AddRadio("Rasteransicht", false)
treeView := viewMenu.AddRadio("Baumansicht", false)

listView.OnClick(func(ctx *application.Context) {
    setViewMode("list")
})
gridView.OnClick(func(ctx *application.Context) {
    setViewMode("grid")
})
```

### Update()

Aktualisiert das Menü, um Änderungen an Menüelementen widerzuspiegeln.

```go
func (m *Menu) Update()
```

**Beispiel:**
```go
item.SetEnabled(false)
menu.Update()  // Muss aufgerufen werden, um Änderungen zu übernehmen
```

**Wichtig:** Rufen Sie `Update()` immer nach dem Ändern der Eigenschaften von Menüelementen auf.

## Menüelementmethoden

### OnClick()

Registriert einen Klick-Handler für das Menüelement.

```go
func (mi *MenuItem) OnClick(callback func(ctx *application.Context)) *MenuItem
```

**Parameter:**
- `callback` - Funktion, die aufgerufen wird, wenn das Element geklickt wird

**Rückgabe:** Das Menüelement (für Kettenaufrufe)

**Beispiel:**
```go
item.OnClick(func(ctx *application.Context) {
    fmt.Println("Menüelement geklickt")
    app.Logger.Info("Benutzer hat Menüelement geklickt")
})
```

### SetLabel()

Ändert das Label des Menüelements.

```go
func (mi *MenuItem) SetLabel(label string) *MenuItem
```

**Beispiel:**
```go
item.SetLabel("Speichern unter...")
menu.Update()
```

### SetEnabled()

Aktiviert oder deaktiviert das Menüelement.

```go
func (mi *MenuItem) SetEnabled(enabled bool) *MenuItem
```

**Beispiel:**
```go
// Speichern deaktivieren, wenn kein Dokument geöffnet ist
saveItem.SetEnabled(hasOpenDocument)
menu.Update()
```

**Häufiges Muster:**
```go
// Menüstatus basierend auf dem Anwendungsstatus aktualisieren
func updateMenuState() {
    saveItem.SetEnabled(hasUnsavedChanges)
    undoItem.SetEnabled(canUndo)
    redoItem.SetEnabled(canRedo)
    menu.Update()
}
```

### SetChecked()

Setzt den Aktivierungszustand für Kontrollkästchen-/Radiomenuelemente.

```go
func (mi *MenuItem) SetChecked(checked bool) *MenuItem
```

**Beispiel:**
```go
darkModeItem.SetChecked(isDarkModeEnabled)
menu.Update()
```

### Checked()

Gibt den aktuellen Aktivierungszustand zurück.

```go
func (mi *MenuItem) Checked() bool
```

**Beispiel:**
```go
if darkModeItem.Checked() {
    // Dunkelmodus ist aktiviert
}
```

### SetAccelerator()

Legt eine Tastenkombination für das Menüelement fest.

```go
func (mi *MenuItem) SetAccelerator(accelerator string) *MenuItem
```

**Parameter:**
- `accelerator` - Tastenkombination (z. B. "Ctrl+S", "Cmd+Q")

**Format des Accelerators:**
- **Modifikatoren:** `Ctrl`, `Cmd`, `Alt`, `Shift`
- **Tasten:** `A-Z`, `0-9`, `F1-F12`, `Enter`, `Backspace`, usw.
- **Plattform:** Verwenden Sie `Cmd` auf macOS, `Ctrl` auf Windows/Linux

**Beispiel:**
```go
saveItem.SetAccelerator("Ctrl+S")
quitItem.SetAccelerator("Ctrl+Q")
newItem.SetAccelerator("Ctrl+N")
```

**Plattformbewusstes Beispiel:**
```go
import "runtime"

var quitShortcut string
if runtime.GOOS == "darwin" {
    quitShortcut = "Cmd+Q"
} else {
    quitShortcut = "Ctrl+Q"
}
quitItem.SetAccelerator(quitShortcut)
```

### SetTooltip()

Legt einen Tooltip fest, der beim Überfahren des Menüelements angezeigt wird.

```go
func (mi *MenuItem) SetTooltip(tooltip string) *MenuItem
```

**Beispiel:**
```go
item.SetTooltip("Öffnet eine Datei von der Festplatte")
```

### SetHidden()

Zeigt oder verbirgt das Menüelement.

```go
func (mi *MenuItem) SetHidden(hidden bool) *MenuItem
```

**Beispiel:**
```go
// Debug-Menü in der Produktion ausblenden
debugItem.SetHidden(!isDevelopment)
menu.Update()
```

## Anwendungsmenü

### app.Menu.Set()

Legt die Hauptmenüleiste der Anwendung fest.

```go
func (mm *MenuManager) Set(menu *Menu)
```

**Beispiel:**
```go
menu := app.NewMenu()

// Datei-Menü
fileMenu := menu.AddSubmenu("Datei")
fileMenu.Add("Neu").SetAccelerator("Ctrl+N").OnClick(newFile)
fileMenu.Add("Öffnen").SetAccelerator("Ctrl+O").OnClick(openFile)
fileMenu.Add("Speichern").SetAccelerator("Ctrl+S").OnClick(saveFile)
fileMenu.AddSeparator()
fileMenu.Add("Beenden").SetAccelerator("Ctrl+Q").OnClick(func(ctx *application.Context) {
    app.Quit()
})

// Bearbeiten-Menü
editMenu := menu.AddSubmenu("Bearbeiten")
editMenu.Add("Rückgängig").SetAccelerator("Ctrl+Z").OnClick(undo)
editMenu.Add("Wiederholen").SetAccelerator("Ctrl+Y").OnClick(redo)
editMenu.AddSeparator()
editMenu.Add("Ausschneiden").SetAccelerator("Ctrl+X").OnClick(cut)
editMenu.Add("Kopieren").SetAccelerator("Ctrl+C").OnClick(copy)
editMenu.Add("Einfügen").SetAccelerator("Ctrl+V").OnClick(paste)

app.Menu.Set(menu)
```

**Plattformhinweise:**
- **macOS:** Menü erscheint in der oberen Menüleiste
- **Windows/Linux:** Menü erscheint in der Fenstertitelleiste
- **macOS:** Fügt automatisch das Anwendungsmenü mit dem App-Namen hinzu

## Kontextmenüs

### app.ContextMenu.Add()

Registriert ein Kontextmenü (Rechtsklick-Menü) mit einem bestimmten Namen.

```go
func (cm *ContextMenuManager) Add(name string, menu *Menu)
```

**Parameter:**
- `name` - Eindeutige Kennung für das Kontextmenü
- `menu` - Das anzuzeigende Menü

**Go:**

```go
// Kontextmenü erstellen
contextMenu := app.NewMenu()
contextMenu.Add("Ausschneiden").OnClick(cut)
contextMenu.Add("Kopieren").OnClick(copy)
contextMenu.Add("Einfügen").OnClick(paste)
contextMenu.AddSeparator()
contextMenu.Add("Alles auswählen").OnClick(selectAll)

// Registrieren
app.RegisterContextMenu("editor", contextMenu)
```

**HTML:**

```html
<!-- Kontextmenü beim Rechtsklick auslösen -->
<div data-wails-context-menu="editor">
    Rechtsklick hier für Kontextmenü
</div>
```

**Dynamische Kontextmenüs:**
```go
// Kontextmenü basierend auf der Auswahl aktualisieren
func updateContextMenu() {
    contextMenu := app.NewMenu()

    if hasSelection {
        contextMenu.Add("Ausschneiden").OnClick(cut)
        contextMenu.Add("Kopieren").OnClick(copy)
    }

    contextMenu.Add("Einfügen").SetEnabled(hasClipboardContent).OnClick(paste)

    app.RegisterContextMenu("editor", contextMenu)
}
```

## Systemablaufleistenmenü

### app.SystemTray.New()

Erstellt ein neues Symbol in der Systemablaufleiste.

```go
func (sm *SystemTrayManager) New() *SystemTray
```

**Beispiel:**
```go
tray := app.SystemTray.New()
```

### SetIcon()Legt das Symbol in der Systemablaufleiste fest.

```go
func (st *SystemTray) SetIcon(icon []byte) *SystemTray
```

**Beispiel:**
```go
iconData, _ := os.ReadFile("icon.png")
tray.SetIcon(iconData)
```

### SetMenu()

Legt das Menü für die Systemablaufleiste fest.

```go
func (st *SystemTray) SetMenu(menu *Menu) *SystemTray
```

**Beispiel:**
```go
trayMenu := app.NewMenu()
trayMenu.Add("Show Window").OnClick(func(ctx *application.Context) {
    window.Show()
    window.SetFocus()
})
trayMenu.Add("Settings").OnClick(openSettings)
trayMenu.AddSeparator()
trayMenu.Add("Quit").OnClick(func(ctx *application.Context) {
    app.Quit()
})

tray.SetMenu(trayMenu)
```

### SetTooltip()

Legt den Tooltip fest, der angezeigt wird, wenn man mit dem Mauszeiger über das Symbol in der Systemablaufleiste fährt.

```go
func (st *SystemTray) SetTooltip(tooltip string) *SystemTray
```

**Beispiel:**
```go
tray.SetTooltip("My Application - Running")
```

### OnClick()

Verarbeitet einen Linksklick auf das Symbol in der Systemablaufleiste.

```go
func (st *SystemTray) OnClick(callback func()) *SystemTray
```

**Beispiel:**
```go
tray.OnClick(func() {
    if window.IsVisible() {
        window.Hide()
    } else {
        window.Show()
        window.SetFocus()
    }
})
```

## Vollständige Beispiele

### Standard-Anwendungsmenü

```go
package main

import (
    "github.com/wailsapp/wails/v3/pkg/application"
)

func createMenu(app *application.Application) *application.Menu {
    menu := app.NewMenu()

    // Datei-Menü
    fileMenu := menu.AddSubmenu("File")
    fileMenu.Add("New").
        SetAccelerator("Ctrl+N").
        OnClick(func(ctx *application.Context) {
            // Neues Dokument erstellen
        })
    fileMenu.Add("Open").
        SetAccelerator("Ctrl+O").
        OnClick(func(ctx *application.Context) {
            // Dateidialog öffnen
        })
    fileMenu.Add("Save").
        SetAccelerator("Ctrl+S").
        OnClick(func(ctx *application.Context) {
            // Dokument speichern
        })
    fileMenu.AddSeparator()
    fileMenu.Add("Exit").OnClick(func(ctx *application.Context) {
        app.Quit()
    })

    // Bearbeiten-Menü
    editMenu := menu.AddSubmenu("Edit")
    editMenu.Add("Undo").SetAccelerator("Ctrl+Z")
    editMenu.Add("Redo").SetAccelerator("Ctrl+Y")
    editMenu.AddSeparator()
    editMenu.Add("Cut").SetAccelerator("Ctrl+X")
    editMenu.Add("Copy").SetAccelerator("Ctrl+C")
    editMenu.Add("Paste").SetAccelerator("Ctrl+V")

    // Ansicht-Menü
    viewMenu := menu.AddSubmenu("View")
    darkMode := viewMenu.AddCheckbox("Dark Mode", false)
    darkMode.OnClick(func(ctx *application.Context) {
        // Dunkelmodus umschalten
        isChecked := darkMode.Checked()
        app.Logger.Info("Dark mode", "enabled", isChecked)
    })
    viewMenu.AddSeparator()
    viewMenu.AddRadio("List View", true)
    viewMenu.AddRadio("Grid View", false)
    viewMenu.AddRadio("Detail View", false)

    // Hilfe-Menü
    helpMenu := menu.AddSubmenu("Help")
    helpMenu.Add("Documentation").OnClick(func(ctx *application.Context) {
        // Dokumentation öffnen
    })
    helpMenu.Add("About").OnClick(func(ctx *application.Context) {
        // Über-Dialog anzeigen
    })

    return menu
}

func main() {
    app := application.New(application.Options{
        Name: "Menu Demo",
    })

    menu := createMenu(app)
    app.Menu.Set(menu)

    window := app.Window.New()
    window.Show()

    app.Run()
}
```

### Anwendung mit Systemablaufleiste

```go
func setupSystemTray(app *application.Application, window *application.Window) {
    // Systemablaufleiste erstellen
    tray := app.SystemTray.New()

    // Symbol festlegen
    iconData, _ := os.ReadFile("icon.png")
    tray.SetIcon(iconData)
    tray.SetTooltip("My App - Running")

    // Linksklick auf das Symbol in der Systemablaufleiste verarbeiten
    tray.OnClick(func() {
        if window.IsVisible() {
            window.Hide()
        } else {
            window.Show()
            window.SetFocus()
        }
    })

    // Menü für die Systemablaufleiste erstellen
    trayMenu := app.NewMenu()

    showItem := trayMenu.Add("Show Window")
    showItem.OnClick(func(ctx *application.Context) {
        window.Show()
        window.SetFocus()
    })

    trayMenu.AddSeparator()

    trayMenu.Add("Settings").OnClick(func(ctx *application.Context) {
        // Einstellungsfenster öffnen
    })

    trayMenu.Add("About").OnClick(func(ctx *application.Context) {
        // Über-Dialog anzeigen
    })

    trayMenu.AddSeparator()

    trayMenu.Add("Quit").OnClick(func(ctx *application.Context) {
        app.Quit()
    })

    tray.SetMenu(trayMenu)
}
```

### Dynamische Menüaktualisierungen

```go
type Editor struct {
    app         *application.Application
    menu        *application.Menu
    undoItem    *application.MenuItem
    redoItem    *application.MenuItem
    saveItem    *application.MenuItem
    undoStack   []string
    redoStack   []string
    hasChanges  bool
}

func (e *Editor) createMenu() {
    e.menu = e.app.NewMenu()

    fileMenu := e.menu.AddSubmenu("File")
    e.saveItem = fileMenu.Add("Save").SetAccelerator("Ctrl+S")
    e.saveItem.OnClick(func(ctx *application.Context) {
        e.save()
    })

    editMenu := e.menu.AddSubmenu("Edit")
    e.undoItem = editMenu.Add("Undo").SetAccelerator("Ctrl+Z")
    e.undoItem.OnClick(func(ctx *application.Context) {
        e.undo()
    })

    e.redoItem = editMenu.Add("Redo").SetAccelerator("Ctrl+Y")
    e.redoItem.OnClick(func(ctx *application.Context) {
        e.redo()
    })

    e.updateMenuState()
    e.app.Menu.Set(e.menu)
}

func (e *Editor) updateMenuState() {
    // Menüelemente basierend auf dem aktuellen Zustand aktualisieren
    e.saveItem.SetEnabled(e.hasChanges)
    e.undoItem.SetEnabled(len(e.undoStack) > 0)
    e.redoItem.SetEnabled(len(e.redoStack) > 0)
    e.menu.Update()
}

func (e *Editor) onChange() {
    e.hasChanges = true
    e.updateMenuState()
}

func (e *Editor) save() {
    // Speicherlogik
    e.hasChanges = false
    e.updateMenuState()
}
```

## Best Practices

### ✅ Empfohlen

- **Standard-Tastenkombinationen verwenden** - Befolgen Sie die Plattformkonventionen (Strg+C zum Kopieren usw.)
- **Update() nach Änderungen aufrufen** - Das Menü spiegelt Änderungen sonst nicht wider
- **Zusammengehörige Elemente gruppieren** - Verwenden Sie Trennlinien, um Menüelemente zu organisieren
- **Nicht verfügbare Aktionen deaktivieren** - Nicht ausblenden, sondern mit SetEnabled(false) deaktivieren
- **Klare Labels verwenden** - Kurz und beschreibend sein
- **Plattformkonventionen beachten** - Unterschiede zwischen macOS und Windows/Linux bei Menümustern

### ❌ Nicht empfohlen

- **Update() nicht vergessen** - Der häufigste Fehler
- **Zu tief verschachteln** - Menüs maximal 2-3 Ebenen tief halten
- **Mehrdeutige Labels verwenden** - "Verarbeiten" vs. "Dokument verarbeiten"
- **Zu komplex machen** - Menüs einfach und fokussiert halten
- **Metaphern mischen** - Konsistente Benennung und Organisation

## Plattformspezifische Hinweise

### macOS

- Anwendungsmenü wird automatisch mit dem App-Namen hinzugefügt
- Verwenden Sie `Cmd` statt `Ctrl` für Tastenkombinationen
- "Über", "Einstellungen" und "Beenden" sind standardmäßig im Anwendungsmenü enthalten

### Windows/Linux

- Kein automatisches Anwendungsmenü
- Verwenden Sie `Strg` für Tastenkombinationen
- "Beenden" befindet sich typischerweise im Datei-Menü
