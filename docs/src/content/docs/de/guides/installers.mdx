## Übersicht

Erstellen Sie professionelle Installer für Ihre Wails-Anwendung auf allen Plattformen.

## Plattform-Installer

<Tabs syncKey="platform">
  <TabItem label="Windows" icon="seti:windows">
    ### NSIS-Installer
    
    ```bash
    # NSIS installieren
    # Download von: https://nsis.sourceforge.io/
    
    # Installer-Skript erstellen (installer.nsi)
    makensis installer.nsi
    ```
    
    **installer.nsi:**
    ```nsis
    !define APPNAME "MyApp"
    !define VERSION "1.0.0"
    
    Name "${APPNAME}"
    OutFile "MyApp-Setup.exe"
    InstallDir "$PROGRAMFILES\${APPNAME}"
    
    Section "Install"
        SetOutPath "$INSTDIR"
        File "build\bin\myapp.exe"
        CreateShortcut "$DESKTOP\${APPNAME}.lnk" "$INSTDIR\myapp.exe"
    SectionEnd
    ```
    
    ### WiX Toolset
    
    Alternative für MSI-Installer.
  </TabItem>

  <TabItem label="macOS" icon="apple">
    ### DMG-Erstellung
    
    ```bash
    # DMG erstellen
    hdiutil create -volname "MyApp" -srcfolder build/bin/MyApp.app -ov -format UDZO MyApp.dmg
    ```
    
    ### Code-Signierung
    
    ```bash
    # Anwendung signieren
    codesign --deep --force --verify --verbose --sign "Developer ID" MyApp.app
    
    # Notarisieren
    xcrun notarytool submit MyApp.dmg --apple-id "email" --password "app-password"
    ```
    
    ### App Store
    
    Verwenden Sie Xcode für die App Store-Verteilung.
  </TabItem>

  <TabItem label="Linux" icon="linux">
    ### DEB-Paket
    
    ```bash
    # Paketstruktur erstellen
    mkdir -p myapp_1.0.0/DEBIAN
    mkdir -p myapp_1.0.0/usr/bin
    
    # Binärdatei kopieren
    cp build/bin/myapp myapp_1.0.0/usr/bin/
    
    # Steuerungsdatei erstellen
    cat > myapp_1.0.0/DEBIAN/control << EOF
    Package: myapp
    Version: 1.0.0
    Architecture: amd64
    Maintainer: Ihr Name
    Description: Meine Anwendung
    EOF
    
    # Paket bauen
    dpkg-deb --build myapp_1.0.0
    ```
    
    ### RPM-Paket
    
    Verwenden Sie `rpmbuild` für RPM-basierte Distributionen.
    
    ### AppImage
    
    ```bash
    # appimagetool verwenden
    appimagetool myapp.AppDir
    ```
  </TabItem>
</Tabs>

## Automatisches Paketieren

### Verwendung von GoReleaser

```yaml
# .goreleaser.yml
project_name: myapp

builds:
  - binary: myapp
    goos:
      - windows
      - darwin
      - linux
    goarch:
      - amd64
      - arm64

archives:
  - format: zip
    name_template: "{{ .ProjectName }}_{{ .Version }}_{{ .Os }}_{{ .Arch }}"

nfpms:
  - formats:
      - deb
      - rpm
    vendor: Ihr Unternehmen
    homepage: https://example.com
    description: Meine Anwendung
```

## Best Practices

### ✅ Tun

- Code-Signierung auf allen Plattformen durchführen
- Versionsinformationen einbeziehen
- Deinstaller erstellen
- Installationsprozess testen
- Klare Dokumentation bereitstellen

### ❌ Nicht tun

- Code-Signierung nicht überspringen
- Dateizuordnungen nicht vergessen
- Pfade nicht hartkodieren
- Tests nicht überspringen
