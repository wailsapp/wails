## NSIS-Installer

Das Standard-Paketierungsformat erstellt einen NSIS-Installer:

```bash
wails3 package GOOS=windows
```

Dies führt `wails3 task windows:package` aus, was:
1. Die Anwendung erstellt
2. Den WebView2-Bootstrapper generiert
3. Einen NSIS-Installer erstellt

Ausgabe: `build/windows/nsis/<AppName>-installer.exe`

### MSIX-Paket

Für die Verteilung über den Microsoft Store oder moderne Windows-Bereitstellungen:

```bash
wails3 package GOOS=windows FORMAT=msix
```

Ausgabe: `bin/<AppName>-<arch>.msix`

<Aside type="note">
MSIX erfordert entweder `makeappx.exe` (Windows SDK) oder das MSIX-Tool. Führen Sie `wails3 tool msix-install-tools` aus, um es einzurichten.
</Aside>

## Anpassen des Installers

Die NSIS-Konfiguration befindet sich in `build/windows/nsis/project.nsi`. Bearbeiten Sie diese Datei, um anzupassen:

- Installer-Benutzeroberfläche und Branding
- Installationsverzeichnis
- Startmenü- und Desktop-Verknüpfungen
- Dateizuordnungen
- Lizenzvereinbarung

Die Anwendungsmetadaten stammen aus `build/windows/info.json`:

```json
{
  "fixed": {
    "file_version": "1.0.0"
  },
  "info": {
    "0000": {
      "ProductVersion": "1.0.0",
      "CompanyName": "My Company",
      "FileDescription": "My Application",
      "ProductName": "MyApp"
    }
  }
}
```

## Codesignierung

Signieren Sie Ihre ausführbare Datei und den Installer, um SmartScreen-Warnungen zu vermeiden:

```bash
# Mit dem Wrapper (automatische Plattformerkennung)
wails3 sign GOOS=windows

# Oder direkt über die Tasks
wails3 task windows:sign
wails3 task windows:sign:installer
```

Konfigurieren Sie die Signierung in `build/windows/Taskfile.yml`:

```yaml
vars:
  SIGN_CERTIFICATE: "path/to/certificate.pfx"
  # Oder den Fingerabdruck für Zertifikate im Windows-Speicher verwenden
  SIGN_THUMBPRINT: "certificate-thumbprint"
  TIMESTAMP_SERVER: "http://timestamp.digicert.com"
```

Speichern Sie Ihr Zertifikat-Passwort sicher:

```bash
wails3 setup signing
```

Weitere Details finden Sie unter [Signing Applications](/de/guides/build/signing).

## Erstellen für ARM

```bash
wails3 build GOOS=windows GOARCH=arm64
wails3 package GOOS=windows GOARCH=arm64
```

## Fehlerbehebung

### makensis nicht gefunden

Installieren Sie NSIS:

```bash
# Windows
winget install NSIS.NSIS

# Oder herunterladen von https://nsis.sourceforge.io/
```

### SmartScreen-Warnung

Ihre ausführbare Datei ist nicht signiert. Siehe [Codesignierung](#codesigning) oben.

### WebView2 fehlt

Der Installer enthält einen WebView2-Bootstrapper, der die Laufzeit bei Bedarf herunterlädt. Wenn Sie eine Offline-Installation benötigen, laden Sie den Evergreen Standalone Installer von Microsoft herunter.
