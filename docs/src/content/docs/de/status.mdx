## Aktueller Status: Alpha

Den neuesten Status finden Sie im [Changelog](./changelog).

Unser Ziel ist es, den Beta-Status zu erreichen. Dieser Fahrplan skizziert die wichtigsten Funktionen und Verbesserungen, die wir vor dem Übergang zur Beta umsetzen müssen. Bitte beachten Sie, dass es sich um ein lebendes Dokument handelt, das aktualisiert werden kann, wenn sich Prioritäten verschieben oder neue Erkenntnisse gewonnen werden.

## Wie Sie beitragen können

- Testen Sie die neueste Alpha-Version und melden Sie eventuelle Probleme
- Tragen Sie zur Dokumentation und zu Beispielen bei
- Nehmen Sie an Diskussionen teil und geben Sie Feedback zu vorgeschlagenen Funktionen
- Reichen Sie Pull Requests für Fehlerkorrekturen oder neue Funktionen ein

Wir freuen uns über Beiträge aus der Community. Wenn Sie bei einem dieser Ziele helfen oder Vorschläge für den Fahrplan haben, öffnen Sie bitte ein Issue oder nehmen Sie an unseren Community-Diskussionen teil.

## Feedback und Updates

Dieser Fahrplan kann sich basierend auf dem Feedback der Community und den Prioritäten des Projekts ändern. Wir werden ihn regelmäßig aktualisieren, um unseren Fortschritt und eventuelle Richtungsänderungen widerzuspiegeln. Ihr Input ist wertvoll für die Gestaltung der Zukunft von Wails! Der Fahrplan ist ein lebendes Dokument und kann sich ändern. Wenn Sie Vorschläge haben, öffnen Sie bitte ein Issue. Jeder Meilenstein wird eine Reihe von Zielen haben, die wir erreichen möchten. Diese können sich ändern.
